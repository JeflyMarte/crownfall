### ステップ 3 — モーンゲート初回完走（目安 8〜12 分）

> **初回クリア想定。** 周回トグルはまだ disabled のはず。戦闘中は観察のみ（pause / x1・x2 は可）。

#### 3-A 画面・UI

□ ヘッダーに `B1 — 部屋 n/m [種別]` が表示される（P3-UI2-011）  
□ バトルログは戦闘中のみ・非戦闘はナレーション Panel（P3-UI2-012）  
□ 味方 **4 体**（編成3＋助っ人）が左側に表示される  
□ 敵スプライト頭上に **Lv + 名前** ネームプレート（P3-A-UI-004）  
□ HP バーが各スプライト頭上に追従する  
□ 命中時に浮動ダメージ数字＋Hit VFX（味方・敵双方）  
□ エリート部屋で金枠ラベル / ボス部屋で赤枠ラベル  
□ 初見敵・素材等で図鑑トースト（上部・約3秒）  
□ 上部 **CT 行動順プレビュー** が更新される（P3-D084）

**BAD:** ___

#### 3-B 進行・部屋

□ 部屋が **自動進行** する（手動「次へ」不要・pause で止まる）  
□ **分岐選択 UI が出ない**（退役確認）  
□ 戦闘 / イベント / 宝箱 / エリート / ボス / 出口 が抽選される（計7フロア前後）  
□ イベント部屋は **自動解決**（碑文・回復・Gold 等がナレーションに出る）  
□ 宝箱で Gold＋装飾品抽選  
□ **セルディオン（ボス）** まで到達し撃破できる  
□ EXIT 到達後、自動でリザルトへ（または短い待機後帰還）

**BAD:** ___

#### 3-C 戦闘 AI・スロット（P3-D085/086）

□ 各味方が **通常 / 防御 / スキル①② / 必殺** のいずれかを選択して行動する（ログで確認）  
□ 防御時に被ダメ軽減が体感できる  
□ 必殺（`ultimate_strike`）が発動し大ダメージになる  
□ 戦術プリセット変更が次戦闘以降に反映される（任意: 事前に「積極」等へ変更）

**BAD:** ___

## NO-GO 時の対応候補

| 症状 | 対応候補 |
|---|---|
| クラッシュ / 進行停止 | 再現手順を記録 → P3-FIX-### 即時 |
| 味方が1体しか出ない | セーブ job_id / ロスター編成を確認（P3-ALPHA-002 再発） |
| CT が進まない / 行動ループ | `CombatController.advance_to_next_actor` 調査 |
| 周回が解禁されない | `GameState.is_dungeon_cleared("mourngate")` 確認 |
| スキップなのに報酬なし | `_execute_combat_skip` → `_award_enemy_kill_at` 調査 |
| Threat が機能しない | P3-D104 `party_threat` / ターゲット選択ログ |
| ボスフェーズが発火しない | P3-D116 `_check_boss_phase_transition` |
| テンポが遅すぎ/速すぎ | x2 / 周回ON / `floor_count` 調整 Decision |
