### ステップ 2 — 装備 / 鑑定 / 比較 UX（目安 60 秒）

> ラン前に装備状態を確認する（セーブデータがある場合）

**EquipmentScene**  
✅武器 / 防具 / 装飾品の装備枠が表示される  
✅ 各装備枠に「比較 1 行」（ATK / DEF / 主要 Affix）が表示される  
✅ 鑑定済み武器に切り替えられる  

**AppraisalScene**  
▶️未鑑定品が一覧表示される  
▶️ 「一括鑑定」ボタンが機能する（Gold 残量が表示され、押すとまとめて鑑定）  
▶️ Gold 不足時は途中で止まり、残りが未鑑定のままになる  

BAD: 拠点画面からは未鑑定品が表示されない（所有していないだけの可能性）

---

### ステップ 3 — ダンジョン 1 ラン（目安 180 秒）

> ダンジョンを選択し探索を開始する。戦闘中は観察のみ（「次の部屋」ボタン以外は操作不要）。

**画面レイアウト**

**✅** バトルフィールド全体が 720px 幅に収まっている（スプライトがはみ出さない）  
✅バトルログパネルが 9-slice フレーム（ Bronze ボーダー）で表示される  
❌味方 3 体が左側（x≈100/240/380）、敵が右側（x≈560）に配置される  

味方が一人しか配置されない

**戦闘 UI**  
✅HP バーが各スプライトの頭上に重なって表示される  
✅ 攻撃ごとに浮動ダメージ数字がスプライト周辺に出る  
✅ バトルログにダメージ / スキル / 状態ログが流れる  
✅戦闘開始時に「[先制:パーティ]」または「[先制:敵]」ログが出る  

**戦闘 AI**  
❓Guardian が生存中は被弾ログに `[守護]` プレフィックスが付く  
❓ Guardian が倒れると次優先（Warrior / ランダム）が被弾する  
✅エリート部屋でのみ curse 状態が味方につく（通常 COMBAT 部屋では付かない）  

**部屋種別（10 部屋）**  
✅COMBAT — 通常敵と戦闘  
✅ EVENT — 2 択イベントが発生（選択肢を選べる）  
✅ TREASURE — Gold+30 と装飾品抽選  
✅ ELITE — 強敵（EXP/Gold x1.5）+ ボーナスドロップ  
✅ MID_BOSS / BOSS — ボス戦  
✅ EXIT — 探索完了、「探索終了」で帰還できる  
✅全滅時は ResultScene へ遷移し蓄積分だけ報酬を持ち帰る  

**Codex 確認（オプション）**  
✅ ≡ メニューから Codex へ遷移できる（または拠点で確認）  
✅ 「Guide」タブに属性 5 種・状態異常 6 種の解説がある
