## 強み（ゲームと相性が良い要素）

1. **王冠は Fall したが滅びていない** — タイトル `Crownfall` と直結する中核ミステリー
2. **王遺産＝所有物ではなく継承物** — 鑑定・発掘ループと一致
3. **世界地図は未完成** — プレイヤーが「一筆書き加える」発見体験と接続しやすい
4. **無名英雄** — `17_WorldBible.md` 原則2（探索者であり英雄ではない）と完全一致
5. **深淵の最下層未到達** — ダンジョン拡張の長期フック
6. **巡礼路・世界遺産の「建造者不明」** — 断片ロア向き（`18_LoreDeliveryGuide` と整合）
7. **第十王遺産 / 消された王** — `03_世界観.md` の中核の謎と接続可能

## 弱点・リスク

### 構造上の問題（raw）

| 問題 | 詳細 |
|---|---|
| GPT メタ行 | `## 次にやること` が約 100 回。公式文書に不適切 |
| 見出し欠落 | **A-10 灯火** の本文が未執筆のまま A-11 へジャンプ |
| 見出し崩れ | 第三英雄リュシアンが `#` 見出しではなく太字のみ |
| 重複 | 世界神ノクスが同一章で 3 回繰り返し |
| 欠番 | 英雄 3-3 が目次上欠落（本文は太字ブロックのみ） |
| スケール膨張 | Phase0→9 まで一気に展開。ゲーム実装フェーズと乖離 |

### ゲーム SSOT との矛盾（要調整）

| GPT 草案 | 現行 SSOT | 調整方針（v1.1） |
|---|---|---|
| 九王は互いに戦わなかった | **九王戦争**が王国時代を終えた（`03_世界観` / HE-002） | 九王**時代の秩序**と、後の**九王戦争**を時系列で分離。戦争の当事者は「九王本人か、その名を継ぐ王国か」は未解明とする |
| 王国 9 つ（アルカディア等） | **王国 5 つ**（K-001〜005 / `19_KingdomBible.md`） | 五王国を正。九王国名は神話地理・失われた時代の呼称として縮小掲載 |
| 王都アルカディアが現役首都 | 王都跡・White Kingdom 系の遺跡 | アルカディアは**失われた第一王都**の古名に降格 |
| 継承の最終章・選ばれし継承者 | プレイヤーはレリックハンター（軽量ストーリー） | Phase9 叙事詩は **Future / 別媒体候補** として附録扱い。ゲーム本編スコープ外 |
| 王遺産＝装備4枠目「王遺産」 | MVP は 3 枠、王遺産は正式版（`01_MVP方針決定`） | 王遺産は世界観用語として維持。装備枠とは用語分離を明記 |

## ネーミングレビュー

### 要改名（v1.1 で反映）

| 現名称 | 問題 | 改訂案 |
|---|---|---|
| 継承剣レガシア（Legacia） | 英語 Legacy の直訳すぎる | **継承剣レガート**（Legart） |
| 翠杖ユグドラ（Yggdra） | 世界樹（Yggdrasil）の連想が強すぎる | **翠杖ヴェルド**（Verd） |
| 深竜アビス（Abyss） | 地形「深淵」と同名で混乱 | **深竜トレンチャ**（Trenches） |
| 王国・王の二重シルヴァリア | 第八王と第八王国が同綴 | 王国側を **シルヴァーン王国**（Sylvane）に分離 |
| セラディア王国 / セラディス王 | 音が近く区別しづらい | 王国を **アーカイヴ王国**（Archive）に（学術国家のイメージ） |

### 許容（良い名前）

| 名称 | 理由 |
|---|---|
| エルディオン、ヴァルケイン、オルグラン | 音感が異なり、役割（開拓・守護・鍛冶）が伝わる |
| リュシアン（深淵の英雄） | 深淵＝Lucifer 連想を避けつつ、ルーツ語感で異質 |
| 星屑（Starseed） | 素材名として直感的。ゲーム内 `relic_shard` 系と接続しやすい |
| 灯火（The Last Flame） | 文明の象徴として詩的。A-10 として新規執筆 |

### 汎用すぎるが文脈で救済可能

| 名称 | 扱い |
|---|---|
| アルカディア（Arcadia） | **古代の呼称**に限定すれば可。現役国名にはしない |
| ネレイオス / ネレイア | 海王・海王国で意図的ペアなら可。初見は似るので初回ロアで区別を書く |
