# 30_装備ステータス定義

> **SSOT** — 装備のステータス一覧・意味・ロール可否・接続状況。  
> ディアブロ型ランダム化（個体ロール拡張）の前提資料。  
> 関連: `07_武器_装備.md` / `09_ドロップ_Affix.md` / `03_Resource設計.md`

---

## 1. ステータス分類

| 区分 | 説明 |
|---|---|
| **ロール** | ドロップ時に個体ごとにランダム化する数値 |
| **Affix** | Prefix/Suffix で個体ごとに抽選される追加効果 |
| **固定メタ** | 装備マスタ（`WeaponData` 等）で決まり、個体差なし |
| **成長** | 装備レベル・炉研ぎなどプレイ中に変化 |

## 2. 武器

### 2.1 必須ロール（全武器個体に必ず付与）

| id | 表示名 | 型 | 意味 | 現状 |
|---|---|---|---|---|
| `attack` | 攻撃力 | int | 与ダメの基礎値 | ✅ `WeaponStatResolver.apply_drop_stats` |

**武器の必須項目は攻撃力のみ。**

### 2.2 任意（未設定時デフォルト）

レア度で付与するランダムステータス数（必須 ATK 除く）: COMMON=1 / RARE=2 / EPIC=3 / LEGENDARY=4。最高値ロールは表示名に `⭐️`（§3.2 参照）。

| id | 表示名 | 型 | デフォルト | 現状 |
|---|---|---|---|---|
| `element` | 属性 | String | `""`（無属性） | ✅ 個体フィールド・マスタからコピー |
| `element_power` | 属性値 | int | `0` | ✅ 案A配線済み |
| `bane_class` | 生態特効 | String | `""`（なし） | ✅ 個体フィールド |
| `bane_multiplier` | 特効倍率 | float | 未使用 | `bane_class` 非空時 1.3 |
| `attack_speed` | 攻撃速度 | float | `1.0` | ✅ `WeaponStatResolver` |
| `critical_rate` | 会心率 | float | `0.05` | ✅ 同上 |
| `critical_damage` | 会心ダメ | float | `1.5` | ✅ 同上 |
| `on_hit_status` | 状態付与 | String + float | なし | ✅ 抽選（⭐️対象） |

#### 状態付与（`on_hit_status` — 武器ランダム抽選）

- **プール:** `poison` / `chill` / `shock` / `ignite` / `curse`（**`stun` 除外**）
- **確率:** レア度別ロール（100% 不可）
  - MIN: COMMON 15% / RARE 20% / EPIC 25% / LEGENDARY 30%
  - MAX: COMMON 25% / RARE 30% / EPIC 35% / LEGENDARY 40%
- **発動:** 通常攻撃ヒット時。`EvolutionTraits.effective_status_chance` で補正
- **個体フィールド:** `on_hit_status_id` / `on_hit_status_chance`

#### 属性値（案A — P3-EQ-STAT-005）

- 式: `damage × (1 + element_power × ELEMENT_POWER_K)`（`ELEMENT_POWER_K=0.01`）
- **無属性時は `element_power` 無視**
- ドロップ時: 属性あり武器は `base_element_power + randi()%(レア度別max+1)`

### 2.3 Affix 対象

武器は Prefix×1 + Suffix×1。付与可能 stat_type は §5 参照。

### 2.4 固定メタ（ロール対象外）

| id | 表示名 | 意味 | 接続 |
|---|---|---|---|
| `weapon_type` | 武器種 | 大剣/弓/杖… | ✅（適合武器ボーナス等） |
| `tags` | タグ | 斬撃/刺突/魔法… コンボ条件 | ✅ |
| `fixed_skill_id` | 固有スキル | 武器専用技（レジェンド等） | ✅ |
| `range_category` | 射程区分 | **melee / mid / long**（後述） | ✅ 戦闘内部 |
| `weight` | 重量 | 表示用 | 表示のみ |
| `rarity` | レアリティ | ドロップ重み・Affix tier・成長率 | ✅ |

#### 射程区分（`range_category`）

- **戦闘で使用中**（P3-D106f）: マスタの `base_attack_range`（数値）→ `CombatRange` が **melee / mid / long** に変換。
- 用途: 陣形与ダメ補正、戦術条件 `self_range`、スキル射程解決のフォールバック。
- **プレイヤー UI 非表示**。ドロップ時の**ロール対象外**（武器種ごとの固定メタ）。
- スキル発動時は `SkillData.range_type` が優先（武器射程より上書き）。

> **オミット方針:** 装備ステータス定義・ランダム化対象から **数値 `attack_range` は除外**。内部実装は `range_category` 固定メタとして維持。

### 2.5 成長

| id | 表示名 | 意味 |
|---|---|---|
| `equip_level` | 装備Lv | 戦闘成長。ATK 等をスケール |
| `equip_exp` | 装備EXP | 次 Lv までの蓄積 |
| `enhance_level` | 炉研ぎ | 鍛冶屋 +1〜+5（武器のみ） |

### 2.6 未接続（定義のみ・ロール対象外）

| id | 備考 |
|---|---|
| `knockback` | 将来 |
| `stagger_power` | 将来（旧よろけ） |

## 3. 防具

### 3.1 必須ロール

| id | 表示名 | 型 | 意味 | 現状 |
|---|---|---|---|---|
| `defense` | 防御力 | int | 被ダメ軽減（DEF 逓減式） | ✅ `ArmorStatResolver.apply_drop_stats`（レア度別レンジ） |

**防具の必須項目は防御力のみ。**

### 3.2 任意（抽選・未設定時デフォルト）

レア度で付与するランダムステータス数（必須 DEF 除く）:

| レア度 | ランダム数 |
|---|---|
| COMMON | 1 |
| RARE | 2 |
| EPIC | 3 |
| LEGENDARY | 4 |

数値ロールが最高値のとき `perfect_roll_count` +1 → 表示名末尾に `⭐️`（武器・防具共通）。

| id | 表示名 | 型 | デフォルト | 現状 |
|---|---|---|---|---|
| `hp_bonus` | HP ボーナス | int | `0` | ✅ 抽選 |
| `resist_elements` | 属性耐性 | Array[String] | `[]` | ✅ 抽選（属性数+軽減率ロール・⭐️対象） |
| `exp_gain_rate` | 経験値獲得率 | float | `0.0` | ✅ 抽選 |
| `gold_gain_rate` | ゴールド獲得率 | float | `0.0` | ✅ 抽選 |
| `rare_drop_rate` | レアドロップ率 | float | `0.0` | ✅ 抽選 |
| `status_immunities` | 状態異常無効 | Array[String] | `[]` | ✅ 抽選（種類数ロール・⭐️対象） |
| `evasion_rate` | 回避率 | float | `0.0` | ✅ 抽選（2〜15%・⭐️対象） |

回避率は防具+装飾品で加算（上限50%）。敵攻撃を完全回避し、戦闘画面に `Miss!` 表示。

### 3.3 固定メタ

| id | 表示名 | 意味 | 接続 |
|---|---|---|---|
| `resist_elements`（マスタ） | 属性耐性候補 | 抽選ヒット時のコピー元 | マスタ設計 |
| `fixed_passive_id` | 固有パッシブ | レジェンド防具の常時効果 | ✅ |
| `rarity` | レアリティ | 同上 | ✅ |

### 3.4 成長

武器と同型（`equip_level` / `equip_exp`）。炉研ぎなし。

## 4. 装飾品

**方針（P3-EQ-STAT-008）:** 必須ステータスなし。レア度でランダム1〜4種（武器・防具と同表）。マスタ `.tres` の数値はロール参考のみ。Affix は別レイヤー。⭐️は武器・防具と同型。

### 4.1 ランダムプール

| id | 表示名 | 型 | デフォルト | 現状 |
|---|---|---|---|---|
| `hp_bonus` | HP | int | `0` | ✅ 抽選 |
| `attack_bonus` | 攻撃力 | int | `0` | ✅ 抽選 |
| `defense_bonus` | 防御力 | int | `0` | ✅ 抽選 |
| `crit_rate_bonus` | 会心率 | float | `0.0` | ✅ 抽選 |
| `exp_gain_rate` | 経験値獲得率 | float | `0.0` | ✅ 抽選 |
| `gold_gain_rate` | ゴールド獲得率 | float | `0.0` | ✅ 抽選 |
| `rare_drop_rate` | レアドロップ率 | float | `0.0` | ✅ 抽選 |
| `evasion_rate` | 回避率 | float | `0.0` | ✅ 抽選（2〜15%・⭐️対象） |

### 4.2 報酬率の意味（Affix 共通）

**`luck` / `luck_bonus` は使用しない。**

| id | 表示名 | 適用タイミング |
|---|---|---|
| `exp_gain_rate` | 経験値獲得率 | `accumulate_rewards` / `grant_exp_to_party` |
| `gold_gain_rate` | ゴールド獲得率 | `accumulate_rewards` |
| `rare_drop_rate` | レアドロップ率 | 装備 rarity 抽選 |

### 4.3 固定メタ

| id | 意味 |
|---|---|
| `fixed_passive_id` | レジェンド装飾の固有パッシブ |
| `rarity` | レアリティ |

## 5. Affix stat_type 一覧（全装備共通）

| stat_type | 表示名 | 効果 | 主な付与先 |
|---|---|---|---|
| Attack | 攻撃力 | ATK +固定 | 武器 |
| Defense | 防御力 | DEF +固定 | 防具 |
| HP | HP | 最大 HP +固定 | 防具・装飾 |
| Critical | 会心率 | CRT +加算 | 武器・装飾 |
| Critical Damage | 会心ダメ | クリ倍率 +加算 | 武器（**新設・未実装**） |
| Attack Speed | 攻撃速度 | SPD +加算 | 武器 |
| Healing | 回復量 | 回復スキル +固定 | 防具 |
| Gold Gain | ゴールド獲得 | `gold_gain_rate` +加算 | 装飾・Affix |
| EXP Gain | 経験値獲得 | `exp_gain_rate` +加算 | 装飾・Affix |
| Rare Drop | レアドロップ | `rare_drop_rate` +加算 | 装飾・Affix |
| Material Gain | 素材獲得 | 素材 +固定 | 防具 |
| Shock / Ignite / Chill / Poison | 状態付与 | 攻撃時付与確率 +加算 | 武器 |

## 6. レアリティと性能の関係

レア度は **単一テーブルで全ステータスを底上げする係数ではない**。以下の複合。

| レイヤー | 内容 |
|---|---|
| マスタ設計 | 高レア武器ほど `base_attack` 等が高い傾向（個別 `.tres`） |
| ドロップ重み | COMMON 120 / RARE 45 / EPIC 15 / LEGENDARY 1（P3-EQ-LEG-DROP-001: ★相対を約1/3へ） |
| Affix tier | 装備レアが高いほど高 tier Affix が付きやすい |
| ロールレンジ | **未実装** — レア度別 min/max を §2〜4 のロール対象に適用予定 |
| 装備Lv成長 | LEGENDARY のみ成長率 ×1.25 |

## 7. 集計ルール（戦闘時）

```
実効ATK     = scale(rolled_attack, equip_level) + enhance + Affix(Attack) + 装飾 + Lv + ジョブ
実効属性    = WeaponInstance.element（空=無属性）
属性値補正  = 属性あり時 damage × (1 + element_power × 0.01)  ✅ P3-EQ-STAT-005
生態特効    = bane_class 一致時 ×1.3（空=なし）
攻撃速度    = resolve_attack_speed（未設定→1.0）
会心率      = resolve_critical_rate（未設定→0.05） + 装飾 + Affix
会心ダメ    = resolve_critical_damage（未設定→1.5） + Affix
実効DEF     = scale(rolled_defense, equip_level) + Affix(Defense) + 装飾 + ...
ゴールド率  = 1.0 + Σ gold_gain_rate（装飾 + 防具 + Affix）
経験値率    = 1.0 + Σ exp_gain_rate（装飾 + 防具 + Affix）
レアドロップ = base_weight × (1 + rare_drop_add × rarity_tier)   tier=0..3（装飾 + 防具 + Affix）
状態異常無効 = 防具 status_immunities に含む effect_id は敵→味方付与をブロック
```

> `??` = 個体にロール値がなければデフォルト（§2.2）。
