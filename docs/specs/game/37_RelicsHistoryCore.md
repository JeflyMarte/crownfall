## Crownfall の語源 — 王冠（The Crown）

**王冠**は、この世界に唯一無二の王権象徴。現在の所在は不明。その喪失は歴史の転換点であり、作品名 **Crownfall（王冠の失墜）** の語源となる。

各時代の記録に共通する一文：

> 「王冠は倒れた。しかし滅びてはいない。」

「破壊された」と記した史料は存在しない。倒れた（Fall）とだけ記録されている。

## 時代区分（年表）

| 時代 | 概要 |
|---|---|
| 九王時代 | 記録文明の始点。継承と秩序の礎 |
| 王国時代 | 複数の王国による繁栄期。多くの遺跡・武器・街道・工房が生まれた |
| 九王戦争 | 王国時代を終焉させた大戦。真相は失われ、地域ごとに異なる伝承が残る |
| 静寂の時代 | 戦後の文明縮小期。多くの知識と歴史が失われた |
| 探索者の時代 | 現在。探索者ギルドが旧世界の遺跡と生態系を調査する（`31_SeekersGuild.md`） |

> 九王戦争の統一文案：九王の名を冠する秩序は長く続いたが、やがて王国間の対立は「九王戦争」と呼ばれる大戦へと至り、王国時代は終わった。九王その人の物語なのか、後世がそう呼んだのかは、今も学者のあいだで争われている。

## 王遺産と伝説武器（Royal Relics）

**王遺産**は、九王が後世へ託した特別な遺物。宝物ではなく、思想・技術・使命の象徴。発見者は「最初の継承者」であり、未来へ渡す責任を負う。

> 「遺産を持ち帰るな。歴史を持ち帰れ。」

探索（Relics 柱）の大きな目的であり、入手した武器説明文・遺物説明から世界の歴史を断片的に開示する。

## 中核の謎

長文では明かさず、武器説明 / 古文書 / 遺物 / NPC会話 / 発見度から断片的に開示する（`03_世界観.md` ストーリー開示方針）。

| 謎 | 核 |
|---|---|
| 九王戦争の真実 | 当事者が九王本人か、名を継いだ王国かは断定できない |
| 消された第十の王 | ある目録に一度だけ「第十王遺産」の記述。最終行に「これは九王の遺産ではない」 |
| 王冠の失墜 | 「倒れたが滅びていない」（Crownfall の語源） |
| 伝説武器の由来 | 王遺産に刻まれる王冠紋様は時代・地域を超えて一致し、現代では再現できない |
| 世界樹は一本ではない | 確認個体は複数。総数は世界共通の未解明事項 |
| 灯火 | 「誰が最初に灯したか」より「なぜ誰も消させないか」が語り継がれる |
