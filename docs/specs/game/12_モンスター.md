# 12_モンスター

> **生態系刷新（2026-06-26 オーナー決定）:** モンスターは魔物・不死者ではなく **全て実在生物を祖先に持つ**（`29_PostwarEcology.md`）。分類は **Class I〜VII**（`30_EcologyClassification.md`）。命名は `34_MonsterNamingGuide.md`。図鑑は `33_EcologyCodex.md`。
>
> **世界観一本化（P3-D038 / 2026-06-26）:** 旧16敵（不死・機械系：fallen_soldier / bone_walker / ゴーレム / royal_guard_captain 等）は**退役**し `resources/_archive/enemies/` へ退避済。現行の有効な敵は **モーンゲートの生物由来6体のみ**（下記）。以下の旧敵一覧（王都跡 / 白骸墓地 / 地下工廠）は**退役済（歴史参照）**。
