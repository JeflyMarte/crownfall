## レベル制（実装 — P3-D035a）

| 項目 | 値 |
|---|---|
| EXP 付与 | ラン**成功時**、パーティ全員へ同量（`run_exp_reward`）。全滅時は付与なし |
| 必要 EXP | `exp_to_next(L) = 100 × L`（Lv1→2=100, 2→3=200 …） |
| 上限 | **Lv20**（到達で EXP 0 固定） |
| 成長 | **+6 HP / +2 ATK** per Lv（flat） |
| 適用点 | HP=`CombatController._init_party_hp`、ATK=`DungeonScene._calc_attack_base`（affix flat と同段） |
| 永続化 | `Adventurer.exp` を追加し SaveManager で保存 |
| 実装 | `scripts/systems/LevelSystem.gd` |
| 表示 | 拠点メンバー行に `Lv/EXP`、Result にレベルアップ通知 |
| 未対応 | DEF 成長・個別 EXP・ジョブ別成長係数（将来。ジョブ進化 P3-D037 の土台） |

## ジョブと AI 行動思想（Combat Vision — P2-D166）

実装済みジョブ（`resources/jobs/`）は、戦闘 AI の**行動思想**を表す。スキルは補助であり、主役は立ち回りである。

| id | 表示名 | AI 思想（Vision） | 実装状態 |
|---|---|---|---|
| `swordsman` | ソードマン | 前線で被弾を引き受ける近接 | データ + 戦闘 modifier 接続済（簡易ヘイト=前衛） |
| `ranger` | レンジャー | 弓で後方から火力 | 同上（preferred=bow） |
| `alchemist` | アルケミスト | 杖で魔法ダメージ+デバフ（呪い） | 同上（preferred=staff、回復/バフは MVP 外） |

> **実装移行（2026-06-26 / P3-D034）:** 旧 `warrior/guardian/scout` を `swordsman/ranger/alchemist` に置換。セーブ移行は `SaveManager._JOB_MIGRATION`（warrior→swordsman / scout→ranger / guardian→alchemist）。スプライトは旧 CHR_* をプレースホルダ流用（オーナー作画待ち）。簡易ヘイトは `guardian` 起点から `swordsman`（前衛）起点に変更。

将来候補: ヴァンガード（盾前衛）・ビーストテイマー — `world/09_Jobs.md`

**現行 Alpha:** Job modifier は stat に接続済み。位置・ターゲット AI は未接続（CombatTimer 抽象戦闘）。
