# 06_キャラクター・ジョブ

## パーティ

3人編成。

想定ロール

- 前衛
- 中衛
- 後衛

## キャラクター成長

レベル制。ただし強さの大部分は装備に依存する。

MVPレベル範囲：Lv1〜30。

## 基本ジョブ

### 戦士

大剣装備で発動。近接アタッカー。

### 騎士

槍装備で発動。タンク。

### 盗賊

双剣装備で発動。クリティカル。

### 狩人

弓装備で発動。遠距離DPS。

### 魔術師

杖装備で発動。範囲攻撃。
