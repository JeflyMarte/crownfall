# 31_SeekersGuild — 探索者ギルド（Seekers Guild）

**Status:** SSOT（オーナー決定 2026-06-26）
**親:** `29_PostwarEcology.md`
**関連:** `33_EcologyCodex.md` / `36_JobBible.md`

---

## コンセプト

探索者ギルドは「冒険者組合」ではない。正式名称は**探索管理機構**であり、文明復興のために遺跡と生態系を調査する組織である。武力組織ではなく、**調査機関**に近い存在。

---

## 設立目的

九王戦争後、人類は多くの知識を失った。各地には危険な遺跡・未知の生物・古代技術が残され、人々だけでは調査できなくなった。そこで各都市が共同で探索者ギルドを設立した。

---

## ギルドの三大任務

### 1. 遺物回収（Relic Recovery）

失われた武器・宝具・古代技術を回収する。目的は売却ではなく、**文明の継承**。

### 2. 生態調査（Ecology Survey）

危険生物の調査。分布 / 生息数 / 繁殖 / 危険度 / 行動記録を蓄積する。**討伐は調査の一手段であり、目的そのものではない。**

### 3. 地図更新（Frontier Mapping）

探索可能地域を調査し、新ルート / 崩落 / 危険区域 / 新遺跡を記録する。

---

## 探索者とは

探索者は戦士ではなく、**専門調査員**である。一人ひとりが専門分野を持つ。

例: 武器研究 / 生態学 / 地質調査 / 古代史 / 地図作成

プレイヤーが率いる探索隊も、この専門家たちで構成される（ジョブ: `36_JobBible.md`）。

---

## ギルドの評価基準

評価されるのは**討伐数ではない**。

- 未知の生物を発見した
- 新素材を持ち帰った
- 遺跡を調査した
- 危険区域を更新した

といった「成果」が評価対象となる。これにより、探索そのものがゲームの中心目的になる。

---

## プレイヤーの役割

プレイヤーは最前線で戦う人物ではない。探索隊の責任者として、

- 調査地域を選ぶ
- 装備を整える
- 人員を編成する
- 回収品を分析する

という判断を下す。これは現在の「直接操作しないゲームデザイン」（`26_CombatVision.md`）とも一致する。

---

## ゲームループとの対応

| ゲームプレイ | 世界観での意味 |
|---|---|
| ダンジョン選択 | 調査地域の指定 |
| モンスター討伐 | 生態調査・危険度確認 |
| 武器入手 | 遺物回収 |
| 素材獲得 | 生物標本の採取 |
| 図鑑更新 | 調査報告書の作成 |
| 帰還 | ギルドへの報告 |

---

## 今後拡張しやすい要素

- 調査依頼（特定種の追跡）
- 希少種の発見報告
- 地域危険度の変化
- 図鑑コンプリート報酬
- 未踏区域の解放
- 学者・鍛冶師との連携

いずれも「探索と調査」という軸から自然に派生できる。

---

## 設計レビュー（Technical Game Director 視点）

「討伐＝調査」だけでは単調になりやすいため、「観察」「追跡」「採取」も調査の一部として扱える余地を残す。図鑑の達成条件を「討伐数」ではなく以下のような複数段階にする（`33_EcologyCodex.md`）。

- 初回発見 / 初回討伐 / 希少素材採取 / 行動観察 / 上位個体確認
