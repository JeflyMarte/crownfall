# 04_ゲームループ

## 基本ループ

```text
拠点 (Base)
↓
ダンジョン探索 (Dungeon)
↓
帰還・結果確認 (Result)
↓
識別 (Appraisal)
↓
装備変更 (Equipment)
↓
拠点 (Base) ← 繰り返し
```

## MVPループ（歴史 — Phase 2 初期）

> **Alpha 現行は下記「Alpha 現行ループ」参照。** 識定屋導線は P3-D072 でオミット済み。

MVPでは以下の6シーン固定順で進行する。

```text
BaseScene
  → ダンジョンへ ボタン
DungeonScene（部屋単位ステップ進行・戦闘・報酬累計）
  → 探索終了 ボタン
ResultScene（EXP・Gold・入手品表示）
  → 識別へ ボタン（Gold加算タイミング）
AppraisalScene（未識別武器一覧・100G消費・識別）
  → 装備へ ボタン
EquipmentScene（識別済み武器一覧・装備変更）
  → 拠点へ戻る ボタン
BaseScene（Gold・装備・パーティ表示更新）
```

## Alpha 現行ループ（2026-07-01 — P3-ECO-001）

```text
拠点 (BaseScene)
  ├ 英雄管理 — 装備・作戦プリセット・ガンビット・探索方針
  ├ 赤鉄の工房 — 素材+Gold消費クラフト（6レシピ）
  └ TopBar 素材チップ → 鍛冶ショートカット
↓
ダンジョン選択 → 自動探索 (DungeonScene)
  ├ 撃破ドロップ（武器/防具/装飾/素材）
  ├ イベント素材・ELITE素材ボーナス
  └ 探索方針効果（図鑑/素材/遺物）
↓
帰還 (ResultScene)
  ├ 素材アイコン行・作成可能レシピヒント
  └ Gold/EXP 確定
↓
拠点へ — 装備更新・鍛冶消費 → 再出撃
```

**素材経済（P3-D067/D128 + P3-ECO-001）:** 通常 COMBAT 撃破で敵 `codex_materials` をドロップ → インベントリ蓄積 → 鍛冶で武器/防具/装飾を生成（鑑定済み+Affix自動付与）。商人・素材ショップは未導線（P3-D070）。

パーティ編成・ガチャ・図鑑は拠点メニューから。ダンジョンはモーンゲート1本（P3-D034f）。

## 進行・周回（P3-D5DG-002）

```text
メイン帯 N（例: モーンゲート）
  ├ 寄り道ファーム（任意・推奨）— 属性素材・装備
  ├ メイン周回 ×3〜6 — ドロップ/鍛冶で戦力整備
  ├ メイン Boss 初回討伐
  └ メイン帯 N+1 解放
```

1周 4〜6 分（D-002）。**初回から次メインへ直進する想定にしない。**

## 1周の長さ

通常ダンジョンは5分基準。

MVPでは10部屋構成。

```text
START
COMBAT
EVENT
TREASURE
ELITE
EVENT
COMBAT
MID_BOSS
BOSS
EXIT
```

## 判断頻度

30秒に1回程度、何かが起こるようにする。

- 部屋遷移
- イベント選択
- 宝箱
- エリート
- レアドロップ

（方針変更は Backlog — 判断頻度には含めない）
