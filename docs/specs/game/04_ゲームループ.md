# 04_ゲームループ

## 基本ループ

```text
拠点 (Base)
↓
ダンジョン探索 (Dungeon)
↓
帰還・結果確認 (Result)
↓
識別 (Appraisal)
↓
装備変更 (Equipment)
↓
拠点 (Base) ← 繰り返し
```

## MVPループ

MVPでは以下の6シーン固定順で進行する。

```text
BaseScene
  → ダンジョンへ ボタン
DungeonScene（部屋単位ステップ進行・戦闘・報酬累計）
  → 探索終了 ボタン
ResultScene（EXP・Gold・入手品表示）
  → 識別へ ボタン（Gold加算タイミング）
AppraisalScene（未識別武器一覧・100G消費・識別）
  → 装備へ ボタン
EquipmentScene（識別済み武器一覧・装備変更）
  → 拠点へ戻る ボタン
BaseScene（Gold・装備・パーティ表示更新）
```

パーティ編成・ダンジョン選択はMVP対象外。正式版で追加する。

## 正式版ループ（将来）

```text
拠点
→ パーティ編成
→ 装備変更
→ ダンジョン選択
→ 自動探索・戦闘・イベント・宝箱
→ ボス撃破または撤退
→ 帰還・結果確認
→ 識別
→ 装備更新
→ 再挑戦
```

## 1周の長さ

通常ダンジョンは5分基準。

MVPでは10部屋構成。

```text
START
COMBAT
EVENT
TREASURE
ELITE
EVENT
COMBAT
MID_BOSS
BOSS
EXIT
```

## 判断頻度

30秒に1回程度、何かが起こるようにする。

- 部屋遷移
- イベント選択
- 宝箱
- エリート
- レアドロップ

（方針変更は Backlog — 判断頻度には含めない）
