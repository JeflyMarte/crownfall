# 04_ゲームループ

## 基本ループ

```text
拠点
↓
パーティ編成
↓
装備変更
↓
ダンジョン選択
↓
自動探索
↓
戦闘・イベント・宝箱
↓
ボス撃破または撤退
↓
帰還
↓
鑑定
↓
装備更新
↓
再挑戦
```

## 探索中のプレイヤー操作

- 探索優先
- 戦闘優先
- 収集優先
- 帰還優先

## 1周の長さ

通常ダンジョンは5分基準。

MVPでは10部屋構成。

```text
START
COMBAT
EVENT
TREASURE
ELITE
EVENT
COMBAT
MID_BOSS
BOSS
EXIT
```

## 判断頻度

30秒に1回程度、何かが起こるようにする。

- 部屋遷移
- イベント選択
- 宝箱
- エリート
- 方針変更
- レアドロップ
