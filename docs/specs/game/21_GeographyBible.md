# Crownfall Geography Bible

> **地理刷新（2026-06-26 オーナー決定）:** 現在の探索地理は **エルド大陸（Eld Continent）** を正とする（`35_WorldGeography.md`）。本書（White Kingdom / Black Forge 等）は **History 柱の旧王国地理史** として保持し、エルド大陸地名との対応は DevelopmentHQ が別途統合する。

## Overview

Geography Bibleは、Crownfallの世界に存在する地理情報を一元管理するSSOT（Single Source of Truth）である。河川・街道・橋梁・港湾・山岳・農業地域などの地理要素を整理し、World Bible・History Bible・Kingdom Bibleへの補完資料として機能する。

本書は地理情報のみを管理する。歴史はHistory Bibleへ、国家設計はKingdom Bibleへ、世界概要はWorld Bibleへ委譲する。各地域の記載は所属王国・概要・現在の状態の概要レベルに留める。

本書には新しい地名・地理設定を追加しない。History Bible・Kingdom Bible・World Bibleに存在する情報の整理・統合のみを目的とする。

## Continental Structure

大陸は4つの主要地域に区分される。

| 地域 | 概要 | 主要王国 |
|---|---|---|
| 中央平原 | 大陸の中心地域。River of Crownsと街道が交差し、交通・物流の要として機能した。 | White Kingdom / Crown River Kingdom |
| 山岳地域 | 大陸北西部の山岳地帯。火山帯と鉱脈に囲まれ、工業施設と城塞が集中した。 | Black Forge Kingdom |
| 穀倉地域 | 大陸南部の広大な平原。灌漑施設が発達し、大陸の食糧供給を担った。 | Emerald Kingdom |
| 海岸地域 | 大陸西岸の沿岸地域。天然港と灯台網が海上交易を支えた。 | Silver Maritime Kingdom |
