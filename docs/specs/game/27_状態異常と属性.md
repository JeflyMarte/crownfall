## 概要

| 体系 | 役割 |
|---|---|
| **属性（Element）** | モンハン型。**弱点時のみ**追加ダメージ（×1.25）。属性自体に特殊効果なし |
| **状態異常（Status）** | DoT・行動阻害・与/被ダメ補正など、時間軸の効果 |

**無属性:** 武器・スキルの `element = ""` → 弱点ボーナスなし（通常ダメージのみ）。

**重要:** **炎属性（`fire`）≠ 炎上（`ignite`）**。属性は弱点ダメージのみ。炎上は状態異常として別経路で付与する。

### ダメージ補正（P3-D022）

| 条件 | 倍率 |
|---|---|
| 攻撃に属性あり **かつ** 敵がその属性弱点 | ×1.25 |
| 上記以外 | ×1.0 |

耐性ペナルティ（×0.75）は **採用しない**（モンハン型簡略化）。

実装: `ElementResolver.gd` — ログ `[弱点:炎]` 等。

### データ

| フィールド | 用途 |
|---|---|
| `WeaponData.element` | 空 = 無属性 |
| `SkillData.element` | 空 = 武器属性を継承 |
| `EnemyData.element_weakness` | 弱点属性 id 配列 |

## 状態異常（Status Effect）— 6 種

CombatTimer 1 tick = 1.5 秒。

| id | 名称 | 効果（実装） |
|---|---|---|
| `poison` | 毒 | DoT 4/tick、5 tick、最大 3 スタック |
| `chill` | 冷却 | 敵行動 50% スキップ、3 tick |
| `shock` | 感電 | 被ダメ ×1.15 + 行動 30% スキップ、3 tick、最大 2 スタック |
| `ignite` | 炎上 | DoT 3/tick、4 tick、最大 3 スタック |
| `curse` | 呪い | 与ダメ ×0.75、4 tick |

**コンテンツ方針（P3-D024d）:** プレイヤー→敵への curse debuff は現行どおり。**敵→味方** on_hit curse はエリート以降 or 状態 UI 整備後に限定（それまでは新規 on_hit curse を抑制）。
| `stun` | スタン | 行動 100% スキップ、2 tick |

### 付与経路（コンテンツ例）

| 状態 | 経路 |
|---|---|
| poison | 墓蝙蝠 被弾 35% |
| chill | 千鐘の墓守 被弾 20% |
| shock | Affix `shocking`（30%）/ スキル `static_strike`（35%） |
| curse | 朽ちた騎士 / 納骨堂の騎士 被弾 |
| stun | 王都守護兵長 被弾 25% |
| ignite | Affix `igniting` / スキル `kindling_strike` |
| chill | Affix `chilling` / スキル `rime_touch` / 千鐘の墓守 被弾 20% |
| poison | Affix `septic` / 墓蝙蝠 被弾 35% |

### 炎上 vs 炎属性

| | 炎属性 `fire` | 炎上 `ignite` |
|---|---|---|
| 種別 | 属性 | 状態異常 |
| 効果 | 弱点時 ×1.25 | DoT 3/tick |
| 付与 | 武器 `element=fire` | Affix / スキル `apply_status` / 敵 on_hit |
| 例 | `heater_blade` 通常攻撃 → `[弱点:炎]` のみ | 同武器の `kindling_strike` → `[炎上] 付与` |
