# Crownfall World Bible

## Overview

World Bibleは、Crownfallの世界全体を俯瞰する最上位のロア文書であり、History BibleおよびKingdom Bibleの入口となるSSOT（Single Source of Truth）である。世界の構造・時代区分・文明・地理の概要を一元管理し、各専門文書への導線として機能する。

History Bibleが個別の歴史エントリ（HE-001〜HE-066）を記録し、Kingdom Bibleが各王国の国家設計を体系化するのに対し、World Bibleは世界全体の概要のみを提供する。詳細はそれぞれの専門文書へ委譲する。

本書には新しい世界設定を追加しない。History BibleおよびKingdom Bibleに存在する情報の整理・統合のみを目的とする。

---

## World Structure

Crownfallの舞台は、かつて複数の王国が統治した広大な大陸である。

| 区分 | 概要 |
|---|---|
| 大陸 | 複数の王国が分割統治した広大な大陸。現在も各地に遺跡が残る。 |
| 王国時代 | 主要王国（K-001〜K-005）が政治・軍需・食糧・海上物流・内陸物流を分担し、大陸文明を支えた時代。 |
| 現在 | 九王戦争後、大陸規模の帝国は消滅。小規模な村・町が残り、人々は遺跡の影の中で生きる。 |
| 探索者時代 | 現在の時代。レリックハンターが旧世界の遺跡へ踏み込み始めている。プレイヤーはこの時代に属する。 |

## Timeline

| 時代 | 概要 |
|---|---|
| Age of Kingdoms（王国時代） | 人類文明が最も発展した時代。多くのレリック・武器・遺跡がこの時代に生まれた。 |
| Nine Kings War（九王戦争） | 王国時代を終焉させた歴史最大の戦争。真相は断片的にしか残っておらず、地域ごとに異なる伝承が存在する。 |
| Age of Silence（静寂の時代） | 戦後の縮小・生存・忘却の時代。多くの知識・技術・歴史記録が失われた。 |
| Age of Seekers（探索者の時代） | 現在。レリックハンターが遺跡探索を始め、失われた歴史が再び発見されつつある。 |

## Kingdoms

王国時代に存在した主要王国。それぞれが文明の異なる基盤を担い、相互依存関係を持っていた。

| Kingdom | Role |
|---|---|
| White Kingdom | 王権・統治 |
| Black Forge Kingdom | 工業・鍛冶 |
| Emerald Kingdom | 農業・食糧 |
| Silver Maritime Kingdom | 海運・交易 |

## Civilization

王国時代の文明は、5王国が異なる役割を分担することで成立していた。

| 分野 | 概要 |
|---|---|
| 王権 | 世襲王政を基盤とした統治国家が大陸の政治・法体系を担った。騎士団・宗教組織が共存した。 |
| 宗教 | 王権・農業・海洋・工業・河川など地域ごとに異なる信仰が並立した。宗教は国家体制と深く結びついていた。 |
| 工業 | 工業国家が大陸の武器・防具の大半を製造し、各王国へ供給した。 |
| 農業 | 農業国家が大陸の食糧供給を担い、王国連合の生命線として機能した。 |
| 交易 | 海洋国家と内陸国家が海上・河川物流を分担し、大陸全体の交易網を構成した。 |

九王戦争によって5王国はいずれも滅亡し、大陸規模の文明は失われた。現在は小規模な村・町が残るのみであり、歴史の正確な記録は断片的にしか伝わっていない。

## Exploration

現在のAge of Seekers（探索者の時代）において、プレイヤーはレリックハンターとして遺跡を探索する。

プレイヤーは選ばれし英雄ではない。歴史の残骸を発見し、武器・レリック・装備を収集する探索者である。世界はプレイヤーが到着するずっと前から存在し、その歴史もプレイヤーが作るものではなく発見するものである。

| 探索対象 | 起源王国 |
|---|---|
| 王都遺跡 | White Kingdom |
| 山岳工廠・鉱山 | Black Forge Kingdom |
| 地下穀倉・灌漑施設 | Emerald Kingdom |
| 沈没港・海底遺跡 | Silver Maritime Kingdom |
| 石橋・水没都市 | Crown River Kingdom |
