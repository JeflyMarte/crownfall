**Status:** SSOT（オーナー決定 2026-06-26）

**関連:** `33_EcologyCodex.md` / `34_MonsterNamingGuide.md`

## 基本方針

Crownfall のモンスターは、すべて現実の生物を祖先に持つ。分類は魔物ではなく、**生物学に近い考え方**を採用する。探索者ギルドでは、調査・討伐・素材管理のために統一分類を使用している。

---

## クラス一覧

### Class I：獣類（Beasts）

哺乳類を祖先に持つ大型・中型生物。

- 例: 灰狼 / 鎧熊 / 大牙猪 / 岩山羊 / 巨角鹿
- 特徴: 高い身体能力 / 毛皮・牙・骨素材 / 群れを作る種が多い

### Class II：鳥類（Avians）

鳥を祖先に持つ種。

- 例: 巨鴉 / 崖鷲 / 灰梟 / 鉄嘴鳥
- 特徴: 高機動 / 羽根素材 / 索敵能力が高い

### Class III：爬虫類（Reptiles）

トカゲ・蛇など。

- 例: 岩蜥蜴 / 毒蛇 / 沼竜 / 火蜥蜴
- 特徴: 高耐久 / 毒 / 鱗素材

### Class IV：昆虫類（Insects）

昆虫・節足動物。

- 例: 巨甲虫 / 毒蜘蛛 / 刃蟷螂 / 結晶蜂
- 特徴: 外殻 / 毒 / 群体行動

### Class V：水棲類（Aquatics）

魚類・両生類・水辺の生物。

- 例: 巨鮫 / 沼蛙 / 深淵鰻 / 河蟹
- 特徴: 水域支配 / 粘液 / 鱗素材

### Class VI：菌・植物類（Flora）

植物や菌類から進化した危険生物。

- 例: 捕食花 / 棘蔦 / 毒胞子茸
- 特徴: 移動能力は低い / 状態異常 / 薬草素材
- ※植物も「生物」であるため世界観との整合性を保てる。

### Class VII：古代種（Ancients）

王国時代から姿を変えず生き残った希少種。

- 例: 古龍種（※ドラゴンではなく大型飛行爬虫類） / 王角鹿 / 深森巨熊
- 特徴: 生息数が極端に少ない / 地域ボス候補 / 高品質素材

---

## 地域による派生

同じ祖先でも環境で姿が変わる。

| 地域 | 狼系統 |
|---|---|
| 王都跡 | 灰狼 |
| 白骸墓地 | 白骨狼 |
| 黒鉄鉱山 | 鉄牙狼 |
| 深森 | 森影狼 |
| 雪原 | 氷牙狼 |

「狼」という共通祖先を持ちながら、生態が地域ごとに分化している。

---

## 図鑑との接続（概要）

各モンスターには戦闘データだけでなく、生態調査記録が蓄積される（詳細: `33_EcologyCodex.md`）。

| 区分 | 項目 |
|---|---|
| 基本情報 | 分類 / 危険度 / 生息地 / 行動時間 / 群れ規模 |
| 調査情報 | 食性 / 縄張り / 天敵 / 特徴 |
| 素材情報 | 採取素材 / 武器素材 / 防具素材 / 希少素材 |

---

## 世界観への効果

この分類体系により、モンスターは単なる戦闘対象ではなく「世界に存在する生態系の一部」として描ける。新ダンジョン追加時も「この地域にはどのような環境があり、どの生物がどう適応したか」という一貫したルールで設計できる。
