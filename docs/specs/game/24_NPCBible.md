# Crownfall NPC Bible

---

## 1. Purpose

NPC Bibleは、Crownfallの世界に登場するNPCを一元管理するSSOT（Single Source of Truth）である。各NPCの役割・帰属・出自・地域・関係性を体系化し、History Bible・Faction Bible・Dungeon Bible・Geography Bibleへの補完資料として機能する。

本書はNPC情報のみを管理する。ゲームプレイ情報（戦闘・ショップ・リクルート・クエスト・セリフ）は含まない。確認できない情報は「Unknown」として記載する。

---

## 2. Scope

対象：History Bible（HE-001〜HE-066）のPlayer Discovery欄に記載されたNPCタイプ、およびHistory Bible Overviewで個人として言及される歴史的人物。

除外：新規NPC / 新規名称 / 新規ロア / ゲームプレイデータ / 推論による関係性
