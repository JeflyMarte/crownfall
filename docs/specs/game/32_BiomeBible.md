## コンセプト

Biome（バイオーム）は、Crownfall の世界を構成する基本単位である。一つのダンジョンではなく、**一つの生態系**として設計する。探索者は遺跡を攻略するのではなく、生態系へ踏み込み調査を行う。

## Biome-01：嘆きの地下水路（モーンゲート）

> **実装状態（P3-D034 / 2026-06-26）:** `mourngate` として実装済（difficulty 1・入門 DG）。実装モンスターは下記「主なモンスター（構想）」とは別に、オーナー確定の 6 体（セピアハウンド / ルーンローチ / 水晶ハリネズミ / 冠喰いネズミ / クロックモス[Elite] / 水晶骸竜セルディオン[Boss]）。詳細は `12_モンスター.md`。

### 世界設定

王都の地下に建設された巨大排水網。九王戦争後、維持管理されなくなり、地下水と自然によって巨大な地下生態系へ変化した。王都の遺物が多く残ることから、多くの探索隊が最初に向かう調査地となっている。

### 環境

地下 / 湿潤 / 暗所 / 石造建築 / 崩落区画 / 地下河川

### 生態系（食物連鎖の例）

```text
菌類・藻類 → 洞窟鼠 → 毒蛙・水棲甲虫 → 大蛇 → ？？（頂点捕食者）
```

### 主なモンスター

洞窟鼠 / 毒蛙 / 水棲甲虫 / 洞窟蜘蛛 / 大蛇

| 項目 | 内容 |

## 共通設計ルール

1. **モンスターはすべて生物由来** — 悪魔・アンデッド・ゴーレムなどは基本的に登場しない。
2. **生態が能力を決める** — 能力は生態的な理由を説明できることを原則とする（毒蛙→毒液を分泌 / 巨角鹿→突進で縄張りを守る / 洞窟蜘蛛→巣を作り待ち伏せ）。
3. **環境が生物を決める** — モンスターを先に考えず、`環境 → 生態系 → モンスター` の順で設計する。
4. **遺物は歴史を語る** — 遺物は強い装備であるだけでなく「この場所が昔どのような施設だったか」を示す。

---

## 設計レビュー（Lead Game Designer 視点）

「地下水路」と「森」はどちらも自然環境が主体である一方、九王戦争の痕跡がやや薄い。地下水路なら「崩壊した水門」「沈んだ兵站庫」、森なら「朽ちた見張り台」「飲み込まれた街道」といったランドマークを配置し、景観で歴史を語る（Crownfall の「説明しすぎない」方針と一致）。
