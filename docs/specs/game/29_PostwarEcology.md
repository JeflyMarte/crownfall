## コンセプト

九王戦争は人類文明を滅ぼした。しかし**自然は滅びなかった**。

戦争跡地に取り残された生物たちは、数百年という時間の中で環境に適応し、新たな生態系を築いた。

現在の探索者は、

- 古代遺物を回収する者
- 危険生物を調査する者

という二つの役割を持つ。

## モンスターとは

モンスターは魔界の住人でも、不死者でもない。**すべて現実の生物を祖先に持つ。**

例: 狼 / 熊 / 鹿 / 蜘蛛 / 蛇 / 鳥 / 昆虫 / 魚 / 爬虫類

長い年月と戦争跡地の特殊環境によって現在の姿になった。

### 変異の原因（地域ごとに異なる）

| 地域 | 変異の原因 |
|---|---|
| 王都跡 | 古代魔術装置の残留エネルギー |
| 白骸墓地 | 大量の死骸による特殊な土壌 |
| 黒鉄鉱山 | 重金属・鉱石による長年の環境変化 |
| 深森 | 人類消失後、生態系が極端に発達 |

これにより地域ごとに全く異なるモンスターが生まれる。

## 探索者の任務

探索者ギルド（`31_SeekersGuild.md`）には二つの依頼がある。

### 遺物探索（Relic Recovery）

武器 / 宝具 / 古代技術。

### 生態調査（Ecology Survey）

危険生物討伐 / 素材採取 / 行動記録 / 危険度評価 / 新種発見。

### なぜ討伐するのか

目的は絶滅ではなく**調査**である。討伐後は **素材回収 → 記録 → 分析 → 図鑑更新** までが任務となる。

---

## ゲームシステムとの接続

```text
探索 → モンスター発見 → 討伐 → 素材回収 → 武器発見
   → 拠点へ帰還 → 武器鑑定 → 素材研究 → 図鑑更新 → 次の探索
```

武器収集とモンスター調査が自然に一つのループへ統合される。

---

## デザインルール

1. 敵はすべて生物由来とする。
2. アンデッド・悪魔・スライムなど異世界起源の種族は基本的に登場させない。
3. 地域ごとに固有の生態系を持たせる。
4. ボスは「その地域の頂点捕食者」として設計する。
5. モンスターの外見・素材・能力は、生態に基づいて説明可能であることを原則とする。
