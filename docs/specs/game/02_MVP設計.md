# 02_MVP設計

## MVPの目的

以下を検証する。

- 5分周回が楽しいか
- 自動探索が退屈ではないか
- 武器ドロップと鑑定が気持ちいいか
- 3ビルドが成立するか

## MVP範囲

### ジョブ

> **世界観一本化（P3-D034 / P3-D038）:** 旧5職案（戦士/騎士/盗賊/狩人/魔術師）は廃止。現行実装は新3職。基本5職の SSOT は `world/09_Jobs.md`（数値は `06_キャラクター_ジョブ.md`）。

- ソードマン（剣 / 前衛）
- レンジャー（弓 / 遠隔）
- アルケミスト（杖 / 魔法+デバフ）

### ビルド

- 状態異常（毒 / 炎上 / 感電 / 冷却 等 — `27_状態異常と属性.md`）
- 属性弱点（5 属性）
- クリティカル

旧 MVP 案（出血・召喚）は Backlog（P3-D024h）。

### ダンジョン

> **世界観一本化（P3-D038）:** 旧3DG（王都跡/白骸墓地/地下工廠）は退役。現行は Postwar Ecology の Biome 制。SSOT は `world/05_Biomes.md`。

- 王都地下モーンゲート（入門 Biome）
- 以降の Biome は順次追加

### 装備枠

- 武器
- 防具
- 装飾品

王遺産は正式版以降。

## 実装しないもの

- PvP
- ギルド
- オンライン要素
- 王遺産
- 上位ジョブ
- 武器継承
- 大規模ストーリー
- 課金

## MVP完成条件

- 3人パーティで出撃できる（メンバー別装備 — P3-D018）
- ダンジョンが自動進行する
- 戦闘が自動で進む
- ~~方針変更が機能する~~ → **Deferred**（P3-D024a — Alpha は準備専用）
- ボス撃破で帰還できる
- 武器がドロップする
- 未鑑定武器を鑑定できる
- 装備変更で強さが変わる
- 発見度が保存される
- 30分以内に初レジェンドを獲得できる
