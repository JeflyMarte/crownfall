# 02_MVP設計

## MVPの目的

以下を検証する。

- 5分周回が楽しいか
- 自動探索が退屈ではないか
- 武器ドロップと鑑定が気持ちいいか
- 3ビルドが成立するか

## MVP範囲

### ジョブ

- 戦士
- 騎士
- 盗賊
- 狩人
- 魔術師

### ビルド

- 出血
- クリティカル
- 召喚

### ダンジョン

- 王都跡
- 白骸墓地
- 地下工廠

### 装備枠

- 武器
- 防具
- 装飾品

王遺産は正式版以降。

## 実装しないもの

- PvP
- ギルド
- オンライン要素
- 王遺産
- 上位ジョブ
- 武器継承
- 大規模ストーリー
- 課金

## MVP完成条件

- 3人パーティで出撃できる
- ダンジョンが自動進行する
- 戦闘が自動で進む
- 方針変更が機能する
- ボス撃破で帰還できる
- 武器がドロップする
- 未鑑定武器を鑑定できる
- 装備変更で強さが変わる
- 発見度が保存される
- 30分以内に初レジェンドを獲得できる
