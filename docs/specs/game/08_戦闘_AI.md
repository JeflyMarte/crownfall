# 08_戦闘・AI

## 戦闘方針

完全自動戦闘。

プレイヤーは攻撃操作をしない。

## ダメージ計算

```text
最終ダメージ =
攻撃力
× スキル倍率
× クリティカル補正
× 状態異常補正
× ボス補正
× その他補正
× 防御軽減
```

## 防御軽減

```text
軽減後倍率 = 100 / (100 + 防御力)
```

## MVP状態異常

### 出血

- 効果時間：5秒
- 攻撃力の20%/秒
- 最大5スタック

## 召喚

- 召喚数上限：3
- 持続時間：20秒
- 攻撃力：召喚者の30〜50%
- HP：召喚者の20〜40%

## AI設定

MVP攻撃対象

- 最寄り
- ボス優先

正式版候補

- エリート優先
- HP最低
- HP最高
- 遠距離敵優先
