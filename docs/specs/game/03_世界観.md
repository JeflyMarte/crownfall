# 03_世界観

> **世界観刷新（2026-06-26 オーナー決定）:** 本作の世界観は **戦後生態系（Postwar Ecology）** を上位方針とする。三本柱は **History / Relics / Ecology**。モンスターは魔物・不死者ではなく**全て実在生物を祖先に持つ**。マスター: `29_PostwarEcology.md`。下記「概要」は History 柱（旧文明史）として保持する。

## 概要

数百年前、大陸では九王戦争と呼ばれる大戦争が起こり、旧文明は崩壊した。

その以前には、**九王**と呼ばれる九人の統治者によって築かれた秩序の時代があり、王国時代へと続いた。九王本人が戦争の当事者だったのか、名を継いだ王国が戦ったのかは、現存史料では断定できない。

現在、各地には滅びた王国、王都、修道院、地下工廠、沈没都市などが遺跡として残っている。

プレイヤーは**探索者（Seeker）**として、探索者ギルドの調査員の一人として旧世界の遺跡と生態系を調査し、伝説の武器や失われた歴史を発掘する。世界を救う選ばれし英雄ではなく、**歴史の空白を埋める無数の継承者**の一人である。組織・役割の詳細は `31_SeekersGuild.md`。

## 時代区分（要約）

| 時代 | 概要 |
|---|---|
| 九王時代 | 記録文明の始点。継承と秩序の礎 |
| 王国時代 | 五大国（K-001〜005）を中心とする繁栄期 |
| 九王戦争 | 王国時代を終焉させた大戦 |
| 静寂の時代 | 戦後の文明縮小期 |
| 探索者の時代 | 現在。遺跡探索文化 |

## 中核の謎

- 九王戦争の真実
- 消された第十の王（第十王遺産の記述）
- 王冠の失墜 — 「倒れたが滅びていない」（Crownfall の語源）
- 伝説武器の由来
- 世界樹は一本ではない — その総数は未解明
