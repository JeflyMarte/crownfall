# 06_Claude運用ルール

## 基本方針

Claude Codeには、必ずTask単位で依頼する。

悪い依頼

```text
このゲームを全部作って
```

良い依頼

```text
Task017〜Task022を実装してください。
既存仕様に従い、EnemyData、UnitController、自動通常攻撃、HP管理、敵全滅時の部屋クリア処理までを実装してください。
```

## 1回の依頼範囲

- 1〜5Taskまで
- 変更ファイル数は10以下
- 大きな設計変更は別依頼

## Claudeへの報告テンプレート

```text
■実装対象
Task番号：

■参照仕様
ファイル：

■実装してほしいこと

■実装しないこと

■完了条件

■完了後に出してほしいもの
- 変更ファイル一覧
- 実装内容
- テスト結果
- 懸念点
```

## レビュー観点

- 仕様から逸脱していないか
- Godot 4構文として正しいか
- Resource中心設計になっているか
- UIとロジックが分離されているか
- マジックナンバーがないか
- 拡張しやすいか
