# 09_Definition of Done

## Task完了条件

各Taskは以下を満たしたら完了。

- 仕様通りに動作する
- エラーが出ない
- 不要な仮実装が残っていない
- 変更ファイルを説明できる
- 既存機能を壊していない

## Phase完了条件

- Phase内Taskが全て完了
- 手動テスト済み
- 仕様書との差分が明記されている
- 次Phaseに進める状態

## MVP完了条件

- 3人パーティで出撃できる
- 3ダンジョンを遊べる
- 自動探索・自動戦闘が成立する
- ~~方針変更が機能する~~ → **Deferred**（P3-D024a — `28_ゲームデザイン点検.md`）
- 武器ドロップがある
- 識別ができる
- 装備更新で強さが変わる
- 発見度が保存される
- 初回30分以内にレジェンドを獲得できる
