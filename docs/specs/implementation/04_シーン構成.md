# 04_シーン構成

## BootScene

役割

- 初期化
- セーブロード
- データロード
- BaseSceneへ遷移

## BaseScene

役割

- 拠点
- ダンジョン選択
- 編成
- 装備
- 鑑定

MVPでは簡易UIでよい。

## DungeonScene

役割

- 自動探索
- 自動戦闘
- 方針変更
- 宝箱
- イベント
- ボス
- 帰還

## ResultScene

役割

- 獲得報酬表示
- 発見度上昇表示
- BaseSceneへ戻る

## UI

HUD

- パーティHP
- ダンジョン名
- 発見度
- 方針ボタン
- 所持品数
- ログ
