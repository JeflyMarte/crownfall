# 04_シーン構成

## BootScene

役割

- 初期化
- セーブロード
- データロード
- BaseSceneへ遷移

## BaseScene

役割

- 拠点ハブ
- Gold・装備武器・パーティ名表示
- DungeonSceneへの導線（ダンジョンへボタン）
- EquipmentSceneへの導線（装備へボタン）

**P2-Task022:** ダンジョン選択 UI（王都跡 / 白骸墓地 placeholder）。`探索開始` で選択 id を確定して DungeonScene へ。

## DungeonScene

役割

- `GameState.get_active_dungeon_id()` で起動ダンジョンを決定（hardcode なし）
- `DungeonController.start_dungeon(dungeon_id)` — DataRegistry 経由で DungeonData ロード
- 部屋単位ステップ進行（**マクロ**。Combat Vision のシームレス戦闘表現は将来層 — `26_CombatVision.md`）
- **戦闘専用シーンへの遷移なし**（Vision 準拠）
- **自動戦闘**（現行: CombatTimer 1.5s 抽象戦闘 → 将来: リアルタイム AI ドット）
- Branch Route 分岐選択（`branch_enabled` ダンジョンのみ）
- 非戦闘部屋: HEAL / TREASURE / MERCHANT / EVENT
- 戦闘 Special Room: ELITE（Phase2-M3）
- 報酬累計（EXP・Gold）
- ランダム武器・防具・装飾品ドロップ（探索終了時）
- ResultSceneへ遷移

### DungeonScene 主要 UI ノード（Phase2-M3）

| ノード | 用途 |
|---|---|
| BranchContainer | 安全 / 危険 / 不明 ルート選択 |
| MerchantContainer | 商人部屋 — 2 商品購入・立ち去る |
| EventContainer | イベント部屋 — 2 択選択 |
| CombatTimer | 自動戦闘ティック |
| LabelRoom / LabelPlayerHp / LabelLog | 状態表示 |

## ResultScene

役割

- EXP・Gold・入手品表示
- Gold加算（ButtonNextタップ時点）
- AppraisalSceneへ遷移

## AppraisalScene（MVP追加）

役割

- 未鑑定アイテム（武器 / 防具 / 装飾品）の一覧表示
- 鑑定処理（100G消費・is_appraised = true）
- 鑑定完了後のログ表示
- EquipmentSceneへの導線
- BaseSceneへの戻り導線（セーブあり）

## EquipmentScene（MVP追加）

役割

- 鑑定済みアイテムの一覧表示
- 武器 / 防具 / 装飾品 装備（GameState 更新・セーブ）
- 現在装備中の表示（**Affix 名・効果** — P2-Task032）
- BaseSceneへの戻り導線（セーブあり）
