# 08_テスト仕様

## MVPテスト観点

### 起動

- BootSceneが正常に起動する
- BaseSceneへ遷移する

### ダンジョン

- DungeonSceneへ遷移する
- 10部屋を進行する
- ボス部屋に到達する
- ResultSceneへ遷移する

### 戦闘

- 味方が自動攻撃する
- 敵が自動攻撃する
- HPが減る
- 死亡する
- 敵全滅で部屋クリアする

### 装備

- 武器を装備できる
- 攻撃力が変わる
- Affixが反映される

### ドロップ

- ボス撃破で武器が出る
- レアリティが付く
- 未鑑定状態で入る

### 鑑定

- 未鑑定武器を鑑定できる
- 名前と能力が表示される

### 発見度

- 探索後に発見度が上がる
- 保存される

## プレイテスト目標値

- 1周4〜6分
- 序盤2〜3周に1回装備更新
- 30分以内にレジェンド獲得
