# 07_コーディングルール

## 命名規則

### Script

PascalCase

```text
GameState.gd
DungeonController.gd
DropManager.gd
```

### 変数

snake_case

```gdscript
current_hp
attack_power
target_enemy
```

### 定数

UPPER_SNAKE_CASE

```gdscript
MAX_PARTY_SIZE
DEFAULT_CRIT_RATE
```

## 禁止事項

- マジックナンバー
- Nodeパス直書き乱用
- 巨大Singleton
- UIから直接ゲームロジック変更
- 1ファイル1000行超え
- 仕様にない機能の勝手な追加

## 推奨

- enum使用
- signal使用
- Resource使用
- Controller分離
- 関数は短くする
- データと表示を分離する
