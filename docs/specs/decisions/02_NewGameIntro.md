# 新規ゲーム導入フロー（P3-INTRO-001）

**Status:** Decision 承認済（2026-07-14 オーナー GO・推奨案）  
**上書き:** Backlog「Future Tutorial」の本編導入部分を本 Decision が定義する。操作講習型チュートリアルは **採用しない**。

---

## 1. 方針

- **最低限の導入** → すぐ世界（拠点）へ解放する
- 操作の細かい説明（属性／状態／CT／陣形／鍛冶／ガチャ）は **導入では教えない**
- 世界観は短いスクロールで空気を渡す。中核の謎の真相は出さない（`world/10_LoreDelivery §9`）
- 案内役は既存NPC **記録官ニーナ**（`world/08` 初心者案内役）

---

## 2. フロー（確定）

```text
Title「はじめから」
  →（セーブ上書き確認：既存どおり）
① 世界観ナレーション（縦スクロール・スキップ可）
② プレイヤー名入力（隊長名）
③ ニーナ紹介（吹き出し最大3）
④ 初期隊員1人選択（既存 StarterPick）
⑤ 拠点 BaseScene へ解放
```

選んだ隊員のみ所持（P3-JOIN-001）。残り基本職は章クリア加入。

**強制ダンジョン突入はしない。**

依存（WIP 可）: `TitleScene` / `StarterPickScene` / 指揮官表示名（TopBar）。

---

## 3. 世界観ナレーション

| 項目 | 値 |
|---|---|
| UI | 暗い背景＋本文縦スクロール。タップ／ドラッグで加速可 |
| 分量 | **6パネル相当**（目安 45〜90 秒） |
| スキップ | **常時表示**（「スキップ」→ 名前入力へ） |
| 下端到達 | 「続ける」で名前入力へ |

### パネル文案（実装用・編集可）

1. 数百年前、大陸は九王戦争で揺れた。王都は崩れ、統一の時代は終わった。
2. 戦後、自然は人と遺構のあいだで新しい均衡を築いた。いま残るのは安全圏の都市と、危険な野外圏だ。
3. 人々は失われた歴史を、断片から拾い集めている。遺跡と記録が、世界を再び知る手がかりになる。
4. 探索者ギルドは調査と記録のために立ち上がった。理念は簡潔だ——「遺産を持ち帰るな、歴史を持ち帰れ」。
5. あなたは調査許可を得たばかりの隊長。新編の調査隊を率いることになる。
6. 最初の地は王都地下モーンゲート。備えを整え、調査を始めよ。
