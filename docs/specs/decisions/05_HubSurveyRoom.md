# 調査室・拠点調査サイクル（P3-HUB-SURVEY-001）

**Status:** Decision 承認済（2026-07-22 オーナー GO）  
**UI 参考:** オーナー提示モック「調査室」（構成・階層・文言トーンの正。**モックの下ナビ案は無視**＝調査室タブ追加はしない。画面には既存の通常 BottomNav を表示する）

---

## 1. 目的

ダンジョン → 拠点（調査室）→ ダンジョンのサイクルを作る。  
主眼は放置経済ではなく、**調査進行による次 Biome 解禁**と**拠点での副回収**。

---

## 2. 三系統（混同禁止）

| ID | 名前 | 役割 |
|---|---|---|
| **SURVEY** | 調査ゲージ（ダンジョン単位・永続） | ②解放の進行。潜行主・調査室配置／時間で副進行 |
| **DISPATCH** | 調査室の調査サイクル | 実時間＋配置で進度し、完了時に報酬受取 |
| **ACHIEVE** | 図鑑実績タブ | 埋め％マイルストーンの一回限り報酬。解放条件に使わない |

---

## 3. 解放（②ウィスパーウッド）

**条件（AND）:**

1. ①モーンゲート **ボス初回討伐**
2. ①の **SURVEY ≥ 70%（調査クリア）**

達成で②を選択可能にする（`BETA_MOURNGATE_ONLY` の恒久封鎖を、本条件付き解禁に置き換える）。

| 表記 | 意味 |
|---|---|
| 調査クリア | SURVEY **70%**（解放ゲート） |
| 完全調査 | SURVEY **100%**（実績／称号／ボーナス用。解放必須にしない） |
| サイクル完了 | 調査室の **当該調査ラン 100%** → 報酬受取可（モックの「必要進度 100%」） |

---

## 4. UI（モック準拠・下ナビ除外）

画面名: **調査室**

拠点からの導線: **左メニュー等の施設入口**（モック下ナビの「調査室」タブは採用しない）。  
画面下部: **既存の通常 BottomNav を他拠点画面と同様に表示**する（タブ構成は現行のまま。調査室専用タブは増やさない）。

### 構成（上から・モック順）

1. **Header** — 戻る／タイトル「調査室」／ヘルプ／通貨
2. **キーアート＋リード文** — 調査室の雰囲気説明
3. **調査対象** — 進行中カード（DG サムネ・名、予想成果アイコン、調査進捗％）、同時枠、空き／拡張枠
4. **調査員の配置** — 速度ボーナス合計、おまかせ配置、担当スロット（変更）、ロック枠
5. **調査の進行状況** — 大ゲージ、完了までの残り時間、％、資料／発見テキスト、資料一覧
6. **調査完了時の期待成果** — 報酬プレビュー（高確率／確定ラベル）＋受取／調査中ボタン

見た目の階層・カード分割・情報密度はモックを守る。トークン／フォントは既存拠点系（鍛冶・図鑑）に合わせる。

---

## 5. DISPATCH（調査サイクル）

| 項目 | 内容 |
|---|---|
| 流れ | DG 選択 → 調査開始 →（時間＋調査員で速度）→ 100% で受取 |
| 時間プリセット | **短**（目安 15–30 分）＋**標準**（目安 2–4 時間）。正確な分数は数値表で後固定 |
| 同時枠（Phase1） | **開放 1**。UI 上は拡張枠をロック表示可（モックの 2/3・魔晶石拡張は **後続**。Phase1 で課金拡張を必須にしない） |
| 調査員 | モックどおり **配置で速度上昇**。Phase1 は最大 **3 スロット**（4枠目は Lv／進行ロックで枠だけ用意してよい）。担当ラベル（考古／地質／文書等）は表示用＋小ボーナス可 |
| 派遣中キャラ | 編成から外す（ロック） |
| 報酬主 | 素材・魔晶石系 |
| 報酬副 | 武器 — **常時低確率 ★1–2**、**稀に ★3**（★4 は出さない） |
| バランス | 潜行より安定・一段下。本命装備は潜行／鍛冶 |
| 次 DG 手がかり | サイクル完了時に **確定フレーバー／SURVEY 加算**（別インベントリの必須アイテムにはしない） |

**SURVEY 加算比率の目安:** 潜行 7–8 割／調査室 2–3 割（数値表で固定）。

## 6. ACHIEVE（図鑑）

- 図鑑内タブ **「実績」**
- 敵図鑑埋め％等のマイルストーン → 石／Gold／チケット等 **一回限り**
- 調査室画面には置かない（図鑑側）

---

## 7. Phase1 スコープ（本実装の初回）

1. 調査室画面（モック構成）＋拠点導線
2. ① SURVEY 永続＋潜行／調査室からの加算
3. 調査サイクル 1 枠・短／標準・調査員配置（速度）
4. 受取報酬（素材・石・武器ルール）
5. 条件達成で②解禁
6. 図鑑「実績」タブ骨格＋マイルストーン数件

**後続（Phase1 に入れない）:** 魔晶石による枠拡張課金、調査室 Lv 本格、②コンテンツ磨き、寄り道調査。
