## 1. 用語定義

| 用語 | 定義 |
|---|---|
| **王遺産（Royal Relic）** | 九王が後世へ託した継承物。 |
| **世界遺産（World Monument）** | 九王以前から存在する超古代構造物。 |
| **既知世界（Known World）** | 記録・往来がある領域。地図に載る部分。 |
| **Biome** | 地域ごとの自然環境・生態系の単位。 |
| **探索者（Seeker）** | 遺物回収と生態調査を担う調査員。 |
| **探索者ギルド（Seekers Guild）** | 探索管理機構（調査機関）。 |

- 表記差異が生じた場合は、本書を基準として統一するか、Decision を経て専門文書を更新する。
- 既存の確定固有名は変更しない。
- 日本語表記を基本とし、必要に応じて英字表記を併記する。
- 漢字名・カタカナ名・英字表記は、本書の登録内容を正規表記とする。
