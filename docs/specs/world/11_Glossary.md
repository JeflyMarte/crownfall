## 1. 用語定義

| 用語 | 定義 |
|---|---|
| **戦後生態系（Postwar Ecology）** | 九王戦争後、数百年をかけて形成された新たな自然環境。本作の上位世界観方針。 |
| **エルダ（Elda）** | エルド大陸に遍在する自然力の総称。物質・生物・遺物・環境に宿り引き出せる。超自然ではなく自然現象として観測される。五属性はエルダの相。詳細 `03_Ecology §6`。 |
| **瘴気（闇のエルダ）** | 腐敗・汚染の気。属性「闇」の正体。状態異常「呪い」もこれによる汚染・衰弱として説明される。 |
| **浄化の光（聖のエルダ）** | 瘴気や異常な個体に作用する力。属性「聖」の正体。特定鉱物・先王時代の浄化技術の名残であり、神の力ではない。 |
| **錬成** | エルダを物質的に扱う技術（触媒・薬品・反応）。アルケミストの「魔法」の正体。杖はエルダを制御する触媒器。 |
| **崩落（the Fall）** | 王都アステリアが失われ王国時代が終わった出来事。歴史の基準点。長期年代の起点（崩落後 A.F.）。 |
| **崩落後（A.F.）** | 崩落を起点とする長期年代尺度。記録散逸で年数は不確定（現在＝崩落後 約300年・諸説）。 |
| **ギルド暦（G.）** | 探索者ギルド創設を起点とする現行の実用暦。1年=12ヶ月=360日、1ヶ月=3旬。**現在 G.118**。詳細 `01_History §7`。 |
| **旬（じゅん）** | ギルド暦の区切り。10日＝1旬、3旬＝1ヶ月（王国時代の「週」に相当）。 |
| **澱みの季（よどみのとき）** | 晩秋〜初冬、地下・遺構で闇のエルダ（瘴気）の活性が高まる危険期。生物が活発化する。 |
| **九王戦争** | 王国時代を終わらせた大戦。真相は失われている。 |
| **王冠（The Crown）** | 唯一の王権の証として伝わるもの。作品名 Crownfall の語源。「倒れたが滅びていない」と語られる。 |
| **王遺産（Royal Relic）** | 九王が後世へ託した継承物。九対の伝説武器と対をなす。 |
| **第十王遺産** | 古い目録に一度だけ現れる記述。最終行に「これは九王の遺産ではない」と残る。 |
| **先王時代（Pre-Nine）** | 九王時代より前の起源不明の超古代（伝承）。世界遺産を遺したとされる。 |
| **世界遺産（World Monument）** | 先王時代に由来するとされる起源不明の超古代構造物。 |
| **白王（the White King）** | 九王以前に「継承」を司ったとされる伝説的存在（伝承）。白王の神殿が残る。 |
| **灯火** | 古い物語に繰り返し現れる、誰も消させなかった火。真相は語られない。 |
| **灯火の信仰** | 記憶と継承を絶やさないことの象徴として灯を守る民間信仰。教義でなく暮らしの作法。象徴的人物＝巫女イリア。 |
| **巫女** | 神に仕える聖職者ではなく、灯と記憶を守り伝える者の呼び名。 |
| **継承の信仰** | 白王にまつわる最古の信仰の跡（白王の神殿）。現在は迷信視されるが「受け継ぐことを尊ぶ」思想は残る。 |
| **鉱物化適応** | 遺物・鉱物を体内へ取り込み、結晶・金属・古代文字状の組織を形成する生態適応（モーンゲートの特徴）。 |
| **共生適応** | 生物が菌糸・胞子・蔓・苔と共生／寄生して形質を得る生態適応（囁きの森の特徴）。鉱物化適応（無機）と対をなす有機の適応。 |
| **伝説個体（Legendary）** | 各地で語り継がれる極めて稀な超大型・古代の生物。多くは Class VII 古代種で土地の象徴。神でも不死でもなく生物。例: セルディオン／フローラベア・グランヴェル。 |
| **古龍種（Elder Reptiles）** | おとぎ話の竜ではなく、王国時代から姿を変えず生き延びた大型飛行爬虫類の総称。生体は極めて稀。セルディオンの骨格の由来。カイル盟約／ビーストテイマーの主題。 |
| **既知世界（Known World）** | 記録・往来がある領域。地図に載る部分。 |
| **共通語** | 各都市で通じる言語。王国時代の交流圏の名残。ギルドが正書法を整える。 |
| **現代文字** | 現在の文字。ギルド記録・図鑑・契約に用いる。王国文字が崩れて成立。 |
| **王国文字** | 王国時代の正書。多くの碑文・古文書に残る。記録部はおおむね解読可。 |
| **古代文字（Elder Script）** | 世界遺産・最古の遺構に刻まれた未解読の文字。文字か否かも不明。「ルーン」と俗称される生物の紋様はこれを取り込んだ模様で魔法ではない。 |
| **安全圏** | 城壁に囲まれた都市・集落の内側。人々が暮らす生存圏。 |
| **野外圏** | 城壁の外。戦後生態系に覆われ危険生物と遺構が眠る領域。探索の舞台。 |
| **隊商（交易隊）** | 護衛を伴い都市間を結ぶ交易隊。王の大街道の名残を利用する。 |
| **ゴールド（Gold）** | 旧王国の金貨。統一王権喪失後も金属価値で流通する事実上の基軸通貨。ギルド・隊商が信用を支える。 |
| **生物素材** | 戦後生態系の生物から採取する素材（牙・甲殻・晶針 等）。生態調査の産物。 |
| **遺物素材** | 遺跡・遺物から回収する素材（遺跡の欠片・古き骨・時環核 等）。遺物回収の産物。 |
| **「魔○○」素材** | エルダを帯びた素材に対する民間の俗称（魔導触角・褪色した魔毛 等）。超自然ではない。 |
| **目利き（めきき）** | 遺物・素材の素性と真贋を見極める営み（ギルド回収部・商人）。 |
| **Biome** | 地域ごとの自然環境・生態系の単位。 |
| **探索者（Seeker）** | 遺物回収と生態調査を担う調査員。プレイヤーが率いる立場。 |
| **探索者ギルド（Seekers Guild）** | 探索管理機構（調査機関）。 |
| **ギルド評議会（Guild Council）** | ギルドの最高意思決定機関。加盟都市代表＋分野長の合議制。命令でなく調整・承認を担う。 |
| **五部門** | 回収部／調査部／測量部／記録部／補給部。三大任務に対応する3部門＋記録・補給。 |
| **上級探索者（Senior Seeker）** | 一専門を究めた者への上級資格。ジョブ到達形（ソードセイバー等）に対応（P3-D048）。 |
| **調査許可等級** | 調査隊・隊長（プレイヤー）に与えられるギルド評価。調査成果で上昇し危険区域・上位任務・補給枠を解放（数値は game 仕様）。 |
| **赤鉄の工房** | アイアンヘイブンの鍛冶場。廃都レッドフォージの鍛冶（鍛冶王〜ラグナ）の系譜を継ぐ。鍛冶機能の拠点。 |
| **辻灯亭（つじびてい）** | アイアンヘイブンの宿場・酒場。探索者の合流・勧誘の場（助っ人・編成導線）。 |
| **境界守（きょうかいもり）** | 守護王ヴァルケインが組織したと伝わる、安全圏と野外圏の境を見張る守人の系譜。現代の前衛職（ヴァンガード／パラディン）の精神的源流とされる。 |
| **封緘書庫（ふうかんしょこ）** | 王立図書院の最奥にあったと伝わる、学識王セラディスが知識を封じた書庫。何を封じたかは不明（公開は問いのみ）。 |
| **炉印（ろじるし）** | 鍛冶王オルグランの星炉で鍛えた品に宿る同一紋様の俗称。九対の王遺産が同じ紋様を持つ理由は説明されていない。 |
| **翠の盟約（みどりのめいやく）** | 森と人が互いを侵さないと交わした、森の盟約国（森護王シルヴァリア）の根本の約定。 |
| **アステリア陥落録（あすてりあかんらくろく）** | 記録者セレス（九英雄）による、王都アステリア陥落の数少ない記録の通称。欠落・破損が多く全容は不明。記録部の最重要史料。 |
