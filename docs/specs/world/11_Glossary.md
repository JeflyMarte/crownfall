## 1. 用語定義

| 用語 | 定義 |
|---|---|
| **戦後生態系（Postwar Ecology）** | 九王戦争後、数百年をかけて形成された新たな自然環境。本作の上位世界観方針。 |
| **九王戦争** | 王国時代を終わらせた大戦。真相は失われている。 |
| **王冠（The Crown）** | 唯一の王権の証として伝わるもの。作品名 Crownfall の語源。「倒れたが滅びていない」と語られる。 |
| **王遺産（Royal Relic）** | 九王が後世へ託した継承物。九対の伝説武器と対をなす。 |
| **第十王遺産** | 古い目録に一度だけ現れる記述。最終行に「これは九王の遺産ではない」と残る。 |
| **先王時代（Pre-Nine）** | 九王時代より前の起源不明の超古代（伝承）。世界遺産を遺したとされる。 |
| **世界遺産（World Monument）** | 先王時代に由来するとされる起源不明の超古代構造物。 |
| **白王（the White King）** | 九王以前に「継承」を司ったとされる伝説的存在（伝承）。白王の神殿が残る。 |
| **灯火** | 古い物語に繰り返し現れる、誰も消させなかった火。真相は語られない。 |
| **鉱物化適応** | 遺物・鉱物を体内へ取り込み、結晶・金属・古代文字状の組織を形成する生態適応（モーンゲートの特徴）。 |
| **既知世界（Known World）** | 記録・往来がある領域。地図に載る部分。 |
| **Biome** | 地域ごとの自然環境・生態系の単位。 |
| **探索者（Seeker）** | 遺物回収と生態調査を担う調査員。プレイヤーが率いる立場。 |
| **探索者ギルド（Seekers Guild）** | 探索管理機構（調査機関）。 |
