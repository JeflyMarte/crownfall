## アウトライン（ChatGPT 執筆）

1. 基本方針（魔物ではなく生物学的分類）
2. クラス一覧（Class I〜VII）
3. 地域による派生（同祖先・環境で分化）
4. 図鑑との接続（概要）

## 必ず保持する正典（Canon — 削除不可）

- **7 クラス:** I 獣類 / II 鳥類 / III 爬虫類 / IV 昆虫類 / V 水棲類 / VI 菌・植物類 / VII 古代種。
- VII 古代種は王国時代から姿を変えず生き残った希少種（地域ボス候補）。古龍種は「ドラゴン」ではなく大型飛行爬虫類。
- 同じ祖先でも環境で姿が変わる（例: 狼系統が地域ごとに灰狼/白骨狼/鉄牙狼/森影狼/氷牙狼）。
