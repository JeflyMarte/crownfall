## 1. 基本方針（魔物ではなく生物学的分類）

探索者ギルドでは、調査対象となる生物を「魔物」としてではなく、生物学的な共通性に基づいて分類している。

この分類は戦闘のためだけではなく、生態調査、素材研究、環境保全、記録の統一を目的として整備されたものである。

分類の基準となるのは、生物の祖先、身体構造、生態、環境への適応であり、外見だけで区別されることはない。

同じ祖先を持つ生物でも、異なる土地で長い年月を過ごすことで大きく姿を変えることがある。

そのため、生物種と地域環境の両方を記録することが、探索者には求められている。

## 2. クラス一覧（Class I〜VII）

### Class I：獣類（Beasts）

哺乳類を祖先とする大型・中型生物。

**例**

- 灰狼
- 鎧熊
- 大牙猪
- 岩山羊
- 巨角鹿

**特徴**

- 高い身体能力
- 毛皮・牙・骨素材を持つ
- 群れを形成する種が多い

### Class II：鳥類（Avians）

鳥類を祖先とする飛行・滑空生物。

**例**

- 巨鴉
- 崖鷲
- 灰梟
- 鉄嘴鳥

**特徴**

- 高い機動力
- 羽根素材を得られる
- 索敵能力に優れる

### Class III：爬虫類（Reptiles）

爬虫類を祖先とする生物。

**例**

- 岩蜥蜴
- 毒蛇
- 沼竜
- 火蜥蜴

**特徴**

- 高い耐久力
- 毒を持つ種が多い
- 鱗素材を得られる

### Class IV：昆虫類（Insects）

昆虫類および節足動物を祖先とする生物。

**例**

- 巨甲虫
- 毒蜘蛛
- 刃蟷螂
- 結晶蜂

**特徴**

- 強固な外殻
- 毒や特殊分泌物を持つ
- 群体行動を行う種が多い

### Class V：水棲類（Aquatics）

河川・湖・湿地・海域などを主な生息域とする生物。

**例**

- 巨鮫
- 沼蛙
- 深淵鰻
- 河蟹

**特徴**

- 水域への高い適応力
- 粘液や鱗素材を持つ
- 水辺の生態系を支配する種が多い

### Class VI：菌・植物類（Flora）

菌類および植物を起源とする生物。

植物もまた生物であり、本分類体系では独立した区分として扱われる。

**例**

- 捕食花
- 棘蔦
- 毒胞子茸

**特徴**

- 移動能力は低い
- 状態異常を引き起こす種が多い
- 薬草素材や胞子などを得られる

### Class VII：古代種（Ancients）

王国時代から大きく姿を変えず、生き残ったと考えられる希少種。

個体数は極めて少なく、多くの地域で生態系の頂点に位置している。

**例**

- 古龍種（ドラゴンではなく大型飛行爬虫類）
- 王角鹿
- 深森巨熊
- 古代の鉱物性共生生物（鉱物へ擬態する寄生性結晶生命。単独では希少だが、大型生物の死骸へ群体化して複合生命体を形成する）
- 水晶骸竜 セルディオン（古龍種の骨格 ＋ 鉱物性共生生物の複合体。モーンゲートの頂点捕食者。不死者ではない）

**特徴**

- 生息数が極端に少ない
- 地域ボスとして語られることが多い
- 高品質な素材を有する
- 複合・共生によって成立する複雑な個体が含まれる

## 3. 地域による派生（同祖先・環境で分化）

分類は祖先を示すものであり、姿を固定するものではない。

同じ祖先を持つ生物でも、異なる環境で長い年月を過ごすことで、それぞれの土地に適応した姿へ分化していく。

現在唯一の確定 Biome である **モーンゲート（嘆きの地下水路）** は、堆積した遺物・鉱物を生物が体内へ取り込む **鉱物化適応** が顕著な土地であり、各分類群がこの環境に応じて分化している。

| 共通祖先 | Class | 地域 | 派生個体群 | 分化の特徴 |
|---|---|---|---|---|
| ハリネズミ | I 獣類 | モーンゲート | 水晶ハリネズミ | 針が鉱物を吸って結晶化 |
| ネズミ | I 獣類 | モーンゲート | 冠喰いネズミ | 宝物庫跡に適応した鉱物食性 |
| ゴキブリ | IV 昆虫類 | モーンゲート | ルーンローチ | 甲殻に古代文字状の結晶模様が析出 |
| 蛾 | IV 昆虫類 | モーンゲート | クロックモス | 鱗粉が金属粉化し歯車状の翅紋を形成 |

これらはいずれも実在生物を共通祖先に持ちながら、モーンゲートの環境へ適応した地域個体群として扱われる。

次の確定 Biome である **囁きの森（ウィスパーウッド／ヴェルディア大森林）** は、生物が菌糸・胞子・蔓・苔と共生／寄生して形質を得る **共生適応** が顕著な土地であり、モーンゲートの鉱物化適応（無機）と対をなす（有機）。

| 共通祖先 | Class | 地域 | 派生個体群（例） | 分化の特徴 |
|---|---|---|---|---|
| 食虫植物・菌類 | VI 菌植物 | 囁きの森 | ブラッドブルーム／ランタンキャップ／モールドアイ 等 | 発光・捕食・寄生胞子など共生網の中核へ分化 |
| 甲虫 | IV 昆虫 | 囁きの森 | アイアンホーン／モスシェル／ルーンスカー 等 | 苔・古代文字状紋様をまとい林床へ擬態 |
| 猪 | I 獣 | 囁きの森 | モスボア／ソーンボア／ブライアボア 等 | 背に棘植物・蔦が共生して生える |
| 蛇 | III 爬虫 | 囁きの森 | ブルームサーペント／ヴァインバイパー／スポアコブラ 等 | 花弁状の鱗・蔦擬態・胞子噴射へ分化 |
| 蟹／両生 | V 水棲 | 囁きの森 | ルーン・カルキノス／エコー・マンダー | 林床の水辺で大型化（Elite 候補） |
| 巨熊（古代種） | VII 古代種 | 囁きの森 | フローラベア・グランヴェル | 森の共生フローラと一体化した頂点捕食者 |

これらはいずれも実在生物を共通祖先に持ちながら、囁きの森の環境へ適応した地域個体群として扱われる（詳細は `05_Biomes §3.2`）。

このように、同じ祖先からでも Biome が異なれば異なる分化が起こる。環境が異なれば、生態も外見も大きく変化する。

## 4. 図鑑との接続（概要）

探索者ギルドの図鑑では、各生物について戦闘だけではない多面的な調査記録が蓄積される。

記録は分類体系を共通言語として整理され、異なる地域の生物同士を比較するための基盤となっている。

調査記録は主に次の区分で整理される。

### 基本情報

- 分類
- 危険度
- 生息地
- 行動時間
- 群れ規模

### 調査情報

- 食性
- 縄張り
- 天敵
- 特徴

### 素材情報

- 採取素材
- 武器素材
- 防具素材
- 希少素材

分類は、生物を名前で管理するためではなく、その生態を理解するための入口である。

探索者は分類を起点として、土地ごとの環境、生態系、そして失われた世界とのつながりを記録していく。

## 5. 伝説個体（Legendary）と古龍種

### 伝説個体とは

各地には、通常の生態系の個体とは別格の、極めて稀な超大型・古代の生物が語り継がれている。探索者ギルドはこれらを **伝説個体（Legendary）** として特別に記録する。

その多くは Class VII 古代種に属し、土地そのものを象徴する存在として、しばしば地域の頂点に立つ。すでに確認されているもの（モーンゲートの **セルディオン**、囁きの森の **フローラベア・グランヴェル** など）もあれば、伝承の中にしか現れず実在が確かめられていないものもある。

重要なのは、伝説個体もまた **生物である** という点である。神でも不死者でもなく、巨大で、長命で、極端に数が少ないだけの生命である。その力や異形も、生態と適応の結果として説明される。

### 古龍種（Elder Reptiles）

伝説個体の代表が **古龍種** である。

古龍種は、おとぎ話の竜ではなく、王国時代からほとんど姿を変えずに生き延びたと考えられる **大型の飛行爬虫類** の総称である。個体数は極めて少なく、生体が確認される機会はまれである。

モーンゲートのセルディオンの骨格は、この古龍種の死骸に由来する。骨格となった個体とは別に、いまも生きた古龍種が辺境のどこかに残っていると伝えられているが、確かな記録は少ない。

### カイルの盟約とビーストテイマー

九英雄の一人 **竜と盟約を結んだ者 カイル**（`01_History §4`）は、「竜」と呼ばれた巨大生物と、戦うのでも従えるのでもない距離を見出したと伝わる。

この思想は、現在の **ビーストテイマー／ビーストロード**（`09_Jobs`）へと受け継がれている。彼らは生物を力で屈服させるのではなく、生態を理解し、共に在ることを目指す。伝説個体や古龍種との関わりは、この職にとって象徴的な主題である。

### 設計枠（伝説個体スロット）

各 Biome・各地方には、土地を象徴する伝説個体を最大1体程度配置しうる（頂点捕食者を兼ねることが多い）。

- 既定: モーンゲート＝セルディオン / 囁きの森＝フローラベア・グランヴェル。
- 他地方の伝説個体（古龍種を含む）は、Biome 設計時に **祖先・Class・生態をHQが設計し、名称はオーナーが確定する**（モンスター命名の方針に準拠）。

### 伝承と実在（ロア方針）

伝説個体の中には、実在が確認されないまま語られ続けるものもある。

特に世界樹や灯火など **永久に未確定とする主題**（`10_LoreDelivery §9`、P3-D043）に結びつく伝説には、答えを用意しない。それらは確かめられないまま、問いとして世界に残り続ける。
