## 1. Biome の概念（ダンジョンではなく生態系）

この世界における **Biome** は、攻略対象となるダンジョンではない。

一つの環境、一つの生態系、一つの循環として成立した自然のまとまりである。

そこには地形、水、植物、生物、遺構が互いに影響し合い、独自の均衡が築かれている。

探索者は遺跡を制圧するためではなく、その生態系へ踏み込み、観察し、記録し、失われた歴史と現在の自然とのつながりを調査するために訪れる。

遺構は過去の文明を語り、生物は現在の環境を語る。

Biome は、その両者が重なり合う場所である。

## 3. Biome-01 嘆きの地下水路（モーンゲート）詳細

### 世界設定

**嘆きの地下水路**は、地理上の名称である**モーンゲート（Mourngate）**に存在する王都地下の巨大排水網を指す、生態系としての呼称である。

九王戦争以前、この地下施設は王都全体を支える高度な排水設備として築かれた。

文明が失われると維持管理は途絶え、水は静かに流れ続けた。

数百年の時間は地下施設を湿潤な洞窟へ変え、植物、菌類、小動物、捕食者が入り込み、一つの巨大な地下生態系を形成した。

王都由来の遺物が数多く残されていることから、多くの探索隊が最初に調査へ向かう土地として知られている。

難易度は入門であり、探索者が戦後生態系を初めて理解する場所でもある。

### 環境

- 地下
- 湿潤
- 暗所
- 石造建築
- 崩落区画
- 地下河川

人工建築と自然環境が長い年月をかけて一体化し、現在では境界を区別することは難しい。

### 食物連鎖（概要）

```text
地下植物・菌類
        │
        ▼
冠喰いネズミ
        │
        ├─────────────┐
        ▼             ▼
ルーンローチ     水晶ハリネズミ
        │             │
        └──────┬──────┘
               ▼
        セピアハウンド
               │
               ▼
      クロックモス（Elite）
               │
               ▼
水晶骸竜 セルディオン（Boss）
```

この連鎖は固定されたものではなく、環境や季節によって変化する。

**水晶骸竜 セルディオン**は、この地下生態系における頂点捕食者として位置付けられる。

その骨格のような外見は、水晶が長い年月をかけて大型生物へ寄生・共生した結果と考えられており、不死者やアンデッドではない。

### 確定モンスター

| 名称 | 概要 |
|---|---|
| **セピアハウンド（sepia_hound）** | 古い記憶を嗅ぎ分ける魔犬。地下水路を高速で巡回する。 |
| **ルーンローチ（rune_roach）** | 甲殻に古代文字が刻まれた巨大蟲。硬い外殻を持つ。 |
| **水晶ハリネズミ（crystal_hedgehog）** | 水晶化した針を一斉に飛ばして身を守る遠隔型生物。 |
| **冠喰いネズミ（crown_eater_rat）** | 王冠や宝石など鉱物質を選んで齧る地下性の害獣。 |
| **クロックモス（clock_moth）【Elite】** | 歯車状の羽を持つ大型の蛾。羽音によって時間感覚を狂わせる。 |
| **水晶骸竜 セルディオン（serdion）【Boss】** | 水晶が寄生した骨格を持つ大型生物。古代種に近い性質を示す頂点捕食者であり、不死者ではない。 |

### 遺物

この Biome では、王都に由来する生活や施設の痕跡が数多く残されている。

- 王国武器
- 地下施設の工具
- 王都設備
- 宝具

遺物は地下施設がかつて人々の暮らしを支えていたことを静かに物語っている。

### 主素材

調査や採取によって確認される代表的な素材。

- 毒腺
- 外殻
- 湿皮
- 古代金属
- 水晶片

## 4. 共通設計ルール

すべての Biome は、次の原則に従って設計される。

- **すべての生物は実在生物を祖先とする。**
- **生態が能力を決定する。**
- **環境 → 生態系 → モンスター の順で設計する。**
- **遺物は過去の歴史を語り、生態系は現在の自然を語る。**

Biome は敵を配置する場所ではなく、生態系を表現する単位である。

---

## 5. 新 Biome 追加テンプレート

新たな Biome を追加する際は、次の様式を用いる。

```markdown
# Biome-XX：名称（地理名）

## 状態
- 実装 / Future

## 世界設定

## 環境

## 食物連鎖

## 生息モンスター

## 頂点捕食者

## 遺物

## 主素材

## 調査メモ
```

すべての項目は、生態系を起点として記述する。

個別モンスターや遺物は、その環境の結果として配置される。
