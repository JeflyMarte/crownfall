## この文書の責務

地域ごとの生態系（Biome）の各論と、新 Biome 追加の雛形を定義する。分類は `04`、地理上の位置は `07_Geography`、実装ロスター・数値は `game/12_モンスター.md` に委譲する。

## アウトライン（ChatGPT 執筆）

1. Biome の概念（ダンジョンではなく生態系）
2. Biome 一覧（MVP / Future）
3. 各 Biome 詳細（世界設定 / 環境 / 食物連鎖 / 主なモンスター / 遺物 / 主素材）
4. 共通設計ルール
5. 新 Biome 追加テンプレート

## 必ず保持する正典（Canon — 削除不可）

- Biome は「一つの生態系」として設計する。探索者は遺跡を攻略するのではなく生態系へ踏み込み調査する。
- MVP Biome: **01 嘆きの地下水路（モーンゲート / 王都地下・実装済 difficulty 1）** / **02 囁きの森（ウィスパーウッド / ヴェルディア）**。
- モーンゲート実装モンスター（オーナー確定 6 体）: セピアハウンド / ルーンローチ / 水晶ハリネズミ / 冠喰いネズミ / クロックモス[Elite] / 水晶骸竜セルディオン[Boss]（詳細は `game/12_モンスター.md`）。
- 共通ルール: ①全て生物由来 ②生態が能力を決める ③環境→生態系→モンスターの順 ④遺物は歴史を語る。
- Biome 名（嘆きの地下水路）と地理名（モーンゲート）は同一地点の別レイヤー呼称。
