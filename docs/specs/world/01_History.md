## 1. 時代区分（年表）

世界の歴史は、大きく五つの時代に分けて語られる。

正確な年代は多くが失われ、現代では遺跡や碑文、断片的な記録をもとにした復元が行われている。

| 時代 | 概要 |
|---|---|
| **九王時代** | 九人の王がそれぞれ異なる理念を掲げ、世界各地で文明を築いた時代。交流と発展が続き、多様な知識と技術が花開いた。 |
| **王国時代** | 九王の思想を継いだ諸王国が広く栄えた時代。街道や港、市場、学術機関が整備され、人々の営みは最も豊かだったと伝えられる。 |
| **九王戦争** | 王国時代を終焉へ導いた大戦。多くの都市と王国が失われ、文明は崩壊した。 |
| **静寂の時代** | 人類文明が衰退した後の長い空白。自然は遺構を覆い、新たな生態系が各地で形成された。 |
| **探索者の時代** | 現在。探索者たちは遺物回収と生態調査を通じて、失われた歴史を少しずつ記録し直している。 |

> **History 基幹エントリ**
>
> - HE-001：王国時代
> - HE-002：九王戦争
> - HE-003：静寂の時代
> - HE-004：探索者の時代
>
> これら四つは生態図鑑の History カテゴリにおける基幹エントリとして扱われる。

## 2. 九王戦争（統一文案・諸説）

九王戦争は、王国時代を終わらせた大戦として知られている。

しかし、その実像は現代までほとんど伝わっていない。

ある地域では九人の王自身が最後の戦いを繰り広げたという。

別の地域では、九王の名を継いだ諸王国同士の戦争であったと語られる。

さらに、戦争そのものよりも、その後に訪れた文明の急速な崩壊こそが本当の災厄だったという伝承も残されている。

一致しているのは、戦争の後に王国時代は終わり、人類文明は急速に姿を消したという事実だけである。

戦争の真相は今も失われたままであり、探索者が各地で見つける記録も互いに食い違うことが少なくない。

## 3. 九王（Nine Kings）

九王は、それぞれ異なる理念を掲げ、人類文明の礎を築いたと伝えられる。

現代では英雄というより、一つの時代を象徴する存在として語られることが多い。

| 称号 | 名 | フック |
|---|---|---|
| 開拓王 | エルディオン（Eldion） | 既知世界の外縁を記録した王。未知へ道を開いた。 |
| 守護王 | ヴァルケイン（Valkein） | 王盾と境界の思想を築き、守ることの意味を示した。 |
| 学識王 | セラディス（Seradis） | 大書庫を築き、知識を集め、封印した王。 |
| 信義王 | アウレクス（Aurex） | 約束と盟約を国家の礎とした王。 |
| 巡礼王 | ルミナス（Luminas） | 街道網の整備を進め、人と文化を結びつけた王。 |
| 鍛冶王 | オルグラン（Orgran） | 星炉を築き、数多くの王遺産を鍛造したと伝わる。 |
| 海統王 | ネレイオス（Nereios） | 潮境を治め、海路を統治した王。 |
| 森護王 | シルヴァリア（Sylvaria） | 森林と生命循環を守り、世界樹を敬った王。 |
| 継承王 | カエルム（Caelum） | 「次の者へ」という思想を遺した最後の継承者として語られる。 |

九王にまつわる逸話は地域ごとに姿を変え、史実と伝承を明確に区別することは難しい。

## 4. 九英雄（Nine Heroes）

九英雄は王ではなく、人々の営みを支えた人物として各地の伝承に残る。

その功績は戦いだけでなく、探究、記録、鍛造、航海、継承など多岐にわたる。

| 称号 | 名 | フック |
|---|---|---|
| 地平の探求者 | アステル | 地図の余白を埋め、未知を既知へ変えた探求者。 |
| 王遺産の守り手 | エレナス | 王遺産を継承物として扱う倫理を示した人物。 |
| 深淵を踏破した者 | リュシアン | 深淵を恐怖ではなく知識の対象として記録した。 |
| 灯火を継いだ巫女 | イリア | 戦乱の中でも灯火と祈りを絶やさなかった。 |
| 星屑の鍛冶師 | ラグナ | 星屑鍛造の基礎を築いた鍛冶師。 |
| 失われた王都の記録者 | セレス | 崩壊する王都を最後まで書き留めた記録者。 |
| 竜と盟約を結んだ者 | カイル | 「竜」と呼ばれた巨大生物との距離を見出したと伝えられる。 |
| 世界の果てを見た航海者 | マレク | 海の彼方へ航海し、帰還後に多くの記録を残した。 |
| 名を残さなかった継承者 | （無名） | 名ではなく意思だけを後世へ託したとされる存在。 |

九英雄もまた、九王と同じく史実と伝承が重なり合っている。

彼らが実在したかどうかよりも、その思想が各時代へ受け継がれたことが重要視されている。

## 5. 各時代の暮らし・遺構（プレイヤーが遺跡で出会う痕跡）

探索者が目にする遺跡は、戦争そのものではなく、人々が確かに暮らしていた痕跡である。

**九王時代**の遺構には、巨大な石碑や境界標、初期の街道跡が残されている。

**王国時代**では、市場跡、農地、水路、港、工房、書庫など、人々の日常を感じさせる建造物が多く見つかる。

**九王戦争**の痕跡として残るのは、崩壊した城壁、焼失した街並み、放棄された砦などであり、その原因を語る記録は少ない。

**静寂の時代**には、人の手を離れた建造物へ植物が根を張り、野生動物が棲み着き、遺構は自然の一部となった。

そして**探索者の時代**。

新たに立てられる標識や調査記録は、未来の誰かに向けた新しい歴史の断片となっていく。

探索者が持ち帰る記録は、小さな発見に過ぎない。

だが、その積み重ねこそが、失われた文明を再び一つの歴史として結び直していく。
