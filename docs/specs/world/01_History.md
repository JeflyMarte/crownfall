## 1. 時代区分（年表）

世界の歴史は、大きく五つの時代に分けて語られる。

正確な年代は多くが失われ、現代では遺跡や碑文、断片的な記録をもとにした復元が行われている。

| 時代 | 概要 |
|---|---|
| **先王時代（伝承）** | 九王時代より前にあったとされる、起源不明の超古代。各地に「世界遺産」と呼ばれる巨大構造物を遺したと伝わるが、確かな記録はほとんど残っていない。九王時代との境界も定かではない。 |
| **九王時代** | 九人の王がそれぞれ異なる理念を掲げ、世界各地で文明を築いた時代。交流と発展が続き、多様な知識と技術が花開いた。 |
| **王国時代** | 九王の思想を継いだ諸王国が広く栄えた時代。街道や港、市場、学術機関が整備され、人々の営みは最も豊かだったと伝えられる。 |
| **九王戦争** | 王国時代を終焉へ導いた大戦。多くの都市と王国が失われ、文明は崩壊した。 |
| **静寂の時代** | 人類文明が衰退した後の長い空白。自然は遺構を覆い、新たな生態系が各地で形成された。 |
| **探索者の時代** | 現在。探索者たちは遺物回収と生態調査を通じて、失われた歴史を少しずつ記録し直している。 |

## 1.5. 先王時代と世界遺産（伝承）

九王時代より前にも、何者かが世界に痕跡を残していた。

各地に点在する起源不明の巨大構造物は **世界遺産（World Monument）** と呼ばれ、九王時代の建造物とは技術も様式も異なる。誰が、いつ、何のために築いたのかは分かっていない。

古い伝承には、九王より前に「継承」を司ったとされる **白王（the White King）** の名が断片的に現れる。`白王の神殿` はその信仰の跡とされるが、白王が実在したのか、何者だったのかは確かめようがない。

先王時代は、記録ではなく伝承の中にだけ残された、最も古い空白である。

## 2. 九王戦争（統一文案・諸説）

九王戦争は、王国時代を終わらせた大戦として知られている。

しかし、その実像は現代までほとんど伝わっていない。

ある地域では九人の王自身が最後の戦いを繰り広げたという。

別の地域では、九王の名を継いだ諸王国同士の戦争であったと語られる。

さらに、戦争そのものよりも、その後に訪れた文明の急速な崩壊こそが本当の災厄だったという伝承も残されている。

一致しているのは、戦争の後に王国時代は終わり、人類文明は急速に姿を消したという事実だけである。

戦争の真相は今も失われたままであり、探索者が各地で見つける記録も互いに食い違うことが少なくない。

### 経過（諸説の骨格）

確かな年表は残っていないが、各地の伝承を重ねると、戦争はおおよそ三つの段階を経たと考えられている。

**発端（諸説）** — 何が戦争を引き起こしたのかについては、複数の説が伝わる。

- 王都に伝わる「継承」を巡って諸王が対立したという説。
- 国境と資源を巡る、ありふれた領土争いだったという説。
- 戦争そのものよりも、同時期に起きた未知の災厄が引き金だったという説。

いずれの説も決め手を欠き、地域によって語られる原因は大きく異なる。

**拡大** — 一度始まった争いは、諸王国を次々と巻き込んでいった。中央の王都アステリアが担っていた諸国の盟主としての機能は急速に失われ、各地は孤立した。街道は寸断され、王国時代を支えた交流網は崩れていった。

**終息と崩壊** — 戦争がどのように終わったのかも定かではない。勝者を記す記録は存在しない。むしろ多くの伝承は、戦争の後に訪れた **急速な文明崩壊** こそが本当の災厄だったと語る。人口は激減し、技術と知識は維持する者を失い、静寂の時代が始まった。

## 3. 九王（Nine Kings）

九王は、それぞれ異なる理念を掲げ、人類文明の礎を築いたと伝えられる。

現代では英雄というより、一つの時代を象徴する存在として語られることが多い。

| 称号 | 名 | フック |
|---|---|---|
| 開拓王 | エルディオン（Eldion） | 既知世界の外縁を記録した王。未知へ道を開いた。 |
| 守護王 | ヴァルケイン（Valkein） | 王盾と境界の思想を築き、守ることの意味を示した。 |
| 学識王 | セラディス（Seradis） | 大書庫を築き、知識を集め、封印した王。 |
| 信義王 | アウレクス（Aurex） | 約束と盟約を国家の礎とした王。 |
| 巡礼王 | ルミナス（Luminas） | 街道網の整備を進め、人と文化を結びつけた王。 |
| 鍛冶王 | オルグラン（Orgran） | 星炉を築き、数多くの王遺産を鍛造したと伝わる。 |
| 海統王 | ネレイオス（Nereios） | 潮境を治め、海路を統治した王。 |
| 森護王 | シルヴァリア（Sylvaria） | 森林と生命循環を守り、世界樹を敬った王。 |
| 継承王 | カエルム（Caelum） | 「次の者へ」という思想を遺した最後の継承者として語られる。 |

九王にまつわる逸話は地域ごとに姿を変え、史実と伝承を明確に区別することは難しい。

## 3.5. 王国時代の諸王国

王国時代は、九王の思想を継いだ諸王国が各地に栄えた時代である。

中央の王都アステリアがゆるやかな盟主として諸国を結び、それぞれの王国が異なる理念と産業を担っていた。以下は、現在までに復元された主要な諸王国の概略である（地理は `07_Geography`）。

| 諸王国（通称） | 継ぐ理念・王 | 地方 | 現在の状態 |
|---|---|---|---|
| 中央王統（アステリア） | 継承・継承王カエルム | 中央 | 廃墟（王都アステリア） |
| 辺境開拓諸侯 | 開拓・開拓王エルディオン | 北（ノースリーチ／フロストウォール） | 離散。山岳都市フロストウォールが名残 |
| 森の盟約国 | 生命・森護王シルヴァリア | 西（ヴェルディア） | 集落として残存（グリーンホロウ） |
| 海洋連合 | 航海・海統王ネレイオス | 西海（シーゲート／島々） | 港町として存続（シーゲート） |
| 鍛冶諸侯連合 | 鍛造・鍛冶王オルグラン | 東（レッドフォージ） | 廃墟（レッドフォージ） |
| 守護の砦国 | 守護・守護王ヴァルケイン | 東（ストームクラウン） | 砦跡のみ |
| 盟約諸国 | 信義・信義王アウレクス | 中央平原（アイゼンプレイン） | 離散 |
| 学府都市 | 知識・学識王セラディス | 王立図書院 | 遺構のみ |
| 巡礼自由都市網 | 道・巡礼王ルミナス | 王の大街道沿い | 街道のみ残存 |

諸王国はそれぞれ独自の文化を築いたが、九王戦争によってそのほとんどが失われた。現在の現存都市の多くは、これら旧王国の名残か、戦後に新たに築かれた拠点である。

## 4. 九英雄（Nine Heroes）

九英雄は王ではなく、人々の営みを支えた人物として各地の伝承に残る。

その功績は戦いだけでなく、探究、記録、鍛造、航海、継承など多岐にわたる。

| 称号 | 名 | フック |
|---|---|---|
| 地平の探求者 | アステル | 地図の余白を埋め、未知を既知へ変えた探求者。 |
| 王遺産の守り手 | エレナス | 王遺産を継承物として扱う倫理を示した人物。 |
| 深淵を踏破した者 | リュシアン | 深淵を恐怖ではなく知識の対象として記録した。 |
| 灯火を継いだ巫女 | イリア | 戦乱の中でも灯火と祈りを絶やさなかった。 |
| 星屑の鍛冶師 | ラグナ | 星屑鍛造の基礎を築いた鍛冶師。 |
| 失われた王都の記録者 | セレス | 崩壊する王都を最後まで書き留めた記録者。 |
| 竜と盟約を結んだ者 | カイル | 「竜」と呼ばれた巨大生物との距離を見出したと伝えられる。 |
| 世界の果てを見た航海者 | マレク | 海の彼方へ航海し、帰還後に多くの記録を残した。 |
| 名を残さなかった継承者 | （無名） | 名ではなく意思だけを後世へ託したとされる存在。 |

九英雄もまた、九王と同じく史実と伝承が重なり合っている。

彼らが実在したかどうかよりも、その思想が各時代へ受け継がれたことが重要視されている。

### 各英雄の伝承（断片）

以下は各地に残る伝承をまとめたものであり、確証のない記録を含む。

**地平の探求者 アステル** — 既知世界の外縁を歩き、地図の余白を埋め続けたと伝わる。探索者ギルドの三大任務「地図更新」は、その姿勢を継いだものだとされる。

**王遺産の守り手 エレナス** — 王遺産を「誰かの所有物」ではなく「後世へ受け継ぐべき記録」として扱う倫理を示した。ギルドに伝わる言葉「遺産を持ち帰るな、歴史を持ち帰れ」の源とされる。

**深淵を踏破した者 リュシアン** — 地下深部や未知の領域を、恐怖ではなく調査対象として記録した。危険区域へ踏み込む生態調査の規範を築いたと語られる。

**灯火を継いだ巫女 イリア** — 戦乱の中でも灯火と祈りを絶やさなかった。人々は彼女を「灯火を継いだ者」と呼び、その姿は今も継承の象徴として語られる。

**星屑の鍛冶師 ラグナ** — 星屑鍛造の基礎を築いた。鍛冶王オルグランの星炉の技を受け継いだとされ、現代の鍛冶もその系譜に連なると伝わる。

**失われた王都の記録者 セレス** — 崩壊していく王都を、最後まで書き留め続けた。九王戦争と王都陥落について残る数少ない記録は、その手によるものだという。

**竜と盟約を結んだ者 カイル** — 「竜」と呼ばれた巨大生物と、戦うのでも従えるのでもない距離を見出したと伝わる。人と巨大生物の関係を探る者たちが、その名を引き合いに出す。

**世界の果てを見た航海者 マレク** — 海の彼方へ航海し、帰還後に多くの記録を残した。既知世界の外縁や遠い海域の伝承の多くは、彼の航海録に由来するとされる。

**名を残さなかった継承者（無名）** — 名ではなく意思だけを後世へ託したとされる。確かな記録は何も残っていないが、「次の者へ受け渡す」という思想そのものを体現した存在として語り継がれている。

## 5. 各時代の暮らし・遺構（プレイヤーが遺跡で出会う痕跡）

探索者が目にする遺跡は、戦争そのものではなく、人々が確かに暮らしていた痕跡である。

**九王時代**の遺構には、巨大な石碑や境界標、初期の街道跡が残されている。

**王国時代**では、市場跡、農地、水路、港、工房、書庫など、人々の日常を感じさせる建造物が多く見つかる。

**九王戦争**の痕跡として残るのは、崩壊した城壁、焼失した街並み、放棄された砦などであり、その原因を語る記録は少ない。

**静寂の時代**には、人の手を離れた建造物へ植物が根を張り、野生動物が棲み着き、遺構は自然の一部となった。

そして**探索者の時代**。

新たに立てられる標識や調査記録は、未来の誰かに向けた新しい歴史の断片となっていく。

探索者が持ち帰る記録は、小さな発見に過ぎない。

だが、その積み重ねこそが、失われた文明を再び一つの歴史として結び直していく。

## 6. 静寂の時代と都市の興亡

九王戦争の後、世界は長い静寂に包まれた。

### 静寂の時代の変遷

人口は激減し、街道の維持は途絶え、都市の多くは放棄された。人の手を離れた建造物には植物が根を張り、野生動物が棲みつき、遺構は少しずつ自然の一部へと変わっていった。

旧文明の技術は、それを扱う者を失って忘れられた。記録は散逸し、歴史は伝承の中にだけ残された。

一方で、自然はその空白を埋めるように再生した。各地で新たな生態系が形成され、生物は旧文明の遺構すら取り込みながら独自の進化を遂げていった（戦後生態系 / `03_Ecology`）。

やがて、点在して生き延びた共同体は、危険な世界で生き抜くために調査と記録を共有し始めた。それが後の **探索者ギルド**（`08_SeekersGuild`）の前身となり、静寂の時代は緩やかに **探索者の時代** へと移り変わっていった。

### 都市の興亡

| 都市 | 帰結 | 補足 |
|---|---|---|
| 王都アステリア | 廃墟 | 中央王統の都。地下にモーンゲート |
| レッドフォージ | 廃墟 | 鍛冶諸侯連合の都 |
| シーゲート | 存続 | 海洋連合の港。海路の名残として機能 |
| グリーンホロウ | 存続 | 森の盟約国の集落として残る |
| フロストウォール | 存続 | 北の山岳都市。辺境開拓諸侯の名残 |
| アイアンヘイブン | 新設 | 戦後に築かれた探索者都市。現在のプレイヤー拠点 |

廃墟となった都は探索の主な舞台となり、存続した都市は探索者たちの活動を支える。そして戦後に生まれたアイアンヘイブンが、失われた世界を再び結び直す拠点となっている。
