現在の探索地理（エルド大陸）を定義する。都市・地域・河川・山脈・海域・遺跡・命名規則。Biome の生態は `05`、歴史地理は `01` に委譲する。

## アウトライン（ChatGPT 執筆）

1. 大陸概要（エルド大陸）
2. 都市・拠点
3. 地域（Region）
4. ダンジョン（地理上の位置・状態）
5. 河川・湖 / 山脈・峰 / 海域 / 古代遺跡
6. 命名ルール（接尾語）

## 必ず保持する正典（Canon — 削除不可）

- 舞台は **エルド大陸（Eld Continent）**。
- 拠点: 王都アステリア（廃墟）/ アイアンヘイブン（探索者都市・現存）ほか。固有名は `11_Glossary` と一致。
- ダンジョン: **モーンゲート（王都地下・MVP）** / ウィスパーウッド（ヴェルディア・MVP）/ 他は Future。
- 命名接尾語: -wood/-gate/-reach/-vale/-marsh/-fen/-ridge/-peak/-shore/-haven/-hollow/-forge/-rest。
- 旧王国地理（White Kingdom / Black Forge 等）は削除済。歴史上の地名が要る場合は `01_History` の範囲で扱う。
