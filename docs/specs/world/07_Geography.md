## 1. 大陸概要（エルド大陸）

物語の舞台となるのは、**エルド大陸（Eld Continent）**である。

現在知られている地図は、人類が再び探索を始めた範囲をまとめた「既知世界」を示している。

その外側にも土地は続くと考えられているが、詳細は記録されておらず、多くが未踏査のままである。

本資料で扱う地理は、探索者ギルドが把握している既知世界の地理情報であり、ゲーム実装の進行状況とは独立した世界設定として位置付けられる。

## 1.5. 大陸の空間構成（マップ）

エルド大陸の既知世界は、**倒れた王都を中心に、街道が放射状に広がる**構造をしている。

### 全体構図（ハブ＆スポーク）

大陸の心臓部には、いまや廃墟となった **王都アステリア** がある。かつて全ての街道がここへ集まり、ここから各地へ伸びていた。王都を起点とする **王の大街道**（巡礼王ルミナスゆかり）は、文明が失われた現在も大陸の骨格として残り、探索者たちの主要な移動路となっている。

王都のすぐ近くには、戦後に築かれた探索者都市 **アイアンヘイブン** があり、プレイヤーはここを拠点として各地方へ向かう。中心の廃墟と、その傍らで生きる新しい都市——この対比が、現在のエルド大陸を象徴している。

### 方角別の配置

| 方角 | 地方・地点 | ゆかり |
|---|---|---|
| **中央** | 王都アステリア（廃墟・地下にモーンゲート）／アイアンヘイブン（拠点）／アイゼンプレイン（平原）／グレイハイランド（高原） | 継承王・信義王 |
| **北** | ノースリーチ（辺境）／フロストリッジ（雪山）／フロストウォール（山岳都市） | 開拓王 |
| **西** | ヴェルディア（大森林）／グリーンホロウ（森の集落） | 森護王 |
| **西海岸** | ブラックショア／シーゲート（港）／ウェストベイ／シャッタードアイルズ（群島） | 海統王 |
| **東** | アイアンピーク／ストームクラウン（最高峰）／レッドリッジ・レッドクレスト（岩山）／レッドフォージ・アイアンフォージ（鍛冶） | 鍛冶王・守護王 |
| **東端** | サンダーピーク（火山地帯）／イースタンオーシャン | — |
| **南** | ブロークンマーシュ（湿地）／ミストフェン（沼地）／ブラックシー（南海） | — |

### 水系

大河 **クラウンリバー** は東の山脈（ストームクラウン）に源を発し、中央の王都アステリアを貫いて西海岸へ注ぐ。王都の繁栄を支えた大動脈であり、「王冠（Crown）」の名を今に残す数少ない地名でもある。支流の **シルバーリバー**・**アッシュクリーク**、点在する湖（ミラーレイク／ブルームウォーター／デッドウォーター）が各地方をうるおす。

### 配置図（概略）

```text

   ブラックショア             │
   シーゲート ── ヴェルディア ── アイゼンプレイン ── アイアンピーク/ストームクラウン
   西：海・港      グリーンホロウ    中央：平原         レッドリッジ/レッドフォージ
        │          西：森              │              東：山・鍛冶
   ウェストベイ                 ★王都アステリア              │
                                ＋アイアンヘイブン        サンダーピーク
                                  中央：拠点              東端：火山
                                       │
                                 グレイハイランド
                                       │
                          ブロークンマーシュ / ミストフェン
```

## 2. 都市・拠点

現在確認されている主要な都市・拠点は次のとおりである。

| 名称 | 種別 | 状態 |
|---|---|---|
| 王都アステリア（Asteria） | 旧文明の首都 | 廃墟 |
| **アイアンヘイブン（Ironhaven）** | 探索者都市（**プレイヤー拠点**） | 現存 |
| シーゲート（Seagate） | 港町 | 現存 |
| フロストウォール（Frostwall） | 山岳都市 | 現存 |
| グリーンホロウ（Greenhollow） | 森の集落 | 現存 |
| レッドフォージ（Redforge） | 鍛冶都市 | 廃墟 |

現存する都市はいずれも旧文明の規模には及ばないが、探索者たちの活動拠点として世界をつないでいる。

プレイヤーが拠点とするのは **アイアンヘイブン** である。探索者ギルドの支部、装備工房、鍛冶屋、図鑑（調査記録）が集まり、各調査地への出発点となる。ゲーム内の「拠点」はこのアイアンヘイブンを指す。

旧文明の首都 **王都アステリア** は廃墟と化しているが、その地下には広大な施設網が今も眠っており、地下深部の多くは未踏査のままである。

各地域には固有の生態系が形成されており、探索者は環境ごとの違いを前提として調査を行う。

## 3.5. 地方誌（Region 詳細）

各地域の地勢・主要地点・旧王国との関係・生態傾向・探索上の注意をまとめる。生態は傾向の記述に留め、固有モンスターや食物連鎖は Biome 設計（`05_Biomes`）で確定する。

### ヴェルディア（大森林・西）

- **地勢:** 大陸西部を覆う広大な原生林。巨木と湿った林床、視界の利かない深い森。
- **主要地点:** グリーンホロウ（森の集落・現存）／**囁きの森（ウィスパーウッド）**（Biome-02・生態確定 / `05_Biomes §3.2`）。
- **旧王国・歴史:** 森の盟約国（森護王シルヴァリア）。世界樹の伝承が最も濃い地。生命の種子／翠杖ヴェルドの所在伝承。
- **生態傾向:** 共生適応の土地。菌・植物類（VI）が共生網の中核を成し、昆虫（IV）・獣（I）・爬虫（III）が分化。fire 弱点・poison 中心。
- **探索メモ:** 視界が利かず、音と気配が頼り。次 Biome 第一候補。

### アイゼンプレイン（平原・中央）

- **地勢:** 中央に広がる開けた平原。旧街道が縦横に走る。
- **主要地点:** 諸国盟約の地の遺構／王の大街道の結節点。
- **旧王国・歴史:** 盟約諸国（信義王アウレクス）。諸国の盟約が結ばれた地。盟約の指環／盟約剣フィデスの伝承。
- **生態傾向:** 獣類（Class I）・鳥類（Class II）。群れを成す種が多い。
- **探索メモ:** 遮蔽が少なく、遭遇は遠距離から始まりやすい。

### グレイハイランド（高原・中央南）

- **地勢:** 風の強い灰色の高原。露岩と短草。
- **主要地点:** 風化した境界標。
- **旧王国・歴史:** 諸国の境界地帯。帰属の曖昧な緩衝地だったとされる。
- **生態傾向:** 鳥類・岩場適応の獣類。
- **探索メモ:** 強風で痕跡が消えやすく、追跡が難しい。

### ノースリーチ（北方辺境・北）

- **地勢:** 既知世界の北の縁。寒冷で、地図の余白に接する。
- **主要地点:** 開拓王の境界標／黎明の羅針盤・王剣レクスの所在伝承。
- **旧王国・歴史:** 辺境開拓諸侯（開拓王エルディオン）。地平の探求者アステルゆかりの地。
- **生態傾向:** 寒冷適応の獣類。古代種（Class VII）の噂が絶えない。
- **探索メモ:** 未踏査区域に最も近い。最果ての調査対象。

### フロストリッジ（雪山・北）

- **地勢:** 北部を貫く雪の山脈。
- **主要地点:** フロストウォール（山岳都市・現存）。
- **旧王国・歴史:** 辺境開拓諸侯の名残。寒冷の中で存続した数少ない都市。
- **生態傾向:** 厚毛の獣類、雪原適応種。
- **探索メモ:** 寒さと雪崩。装備の備えが要る。

### レッドリッジ（岩山・峡谷・東）

- **地勢:** 赤い岩肌の峡谷地帯。鉱脈が豊富。
- **主要地点:** レッドフォージ（鍛冶都市・廃墟）／アイアンフォージ（古代鍛冶場）／レッドクレスト。
- **旧王国・歴史:** 鍛冶諸侯連合（鍛冶王オルグラン）。星炉の槌・星鍛槌グランフォージの所在伝承。
- **生態傾向:** 甲殻系・鉱物食。鉱物化適応の事例が見られる。
- **探索メモ:** 落石・鉱毒。

### サンダーピーク（火山地帯・東端）

- **地勢:** 活発な火山と溶岩原。
- **主要地点:** 火口の遺構。
- **旧王国・歴史:** 帰属不明。古くからの禁足地の伝承。
- **生態傾向:** 爬虫類（Class III）・耐熱種。古代種の噂。
- **探索メモ:** 高温と噴気。長時間の滞在は困難。

### ブラックショア（海岸・西海）

- **地勢:** 黒砂の海岸線。
- **主要地点:** シーゲート（港・現存）／ウェストベイ／シャッタードアイルズ（群島）。
- **旧王国・歴史:** 海洋連合（海統王ネレイオス）。潮境の海図・潮槍ネレイスの所在伝承。
- **生態傾向:** 水棲類（Class V）・甲殻系。
- **探索メモ:** 潮汐と座礁船。海域は危険度が高い。

### ミストフェン（沼地・南）

- **地勢:** 霧の立ち込める沼地。
- **主要地点:** 沈んだ集落跡。
- **旧王国・歴史:** 戦後に水没した土地。
- **生態傾向:** 水棲・菌類。毒を持つ種が多い。
- **探索メモ:** 視界不良と不安定な足場。

### ブロークンマーシュ（湿地・南）

- **地勢:** 地盤の崩れた広大な湿地。
- **主要地点:** 崩落した街道橋。
- **旧王国・歴史:** 九王戦争で地盤が崩れた旧境界。
- **生態傾向:** 昆虫類・水棲類。毒・病。
- **探索メモ:** 泥濘と病。長距離移動が難しい。

## 4. ダンジョン（地理上の位置・状態）

探索対象となる地下施設や遺跡は、地理上は一地点として扱われる。

一方、生態系としては個別の Biome が定義される。

現時点で正式に確定している調査地は一つのみである。

| 名称 | 地理 | 状態 |
|---|---|---|
| モーンゲート（Mourngate） | 王都アステリア地下（旧排水網） | 実装済（唯一の確定ダンジョン） |
| 囁きの森（ウィスパーウッド） | ヴェルディア（大森林） | 構想（次 Biome 第一候補。生態確定 / `05_Biomes §3.2`） |
| Future（未踏査・構想） | 未踏査区域 | 未確定 |

モーンゲートは、アイアンヘイブンから最初に向かう調査地であり、王都アステリア地下の旧排水網に形成された地下生態系である（生態は `05_Biomes.md`）。

次の調査地は **ヴェルディア（大森林）の囁きの森（ウィスパーウッド）** を第一候補とする。生態は確定済み（共生適応 / `05_Biomes §3.2`）。ゲームへの実装・採用個体は別途選定する。

それ以外の未踏査区域は存在が想定されているのみであり、名称・位置・内容はいずれも確定していない。

## 5.5. 九王ゆかりの地（伝承）

各地には、九王（`01_History` §3）の理念や事績にちなむと伝わる土地が点在する。いずれも伝承であり、確証のある記録は少ない。対応する王遺産・伝説武器は `02_Relics` を参照。

| 九王 | ゆかりの地 | 伝わる縁 |
|---|---|---|
| 開拓王エルディオン | ノースリーチ（北方辺境） | 既知世界の外縁を切り開いた地 |
| 守護王ヴァルケイン | ストームクラウン砦 | 境界を守った要塞 |
| 学識王セラディス | 王立図書院 | 知識を集め封じた大書庫 |
| 信義王アウレクス | アイゼンプレイン（平原） | 諸国の盟約が結ばれた地 |
| 巡礼王ルミナス | 王の大街道 | 人と文化を結んだ街道網 |
| 鍛冶王オルグラン | アイアンフォージ／レッドフォージ | 星炉を築いた鍛冶の地 |
| 海統王ネレイオス | シーゲート／シャッタードアイルズ | 海路を治めた港と島々 |
| 森護王シルヴァリア | ヴェルディア／グリーンホロウ | 森と生命循環を守った地 |
| 継承王カエルム | 王都アステリア／王家霊廟 | 「次の者へ」の理念を遺した王都 |

> フロストウォール等の現存都市の多くは九王戦争後に成立した探索者の拠点であり、特定の九王とは結び付かない。

## 6. 命名ルール（接尾語）

地名には、土地の性質や役割を示す接尾語が用いられることが多い。

| 接尾語 | 意味 |
|---|---|
| **-wood** | 森 |
| **-gate** | 地下施設・関所 |
| **-reach** | 辺境・奥地 |
| **-vale** | 谷 |
| **-marsh** | 湿地 |
| **-fen** | 沼地 |
| **-ridge** | 尾根・山脈 |
| **-peak** | 山・火山 |
| **-shore** | 海岸 |
| **-haven** | 街・避難港 |
| **-hollow** | 谷・窪地 |
| **-forge** | 鍛冶都市 |
| **-rest** | 墓所・眠る地 |

これらの接尾語は、探索者が未知の土地を記録する際にも共通の命名基準として用いられる。
