## 1. 大陸概要（エルド大陸）

物語の舞台となるのは、**エルド大陸（Eld Continent）**である。

現在知られている地図は、人類が再び探索を始めた範囲をまとめた「既知世界」を示している。

その外側にも土地は続くと考えられているが、詳細は記録されておらず、多くが未踏査のままである。

本資料で扱う地理は、探索者ギルドが把握している既知世界の地理情報であり、ゲーム実装の進行状況とは独立した世界設定として位置付けられる。

## 1.5. 大陸の空間構成（マップ）

エルド大陸の既知世界は、**倒れた王都を中心に、街道が放射状に広がる**構造をしている。

### 全体構図（ハブ＆スポーク）

大陸の心臓部には、いまや廃墟となった **王都アステリア** がある。かつて全ての街道がここへ集まり、ここから各地へ伸びていた。王都を起点とする **王の大街道**（巡礼王ルミナスゆかり）は、文明が失われた現在も大陸の骨格として残り、探索者たちの主要な移動路となっている。

王都のすぐ近くには、戦後に築かれた探索者都市 **アイアンヘイブン** があり、プレイヤーはここを拠点として各地方へ向かう。中心の廃墟と、その傍らで生きる新しい都市——この対比が、現在のエルド大陸を象徴している。

### 方角別の配置

| 方角 | 地方・地点 | ゆかり |
|---|---|---|
| **中央** | 王都アステリア（廃墟・地下にモーンゲート）／アイアンヘイブン（拠点）／アイゼンプレイン（平原）／グレイハイランド（高原） | 継承王・信義王 |
| **北** | ノースリーチ（辺境）／フロストリッジ（雪山）／フロストウォール（山岳都市） | 開拓王 |
| **西** | ヴェルディア（大森林）／グリーンホロウ（森の集落） | 森護王 |
| **西海岸** | ブラックショア／シーゲート（港）／ウェストベイ／シャッタードアイルズ（群島） | 海統王 |
| **東** | アイアンピーク／ストームクラウン（最高峰）／レッドリッジ・レッドクレスト（岩山）／レッドフォージ・アイアンフォージ（鍛冶） | 鍛冶王・守護王 |
| **東端** | サンダーピーク（火山地帯）／イースタンオーシャン | — |
| **南** | ブロークンマーシュ（湿地）／ミストフェン（沼地）／ブラックシー（南海） | — |

### 水系

大河 **クラウンリバー** は東の山脈（ストームクラウン）に源を発し、中央の王都アステリアを貫いて西海岸へ注ぐ。王都の繁栄を支えた大動脈であり、「王冠（Crown）」の名を今に残す数少ない地名でもある。支流の **シルバーリバー**・**アッシュクリーク**、点在する湖（ミラーレイク／ブルームウォーター／デッドウォーター）が各地方をうるおす。

   シーゲート ── ヴェルディア ── アイゼンプレイン ── アイアンピーク/ストームクラウン
   西：海・港      グリーンホロウ    中央：平原         レッドリッジ/レッドフォージ
        │          西：森              │              東：山・鍛冶
   ウェストベイ                 ★王都アステリア              │
                                ＋アイアンヘイブン        サンダーピーク
                                  中央：拠点              東端：火山
                                       │

                                       │

## 2. 都市・拠点

現在確認されている主要な都市・拠点は次のとおりである。

| 名称 | 種別 | 状態 |
|---|---|---|
| 王都アステリア（Asteria） | 旧文明の首都 | 廃墟 |
| **アイアンヘイブン（Ironhaven）** | 探索者都市（**プレイヤー拠点**） | 現存 |
| シーゲート（Seagate） | 港町 | 現存 |
| フロストウォール（Frostwall） | 山岳都市 | 現存 |
| グリーンホロウ（Greenhollow） | 森の集落 | 現存 |
| レッドフォージ（Redforge） | 鍛冶都市 | 廃墟 |

現存する都市はいずれも旧文明の規模には及ばないが、探索者たちの活動拠点として世界をつないでいる。

プレイヤーが拠点とするのは **アイアンヘイブン** である。探索者ギルドの支部、装備工房、鍛冶屋、図鑑（調査記録）が集まり、各調査地への出発点となる。ゲーム内の「拠点」はこのアイアンヘイブンを指す。

旧文明の首都 **王都アステリア** は廃墟と化しているが、その地下には広大な施設網が今も眠っており、地下深部の多くは未踏査のままである。

## 2.5. 都市誌（City 詳細）

各都市の立地・旧王国との関係・現在の姿・探索上の役割をまとめる。現存都市は旧文明の規模に及ばないが、いずれも探索者の活動を支えている。対応する九王・王遺産は `01_History §3.5` / `02_Relics` を正とする。

### アイアンヘイブン（探索者都市・拠点 / 中央）

- **立地:** 王都アステリアのほど近く、クラウンリバー沿い。九王戦争後に新設。
- **旧王国:** なし（戦後都市）。特定の九王とは結び付かない。
- **現在:** 探索者ギルド支部を核に、赤鉄の工房・中央市場・辻灯亭・認定所が集まる調査の前線基地（`08_SeekersGuild §10/§14`）。各地から探索者・隊商・職人が集う、最も活気ある都市。
- **ゲーム:** プレイヤー拠点。ダンジョン選択・編成・装備・図鑑の起点。
- 中心の廃墟（アステリア）と、その傍らで生きる新都市——この対比が現在のエルド大陸を象徴する。

### 王都アステリア（旧文明の首都・廃墟 / 中央）

- **立地:** 大陸の心臓部。クラウンリバーが貫き、かつて全街道が集まった中心。
- **旧王国:** 中央王統（継承王カエルム）。王家霊廟を擁する。
- **現在:** 地上は崩れ、王都を結んだ大橋 **王冠橋**（クラウンリバーに架かっていた）も落ちた。地下には旧排水網＝モーンゲートが広がり、深部の多くは未踏査。
- **ゲーム:** 地下モーンゲートが最初の調査地（`05_Biomes`）。

!!! note "残された問い"
    王冠は「倒れたが滅びていない」と各地に刻まれる。何が滅びずに残ったのか——その答えは王都の地下深部にあると噂される（真相は語られない）。

### シーゲート（港町・現存 / 西海岸）

- **立地:** ブラックショアの良港。シャッタードアイルズを望む。
- **旧王国:** 海洋連合（海統王ネレイオス）。
- **現在:** 九王戦争を生き延びた数少ない都市。港の **潮見の市（しおみのいち）** に各地の物資と海域の噂が集まる。港長 **潮見のマレーナ** が交易を取り仕切る。漁師 **潮笛のジーン** が干潮刻を知らせ、灯台番 **キル** が潮見灯台跡を補修する。灯の届かぬ **潮下市（うしおしたいち）** —通称・闇市——の影に **帳簿屋シル** がいる（`14_Society §2.6`／`13_Characters`）。
- **探索:** 海域は危険度が高く、外洋の調査はまだ進んでいない。

### フロストウォール（山岳都市・現存 / 北）

- **立地:** フロストリッジの雪山に抱かれた高所。北辺へ **最果ての門（さいはてのもん）** が開く。
- **旧王国:** 辺境開拓諸侯（開拓王エルディオン）の名残。
- **現在:** 厳しい寒冷の中で存続した堅牢な都市。**門番ウルフ** が最果ての門を守り、北へ向かう探索者の最終確認を行う。補給係 **氷砕のリディア** と測量部駐在の **北標のイージン** が門前を支える（`13_Characters`）。開拓者の気風が残り、未踏査の北（ノースリーチ）へ向かう探索者の最前線拠点。
- **探索:** 寒さ・雪崩への備えが要る、最果ての調査拠点。

### グリーンホロウ（森の集落・現存 / 西）

- **立地:** 大森林ヴェルディアの窪地。囁きの森（ウィスパーウッド）への入口 **翠の門（みどりのもん）** を擁する。
- **旧王国:** 森の盟約国（森護王シルヴァリア）。**翠の盟約** の地。
- **現在:** 森と人が互いを侵さぬ作法を守って暮らす小さな集落。長老 **盟約のエルナ** が翠の盟約の解釈を司る。見習い **ファウナ** と森の斥候 **ドルン** が翠の門を守る（`13_Characters`）。生物との距離を探るビーストテイマーが集う。次の大規模調査地・囁きの森への玄関口。
- **探索:** 視界の利かぬ森。音と気配が頼り。

### レッドフォージ（鍛冶都市・廃墟 / 東）

- **立地:** レッドリッジの赤い峡谷。古代鍛冶場アイアンフォージに連なる。
- **旧王国:** 鍛冶諸侯連合（鍛冶王オルグラン）。
- **現在:** 廃墟だが、星炉の系譜を伝える大工房跡 **大鞴場（おおふいごば）** が残り、鍛冶遺物の宝庫として探索者を引きつける。**炉跡守ベケット** が外縁の野営地で入山者に熱害の順番を教える（`13_Characters`）。星屑の鍛冶師ラグナを経た技は、いまアイアンヘイブンの赤鉄の工房へ受け継がれている。

!!! note "残された問い"
    炉印（九対の同一紋様 / `13_Characters` Q-01）の手がかりが、この地の炉壁に眠るとされる。

## 3. 地域（Region）

エルド大陸は、多様な自然環境によって区分される。

各地域には固有の生態系が形成されており、探索者は環境ごとの違いを前提として調査を行う。

| 名称 | 環境 |
|---|---|
| ヴェルディア（Verdia） | 大森林 |
| アイゼンプレイン（Eisen Plain） | 平原 |
| ブロークンマーシュ（Broken Marsh） | 湿地 |
| グレイハイランド（Gray Highlands） | 高原 |
| サンダーピーク（Thunder Peak） | 火山地帯 |
| レッドリッジ（Red Ridge） | 岩山・峡谷 |
| フロストリッジ（Frost Ridge） | 雪山 |
| ブラックショア（Black Shore） | 海岸 |
| ミストフェン（Mistfen） | 沼地 |
| ノースリーチ（Northreach） | 北方辺境 |

地域の詳細な生態系については、それぞれの Biome において記録される。

## 3.5. 地方誌（Region 詳細）

各地域の地勢・主要地点・旧王国との関係・生態傾向・探索上の注意をまとめる。生態は傾向の記述に留め、固有モンスターや食物連鎖は Biome 設計（`05_Biomes`）で確定する。

### ヴェルディア（大森林・西）

- **地勢:** 大陸西部を覆う広大な原生林。巨木と湿った林床、視界の利かない深い森。
- **主要地点:** グリーンホロウ（森の集落・現存）／**囁きの森（ウィスパーウッド）**（Biome-02・生態確定 / `05_Biomes §3.2`）／奥地 **翠眼の聖域（ヴェルド・アイ）**。
- **旧王国・歴史:** 森の盟約国（森護王シルヴァリア）。世界樹の伝承が最も濃い地。生命の種子／翠杖ヴェルドの所在伝承。
- **生態傾向:** 共生適応の土地。菌・植物類（VI）が共生網の中核を成し、昆虫（IV）・獣（I）・爬虫（III）が分化。fire 弱点・poison 中心。
- **探索メモ:** 視界が利かず、音と気配が頼り。次 Biome 第一候補。

### アイゼンプレイン（平原・中央）

- **地勢:** 中央に広がる開けた平原。旧街道が縦横に走る。
- **主要地点:** **盟約の丘（フィデス・ヒル）** — 信義王アウレクスが諸国の盟約を結んだと伝わる地。最後の盟議（`01 §2`）の舞台。盟約剣フィデスの所在伝承と接続。
- **旧王国・歴史:** 盟約諸国（信義王アウレクス）。諸国の盟約が結ばれた地。盟約の指環／盟約剣フィデスの伝承。
- **生態傾向:** 獣類（Class I）・鳥類（Class II）。群れを成す種が多い。
- **探索メモ:** 遮蔽が少なく、遭遇は遠距離から始まりやすい。盟約碑が風化して読める／読めない日がある。

### グレイハイランド（高原・中央南）

- **地勢:** 風の強い灰色の高原。露岩と短草。
- **主要地点:** 風化した境界標／**灰風の回廊（はいかぜのかいろう）** — 一方向に灰風が吹き続ける区画。寄り道 Biome・灰風高原（`05_Biomes §3.8`）の入口。
- **旧王国・歴史:** 諸国の境界地帯。帰属の曖昧な緩衝地だったとされる。
- **生態傾向:** 鳥類・岩場適応の獣類。風蝕適応。
- **探索メモ:** 強風で痕跡が消えやすく、追跡が難しい。視界は開けるが立ち止まると体力を奪われる。

### ノースリーチ（北方辺境・北）

- **地勢:** 既知世界の北の縁。寒冷で、地図の余白に接する。
- **主要地点:** 開拓王の境界標／黎明の羅針盤・王剣レクスの所在伝承。
- **旧王国・歴史:** 辺境開拓諸侯（開拓王エルディオン）。地平の探求者アステルゆかりの地。
- **生態傾向:** 寒冷適応の獣類。古代種（Class VII）の噂が絶えない。
- **探索メモ:** 未踏査区域に最も近い。最果ての調査対象。

### フロストリッジ（雪山・北）

- **地勢:** 北部を貫く雪の山脈。
- **主要地点:** フロストウォール（山岳都市・現存）。
- **旧王国・歴史:** 辺境開拓諸侯の名残。寒冷の中で存続した数少ない都市。
- **生態傾向:** 厚毛の獣類、雪原適応種。
- **探索メモ:** 寒さと雪崩。装備の備えが要る。

### レッドリッジ（岩山・峡谷・東）

- **地勢:** 赤い岩肌の峡谷地帯。鉱脈が豊富。
- **主要地点:** レッドフォージ（鍛冶都市・廃墟）／アイアンフォージ（古代鍛冶場）／レッドクレスト。
- **旧王国・歴史:** 鍛冶諸侯連合（鍛冶王オルグラン）。星炉の槌・星鍛槌グランフォージの所在伝承。
- **生態傾向:** 甲殻系・鉱物食。鉱物化適応の事例が見られる。
- **探索メモ:** 落石・鉱毒。

### サンダーピーク（火山地帯・東端）

- **地勢:** 活発な火山と溶岩原。
- **主要地点:** 火口の遺構／**黒曜の禁足地（オブシディアン・バン）** — 開拓王の境界標が「ここより下へ」とだけ刻まれた伝承の地。**禁足守ルオ** が入口で立ち帰りを促す（`13_Characters`）。噴気に黒曜粉が含まれ、長時間滞在は困難。
- **旧王国・歴史:** 帰属不明。古くからの禁足地の伝承。
- **生態傾向:** 爬虫類（Class III）・耐熱種。古代種の噂。
- **探索メモ:** 高温と噴気。火山口の縁に、白王の神殿と同型の小さな空祠が一基だけ残るという噂がある（建造者・用途不明 / 先王時代の余白）。

### ブラックショア（海岸・西海）

- **地勢:** 黒砂の海岸線。
- **主要地点:** シーゲート（港・現存）／ウェストベイ／**シャッタードアイルズ（千破列島 / シャッタード・チェイン）** — 百以上の小島が直線上に座礁した群島。骨島・鏡島・空島の三種に大別される（`§5.6`）。
- **旧王国・歴史:** 海洋連合（海統王ネレイオス）。潮境の海図・潮槍ネレイスの所在伝承。
- **生態傾向:** 水棲類（Class V）・甲殻系。
- **探索メモ:** 潮汐と座礁船。海域は危険度が高い。

### ミストフェン（沼地・南）

- **地勢:** 霧の立ち込める沼地。
- **主要地点:** 沈んだ集落跡。
- **旧王国・歴史:** 戦後に水没した土地。
- **生態傾向:** 水棲・菌類。毒を持つ種が多い。
- **探索メモ:** 視界不良と不安定な足場。

### ブロークンマーシュ（湿地・南）

- **地勢:** 地盤の崩れた広大な湿地。
- **主要地点:** **盟約崩れ橋（フィデスブリッジ）** — 信義王アウレクスの盟約で結ばれた諸国を繋いだ大橋の半壊遺構。**渡し守トーン** が綱渡りで渡し、**碑林のオルド** が Z2 碑林を守る（`13_Characters`）。車輪同盟が避ける区間（S-03 / `14_Society`）。寄り道 Biome・断盟湿地（`05_Biomes §3.9`）。
- **旧王国・歴史:** 九王戦争で地盤が崩れた旧境界。盟約諸国（信義王）の最後の結節点。
- **生態傾向:** 昆虫類・水棲類。毒・病。断裂適応。
- **探索メモ:** 泥濘と病。長距離移動が難しい。盟約文の断片が風化して残る。

## 4. ダンジョン（地理上の位置・状態）

探索対象となる地下施設や遺跡は、地理上は一地点として扱われる。

一方、生態系としては個別の Biome が定義される。

現時点で正式に確定している調査地は一つのみである。

| 名称 | 地理 | 状態 |
|---|---|---|
| モーンゲート（Mourngate） | 王都アステリア地下（旧排水網） | 実装済（唯一の確定ダンジョン） |
| 囁きの森（ウィスパーウッド） | ヴェルディア（大森林） | 構想（次 Biome 第一候補。生態確定 / `05_Biomes §3.2`） |
| 紅炉峡谷（クリムゾン・フォージ） | レッドフォージ地下 | 構想（Biome-06。生態確定 / `05_Biomes §3.6`） |
| 嵐冠砦跡（ストームクラウン・シタデル） | ストームクラウン砦跡 | 構想（寄り道 Biome / `05_Biomes §3.7`）。**嵐読みのヴェイト**・吹星崖の灯番リーヴが入口を案内 |
| 灰風高原（グレイハイランド・ウィンドスコープ） | グレイハイランド | 構想（寄り道 Biome / `05_Biomes §3.8`） |
| 断盟湿地（ブロークンマーシュ・ブリッジレスト） | ブロークンマーシュ | 構想（寄り道 Biome / `05_Biomes §3.9`） |
| 黒曜禁足地（オブシディアン・バン） | サンダーピーク | 構想（寄り道 Biome / `05_Biomes §3.10`） |
| Future（未踏査・構想） | 未踏査区域 | 未確定 |

モーンゲートは、アイアンヘイブンから最初に向かう調査地であり、王都アステリア地下の旧排水網に形成された地下生態系である（生態は `05_Biomes.md`）。

次の調査地は **ヴェルディア（大森林）の囁きの森（ウィスパーウッド）** を第一候補とする。生態は確定済み（共生適応 / `05_Biomes §3.2`）。ゲームへの実装・採用個体は別途選定する。

それ以外の未踏査区域は存在が想定されているのみであり、名称・位置・内容はいずれも確定していない。

### 千破列島（シャッタード・チェイン）

海統王ネレイオスゆかりの **シャッタードアイルズ** を構成する座礁群。百以上の小島が、不自然なほど直線上に並ぶように沈没している。

船乗りは「ネレイオスが航路を振り返った痕跡」と言い、学者は地殻変動と言う。どちらも確証はない。沈没航路 Biome（`05_Biomes §3.4`）の海上版寄り道として踏査が進む。

| 種類 | 通称 | 概要 |
|---|---|---|
| 骨島 | **リブ・アイル** | 座礁船の肋骨が島そのもの。潮槍ネレイスの断片噂 |
| 鏡島 | **ミラー・アイル** | 塩結晶の平原。干潮時だけ渡れる |
| 空島 | **ホロー・アイル** | 中が空洞。潮の満ち引きで水柱が噴き上がる |

## 5.5. 九王ゆかりの地（伝承）

各地には、九王（`01_History` §3）の理念や事績にちなむと伝わる土地が点在する。いずれも伝承であり、確証のある記録は少ない。対応する王遺産・伝説武器は `02_Relics` を参照。

| 九王 | ゆかりの地 | 伝わる縁 |
|---|---|---|
| 開拓王エルディオン | ノースリーチ（北方辺境） | 既知世界の外縁を切り開いた地 |
| 守護王ヴァルケイン | ストームクラウン砦 | 境界を守った要塞 |
| 学識王セラディス | 王立図書院 | 知識を集め封じた大書庫 |
| 信義王アウレクス | アイゼンプレイン（平原） | 諸国の盟約が結ばれた地 |
| 巡礼王ルミナス | 王の大街道 | 人と文化を結んだ街道網 |
| 鍛冶王オルグラン | アイアンフォージ／レッドフォージ | 星炉を築いた鍛冶の地 |
| 海統王ネレイオス | シーゲート／シャッタードアイルズ | 海路を治めた港と島々 |
| 森護王シルヴァリア | ヴェルディア／グリーンホロウ | 森と生命循環を守った地 |
| 継承王カエルム | 王都アステリア／王家霊廟 | 「次の者へ」の理念を遺した王都 |

> フロストウォール等の現存都市の多くは九王戦争後に成立した探索者の拠点であり、特定の九王とは結び付かない。

## 6. 命名ルール（接尾語）

地名には、土地の性質や役割を示す接尾語が用いられることが多い。

| 接尾語 | 意味 |
|---|---|
| **-wood** | 森 |
| **-gate** | 地下施設・関所 |
| **-reach** | 辺境・奥地 |
| **-vale** | 谷 |
| **-marsh** | 湿地 |
| **-fen** | 沼地 |
| **-ridge** | 尾根・山脈 |
| **-peak** | 山・火山 |
| **-shore** | 海岸 |
| **-haven** | 街・避難港 |
| **-hollow** | 谷・窪地 |
| **-forge** | 鍛冶都市 |
| **-rest** | 墓所・眠る地 |

これらの接尾語は、探索者が未知の土地を記録する際にも共通の命名基準として用いられる。
