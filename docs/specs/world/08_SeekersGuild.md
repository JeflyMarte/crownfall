## この文書の責務

探索を担う組織と、その中でのプレイヤーの立場を定義する。職の世界観は `09_Jobs`、ゲームループ数値は `game/` に委譲する。

## アウトライン（ChatGPT 執筆）

1. コンセプト（調査管理機構 / 武力組織ではない）
2. 設立目的
3. 三大任務（遺物回収 / 生態調査 / 地図更新）
4. 探索者とは（専門調査員）
5. 評価基準（討伐数ではない）
6. プレイヤーの役割
7. ゲームループとの対応

## 必ず保持する正典（Canon — 削除不可）

- 探索者ギルドは「冒険者組合」ではなく **探索管理機構**＝調査機関。
- 三大任務: **遺物回収（Relic Recovery）/ 生態調査（Ecology Survey）/ 地図更新（Frontier Mapping）**。討伐は調査の一手段で目的ではない。
- 評価は討伐数ではなく **発見・新素材・遺跡調査・危険区域更新** の成果。
- プレイヤーは最前線で戦わず、探索隊の責任者として地域選択・装備・編成・分析を判断する（`game/26_CombatVision.md` の非直接操作と一致）。
