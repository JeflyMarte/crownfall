## この文書の責務

モンスター名の命名規則を定義する。分類は `04`、出現は `05` に委譲する。

## アウトライン（ChatGPT 執筆）

1. 基本方針（漢字／カタカナ両用）
2. 命名パターン（和名型 / カタカナ型）
3. 命名ルール（実在生物ベース必須）
4. 図鑑表記の書き方

## 必ず保持する正典（Canon — 削除不可）

- 漢字・カタカナどちらも可。重要なのは「名前から生態や特徴が想像できること」。
- カタカナ名でも **必ず実在生物をベースにする**（可: ミラーボア / 不可: デモニックビースト・カオスイーター・アビスデーモン）。
- 図鑑では正式名称＋元となる生物を調査記録で説明する。
