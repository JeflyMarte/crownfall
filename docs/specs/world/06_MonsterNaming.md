## 1. 基本方針（漢字／カタカナ両用）

モンスター名は、**漢字表記**・**カタカナ表記**のいずれも使用できる。

重要なのは表記の形式ではなく、その名前から生物の特徴や生態が自然に想像できることである。

探索者や探索者ギルドは、生物を識別し記録するために通称を与えることがある。

一方で、古くから地域に伝わる呼び名が、そのまま正式名称として定着している場合もある。

どちらの表記であっても、名称は世界の自然や歴史と調和していなければならない。

---

## 2. 命名パターン（和名型 / カタカナ型）

### パターンA：和名型（生態が直感的に伝わる）

土地や外見、生態を短く表現する命名。

一目でどのような生物なのか想像しやすく、地域住民にも広く使われる呼称となる。

**例**

- 灰狼
- 巨角鹿
- 洞窟蜘蛛
- 森熊
- 毒蛙

### パターンB：カタカナ型（探索者・ギルドが付けた通称）

探索者ギルドが調査対象として整理する際に用いられる呼称。

基本構成は**「特徴 + 生物名」**とし、生物としての起源が名称から読み取れることを重視する。

**例**

- ミラーボア（鏡のような鱗を持つ大蛇）
- アイアンスタッグ（鉄のような角を持つ鹿）
- ブラッドウルフ（赤い毛並みの狼）
- モスラット（苔をまとった地下鼠）
- ソーンスパイダー（棘を持つ蜘蛛）

---

## 3. 命名ルール（実在生物ベース必須）

すべての名称は、現実に存在する生物を起点として命名する。

外見や能力が大きく変化していても、生物としての祖先が名称から読み取れることを原則とする。

命名では次の点を守る。

- 実在生物を必ず名称に含める。
- 生態や外見を特徴語で表現する。
- 地域環境との関係が想像できる名称とする。
- 異世界起源や超自然的な存在を連想させる語は避ける。

### 可否の例

| 可 | 理由 |
|---|---|
| ミラーボア | 「ボア（大蛇）」を基に特徴を表現している。 |
| ストーンクラブ | 「カニ」を祖先とすることが分かる。 |
| アイアンベア | 「熊」を基に特徴を示している。 |
| フロストホーク | 「鷹」を基に環境適応を表現している。 |

| 不可 | 理由 |
|---|---|
| デモニックビースト | 実在生物を基点としていない。 |
| カオスイーター | 生物種が読み取れない。 |
| アビスデーモン | 異世界起源を想起させ、世界観と整合しない。 |

## 4. 図鑑表記の書き方

図鑑では、正式名称だけでなく、その名称の基となった生物を調査記録として記載する。

記録は断定的な伝承ではなく、生態観察に基づく説明としてまとめる。

基本的な記述は次の形式を用いる。

> **正式名称** — 元となる生物の一種。環境への長年の適応によって現在の特徴を獲得した。

### 記述例

> **ミラーボア** — 大蛇の一種。囁きの森の環境に適応し、鏡のように光を反射する鱗を獲得して、薄暗い林床で姿をくらます。

名称は、生物を飾るためのものではない。

探索者にとって名前とは、その生物が何者であり、どのような環境で生きてきたのかを伝える最初の記録である。
