## この文書の責務

ジョブ＝探索者ギルドの専門資格としての**世界観・イメージ・由来**を定義する。ステータス・武器対応・成長・AI は `game/` に委譲する。

## アウトライン（ChatGPT 執筆）

1. ジョブとは（探索者ギルドの専門資格）
2. 基本 5 職の世界観（役割イメージ・由来・探索隊での位置）
3. デザイン方針（共通制服 + 専門装備 / 上位下位を設けない）
4. 将来プログレッション（進化）の世界観的扱い

## 必ず保持する正典（Canon — 削除不可）

- ジョブは探索者ギルドの**専門資格**。武器が戦闘スタイル、ジョブが役割を決める。上位・下位を設けず**すべて対等な基本職**。
- 基本 5 職: **ソードマン / レンジャー / ヴァンガード / アルケミスト / ビーストテイマー**（※実装済みは swordsman / ranger / alchemist。vanguard・beast_tamer は設計上の基本職）。
- 全職が探索者ギルド所属。共通制服をベースに専門装備で差別化。
- ジョブ進化（将来）は対等設計と矛盾しない＝到達後の上位形態であり初期選択の優劣を作らない。
