## この文書の責務

世界観を「どう届けるか」を定義する。設定そのものではなく、開示の媒体・分量・文体の規則。

## アウトライン（ChatGPT 執筆）

1. 基本方針（説明しすぎない）
2. 情報の量（断片の設計）
3. 媒体別ロール（武器説明 / 古文書 / 遺物説明 / NPC会話 / ダンジョン発見度 / 図鑑）
4. 発見ループ（断片→推測→次の発見）
5. ライティングルール（トーン・人称・禁則）
6. ライティング例（良い例 / 悪い例）

## 必ず保持する正典（Canon — 削除不可）

- ムービーや長文会話で説明せず、**武器説明文 / 古文書 / 遺物説明 / NPC会話 / ダンジョン発見度** から断片的に開示する（`game/03_世界観` 開示方針）。
- 中核の謎（第十の王 / 王冠 / 世界樹 等）は **断定しない**。
- プレイヤーを「選ばれし英雄」として描かない。

- UI 仕様 → `game/10_UI_UX.md`
