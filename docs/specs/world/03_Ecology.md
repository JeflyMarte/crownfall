## 1. モンスターとは（生物由来の原則）

現在「モンスター」と呼ばれている生物は、特別な種族ではない。

彼らは魔界の住人でも、不死者でもなく、すべて現実の生物を祖先に持つ生命である。

狼、熊、鹿、蜘蛛、蛇、鳥、昆虫、魚、爬虫類。

人類が文明を築いていた時代にも存在した生物たちは、長い静寂の時代を経て、それぞれの土地へ適応し、新たな姿を獲得した。

その変化は地域ごとに異なり、同じ祖先を持つ生物であっても、異なる土地では全く異なる生態を示すことがある。

この世界では、アンデッド、悪魔、スライムなど、異世界を起源とする種族は基本的に登場しない。

未知に見える存在であっても、その起源は常に自然と生物の営みの中にある。

## 2. 変異の原因（地域ごとに異なる）

戦後の世界では、生物の変化に単一の理由は存在しない。

土地ごとに異なる環境が、長い年月をかけて異なる生態系を育み、その結果として多様な生物が生まれた。

変異の要因は地域ごとに異なり、一つの理論だけで世界全体を説明することはできない。

現在知られている例として、次のようなものが挙げられる。

| 地域 | 変異要因の例 |
|---|---|
| 王都跡 | 古代魔術装置の残留エネルギーが長期間環境へ影響したと考えられている。 |
| 白骸墓地 | 大量の死骸が積み重なったことで特殊な土壌が形成されたという説がある。 |
| 黒鉄鉱山 | 重金属や鉱石に富む環境が長い年月をかけて生態へ影響したとみられる。 |
| 深森 | 人類消失後、生態系そのものが極端に発達した地域と考えられている。 |

これらは現在有力とされる解釈の一部であり、新たな調査によって見直される可能性がある。

## 3. 生態系の考え方（食物連鎖 / 頂点捕食者）

生物は単独で存在しているわけではない。

植物、菌類、小型生物、中型生物、大型生物。

それぞれが環境と関わり合い、一つの生態系を形作っている。

探索者が遭遇する危険な生物も、その土地だけを切り離して理解することはできない。

どのような植物が育ち、どの生物を捕食し、どの季節に移動するのか。

そうした積み重ねが、生態系全体の姿を決めている。

各地域には、生態系の均衡を支える**頂点捕食者**が存在することがある。

彼らは単なる強敵ではなく、その土地の環境を象徴する存在であり、生態系全体の一部として位置付けられる。

---

## 4. デザインルール

戦後生態系における生物設計は、次の原則に基づく。

1. **敵はすべて生物由来である。**
2. **アンデッド・悪魔・スライムなど、異世界起源の存在は基本的に登場させない。**
3. **地域ごとに独自の生態系を持ち、同じ祖先でも異なる姿へ適応し得る。**
4. **ボスは地域生態系の頂点捕食者として位置付ける。**
5. **外見・素材・能力は、その土地の生態や環境から説明できるものとする。**

モンスターの設計は、個体から考え始めない。

常に次の順序で世界を組み立てる。

> **環境 → 生態系 → モンスター**

一体の生物は、その土地全体の結果として存在する。

---

## 5. 図鑑・調査との接続（概要）

探索者の仕事は討伐だけではない。

どの土地に、どのような生物が生息し、何を食べ、どのような環境で暮らしているのかを観察し、記録することも重要な役割である。

生態図鑑は、単なる生物一覧ではない。

それは地域ごとの生態系を理解し、人類が失った世界を再び結び直していくための記録である。

一体の生物を知ることは、その土地を知ることにつながる。

そして土地を知ることは、戦後の世界そのものを理解する第一歩となる。
