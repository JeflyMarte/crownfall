## この文書の責務

戦後生態系の総論。モンスター観の原則・変異要因・設計ルールを定義する。分類体系・各論は下位文書へ委譲する。

## アウトライン（ChatGPT 執筆）

1. モンスターとは（生物由来の原則）
2. 変異の原因（地域ごとに異なる）
3. 生態系の考え方（食物連鎖 / 頂点捕食者）
4. デザインルール
5. 図鑑・調査との接続（概要。詳細は図鑑システム spec）

## 必ず保持する正典（Canon — 削除不可）

- モンスターは魔界の住人でも不死者でもなく、**すべて現実の生物を祖先に持つ**（狼/熊/鹿/蜘蛛/蛇/鳥/昆虫/魚/爬虫類 等）。
- **アンデッド・悪魔・スライム等の異世界起源種族は基本的に登場させない。**
- 変異は地域ごとに原因が異なり、地域ごとに全く異なるモンスターが生まれる。
- デザインルール: ①敵は全て生物由来 ②異世界起源を排除 ③地域固有の生態系 ④ボス＝地域の頂点捕食者 ⑤外見・素材・能力は生態で説明可能。
- 設計順は **環境 → 生態系 → モンスター**。
