## 1. モンスターとは（生物由来の原則）

現在「モンスター」と呼ばれている生物は、特別な種族ではない。

彼らは魔界の住人でも、不死者でもなく、すべて現実の生物を祖先に持つ生命である。

狼、熊、鹿、蜘蛛、蛇、鳥、昆虫、魚、爬虫類。

人類が文明を築いていた時代にも存在した生物たちは、長い静寂の時代を経て、それぞれの土地へ適応し、新たな姿を獲得した。

その変化は地域ごとに異なり、同じ祖先を持つ生物であっても、異なる土地では全く異なる生態を示すことがある。

この世界では、アンデッド、悪魔、スライムなど、異世界を起源とする種族は基本的に登場しない。

未知に見える存在であっても、その起源は常に自然と生物の営みの中にある。

## 2. 変異の原因（地域ごとに異なる）

戦後の世界では、生物の変化に単一の理由は存在しない。

土地ごとに異なる環境が、長い年月をかけて異なる生態系を育み、その結果として多様な生物が生まれた。

変異の要因は地域ごとに異なり、一つの理論だけで世界全体を説明することはできない。

現在知られている例として、次のようなものが挙げられる。

| 地域 | 変異要因の例 |
|---|---|
| モーンゲート（王都アステリア地下） | 堆積した遺物・鉱物を体内へ取り込む鉱物化適応。地下に偏在する自然力（エルダ）の影響も指摘される。 |
| ヴェルディア（大森林） | 共生適応——菌糸・胞子・蔓との共生／寄生（`05 §3.2`）。 |
| ミストフェン（沼地） | 腐生適応——腐植土・死骸・沼泥の再利用（`05 §3.3`）。 |
| ブラックショア（海岸） | 潮汐適応——満潮と干潮で生息域が入れ替わる（`05 §3.4`）。 |
| フロストリッジ／ノースリーチ（北） | 寒冷適応——極寒・吹雪・氷河（`05 §3.5`）。 |
| レッドフォージ（廃鍛冶国） | 熱適応——星炉余熱・溶岩脈・金属蒸気（`05 §3.6`）。 |
| ストームクラウン砦跡 | 風適応——峡谷風・標高への適応（`05 §3.7`）。 |
| グレイハイランド（高原） | 風蝕適応——強風・砂礫・露岩（`05 §3.8`）。 |
| ブロークンマーシュ（湿地） | 断裂適応——地盤変動・旧街道金属・腐植の混在（`05 §3.9`）。 |
| レッドリッジ（岩山・鉱脈） | 重金属や鉱石に富む環境が長い年月をかけて生態へ影響したとみられる。 |
| サンダーピーク（火山地帯） | 高温と火山性の環境が、耐熱性の生態を育んだとされる（耐熱適応 / `05 §3.10`）。 |

これらは現在有力とされる解釈の一部であり、新たな調査によって見直される可能性がある。超自然や魔術による変異は確認されていない（自然力＝エルダについては §6）。

## 3. 生態系の考え方（食物連鎖 / 頂点捕食者）

生物は単独で存在しているわけではない。

植物、菌類、小型生物、中型生物、大型生物。

それぞれが環境と関わり合い、一つの生態系を形作っている。

探索者が遭遇する危険な生物も、その土地だけを切り離して理解することはできない。

どのような植物が育ち、どの生物を捕食し、どの季節に移動するのか。

そうした積み重ねが、生態系全体の姿を決めている。

各地域には、生態系の均衡を支える**頂点捕食者**が存在することがある。

彼らは単なる強敵ではなく、その土地の環境を象徴する存在であり、生態系全体の一部として位置付けられる。

---

## 4. デザインルール

戦後生態系における生物設計は、次の原則に基づく。

1. **敵はすべて生物由来である。**
2. **アンデッド・悪魔・スライムなど、異世界起源の存在は基本的に登場させない。**
3. **地域ごとに独自の生態系を持ち、同じ祖先でも異なる姿へ適応し得る。**
4. **ボスは地域生態系の頂点捕食者として位置付ける。**
5. **外見・素材・能力は、その土地の生態や環境から説明できるものとする。**

モンスターの設計は、個体から考え始めない。

常に次の順序で世界を組み立てる。

> **環境 → 生態系 → モンスター**

一体の生物は、その土地全体の結果として存在する。

---

## 5. 図鑑・調査との接続（概要）

探索者の仕事は討伐だけではない。

どの土地に、どのような生物が生息し、何を食べ、どのような環境で暮らしているのかを観察し、記録することも重要な役割である。

生態図鑑は、単なる生物一覧ではない。

それは地域ごとの生態系を理解し、人類が失った世界を再び結び直していくための記録である。

一体の生物を知ることは、その土地を知ることにつながる。

そして土地を知ることは、戦後の世界そのものを理解する第一歩となる。

## 6. 自然力（エルダ）と属性 — 魔法の不在

この世界に、超自然の「魔法」は存在しない。

戦闘で見られる炎や氷、雷、あるいは「呪い」や「浄化」と呼ばれる現象は、いずれも超自然の力ではなく、**自然力＝エルダ（Elda）** の働きとして説明される。

### エルダとは

**エルダ（Elda）** は、エルド大陸に遍在する自然力の総称である。物質・生物・遺物・環境に宿り、引き出して利用できる。神や魔界に由来するものではなく、自然現象・物質反応として観測される。

エルダには偏り（相）があり、探索者ギルドはこれを五つの **属性** として整理している。属性名・分類はゲーム仕様（`game/27_状態異常と属性.md`）と一致させる。

| 属性 | エルダの相 | 世界観上の正体 |
|---|---|---|
| 炎 | 炎のエルダ | 燃焼・発火。可燃物や発火触媒による自然現象。 |
| 氷 | 氷のエルダ | 冷気・凍結。冷却反応や寒冷環境の現象。 |
| 電気 | 電気のエルダ | 放電。蓄電性の鉱物や自然の雷。 |
| 闇 | 闇のエルダ＝**瘴気** | 腐敗・汚染の気。朽ちた遺物や腐敗環境が放つ毒気。 |
| 聖 | 聖のエルダ＝**浄化の光** | 瘴気や異常な個体に作用する、特定鉱物・先王時代の浄化技術の名残。神の力ではない。 |

### 錬成と「杖」

アルケミストが扱う「魔法」も例外ではない。

それは呪文ではなく、**エルダを物質的に扱う技術＝錬成**（触媒・薬品・反応）である。杖は呪具ではなく、エルダを引き出し制御する **触媒器** にあたる。武器に宿る属性は、その素材に宿るエルダである。

### 状態異常との関係

状態異常の多くは自然由来である。毒・炎上・冷却・感電・スタンは、生物や環境がもたらす自然な作用にすぎない。

「**呪い**」もまた超自然ではなく、**闇のエルダ（瘴気）による汚染・衰弱**として説明される。

### 弱点の考え方

ある生物が特定の属性に弱いのは、その身体構造や適応の結果である。たとえば鉱物化適応で水晶質を帯びた個体は放電（電気）に脆く、寄生する鉱物性の生命は浄化の光（聖）に弱い、といった具合に、弱点は生態から説明できる。

> エルダは「魔法の言い換え」ではない。あくまで自然力であり、観察・記録・利用の対象である。神話的・超自然的な断定は避ける（`10_LoreDelivery`）。

### 7.1. 適応軸の対照

| 適応軸 | 基質 | 主 Biome | 頂点個体の俗称例 | 味方有利属性（案） | 主な状態異常 |

| 共生 | 生きた有機 | 囁きの森 | フローラベア・グランヴェル | fire | poison |
| 腐生 | 死せる有機 | 霧沼 | 底なしの王モルドガル | thunder | poison, bleed |
| 潮汐 | 塩・干満 | 沈没航路 | 潮鳴王ネレイオン | holy | mark, chill |
| 寒冷 | 氷・吹雪 | 最果て氷裂 | 始祖の竜エルディオン | ice | chill, stun |
| 熱 | 炉・金属蒸気 | 紅炉峡谷 | 熔鉄竜オルグラン | ice | burn, stun |
| 風 | 峡谷風・標高 | 嵐冠砦跡 | 嵐鎧竜バスティオン | earth | stun |
| 風蝕 | 強風・砂礫 | 灰風高原 | 灰風王ガレオン | earth | stun |

**着生の系譜:** 鉱物化（無機取り込み）と、Class VII 頂点の **環境着生複合体**（風化城壁・潮藻・炉金属・風化岩など）は別概念。前者は雑魚〜Elite の適応、後者は古代種の骨格／生体に **土地の物質が長年着生** した頂点型である（`05 §3.11` Boss 群）。

### 7.2. Class VII 頂点個体の共通型

メイン6 Biome＋寄り道4区画の Boss は、いずれも次の原則を共有する。

1. **祖先は実在生物系統**（古龍種・古代型ワニ・巨鯨等）。アンデード・悪魔ではない。
2. **王の名は民間の俗称** であり、王そのものではない（`04_Classification §5.5`）。
3. **不死者ではない** — 討伐・観察の対象は生物として記録する。
4. **環境の象徴** — その土地の適応軸が最も強く現れた個体（または複合体）。

着生複合体の例: セルディオン（骨＋結晶寄生）／グランヴェル（熊＋森フローラ）／ネレイオン（鯨＋潮藻）／オルグラン（古龍＋炉金属）／バスティオン（飛竜＋風化城壁）／ガレオン（古龍死骸＋風化岩）／フィデスロア（ワニ＋崩橋石造）／エンバーロア（耐熱古龍＋火山鉱物）。

### 7.3. 状態異常の由来（混同防止）

| 状態異常 | 典型の由来 | 代表 Biome | 記録上の注意 |
|---|---|---|---|
| poison | 胞子・神経毒 | 囁きの森 | 菌共生由来。沼毒と区別 |
| poison | 沼毒・病原 | 霧沼・断盟湿地 | 腐生・湿地の生物化学 |
| bleed | 吸着・蛭・爪 | 霧沼・モーンゲート | 出血止まりにくさは自然作用 |
| mark | 標的・潮の読み | 沈没航路 | 狩りの仕留め行動 |
| chill | 海水・吹雪 | 沈没航路・最果て氷裂 | 寒冷適応の副産 |
| burn | 熱傷・噴気 | 紅炉・黒曜禁足地 | 炉熱と火山熱は別軸 |
| stun | 風圧・金属共鳴 | 嵐冠・灰風・紅炉 | 平衡感覚・聴覚への自然作用 |

敵→味方への **curse 付与は行わない**（闇のエルダ＝瘴気は環境・素材の話であり、生物の「呪い攻撃」ではない / `game/27` P3-D024d）。

### 7.4. エルダ偏在（観測メモ）

| 環境 | 偏りやすいエルダの相 | 民間の俗称例 |
|---|---|---|
| 地下湿潤・遺物堆積 | 電気・闇（瘴気） | 魔導触角・呪われた鉄 |
| 過剰繁茂・菌糸 | 闇（澱みの季）・毒 | 魔毛・瘴気 |
| 沼沢・腐植 | 闇（沼気）・電気（帯電） | 瘴気・沼の呪い |
| 潮間帯・灯台遺構 | 聖・氷（夜霧） | 潮の加護 |
| 極寒・氷河 | 氷 | 霜の息 |
| 炉・溶岩 | 炎・聖（浄化の名残） | 炉の加護 |
| 火山噴気 | 炎・闇（硫化） | 地の息 |
| 強風・露岩 | 電気（雷気）・土 | 風の加護 |

いずれも **観測と俗称** であり、神や魔法の介入ではない（§6）。
