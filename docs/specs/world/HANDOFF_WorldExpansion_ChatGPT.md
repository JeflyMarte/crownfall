# Crownfall 世界観 拡張ハンドオフ（ChatGPT 用）

**用途:** この1ファイルをそのまま ChatGPT に渡し、世界観の続きを考えさせるための「現状まとめ＋作業指示」。
**出力の扱い:** ChatGPT の成果は後日 Cursor HQ（実装判断者）へ戻し、採否・実装を決める。**ChatGPT は確定者ではなく提案者。**
**重要:** 本ファイルは **プレイヤー公開可能な範囲のみ** を含む。中核の謎の「答え」は意図的に載せていない（後述のガードレール厳守）。

---

## 0. このプロジェクトは何か（30秒要旨）

- **Crownfall** — 2D見下ろし型・自動探索ハクスラRPG（Godot 4 / モバイル縦持ち想定）。
- プレイヤーは英雄ではなく、**探索者ギルドの調査隊を率いる隊長**。冒険者を直接操作せず、方針・装備・編成を決める。
- 舞台は、統一王権が倒れた後の世界。**「倒れたが、滅びてはいない」**——再起と継承がテーマ。
- 世界の3本柱: **歴史（失われた文明）／遺物（過去の証）／生態（戦後に再生した自然）**。

### 文体・世界の大原則（必須）

1. **奇跡を起こす神はいない。** 信仰は「人が意味を見いだす文化的営み」として描く。
2. **魔法ではなく「エルダ（Elda）」。** 発光・幻覚・放電・霧などの不思議な現象は、すべて **自然現象（エルダ）** として説明する。「魔法」「呪い」「術式」は民間の俗称としてのみ可。
3. **断定を避ける伝承体。** 多くは「〜と伝わる」「諸説ある」「確かめようがない」。記録は互いに食い違ってよい。
4. **生物は実在生物を祖先に持つ。** モンスターは「魔物」ではなく、環境に適応して分化した生物。

## 1. 歴史の骨子

### 時代区分

先王時代（伝承・超古代）→ 九王時代 → 王国時代（最盛）→ **九王戦争**（崩壊）→ 静寂の時代（長い空白）→ **探索者の時代（現在）**。

### 崩落（the Fall）と暦

- **崩落** = 王国時代を終わらせた九王戦争で、首都 **王都アステリア** が失われた出来事。作品名 *Crownfall* の由来。
- 長期年代 = **崩落後（A.F.）**、実用暦 = **ギルド暦（G.）**。**現在 G.118 ≒ 崩落後およそ300年（諸説）**。
- 1年=12ヶ月=360日、1ヶ月=3旬（1旬=10日）。
- **澱みの季（よどみのとき）** — 晩秋〜初冬、地下や遺構で闇のエルダ（瘴気）が高まる危険期。

### 伝承年表（復元・諸説）

崩落前約20年「最後の盟議」決裂 → 街道寸断 → 「灯断ちの年」 → **崩落（アステリアの長夜）** → 静寂の時代 → 崩落後約180年 ギルド前身発足 → 現在。

## 4. 地理・都市

- **拠点 = アイアンヘイブン**（戦後新設の探索者都市。王都アステリアの近く）。
- 都市: 王都アステリア（廃墟・地下にダンジョン「モーンゲート」）／シーゲート（港・海統王）／フロストウォール（山岳・開拓王）／グリーンホロウ（森の集落・森護王）／レッドフォージ（鍛冶都市・廃墟・鍛冶王）。

## 5. 社会・文化（ギルド外）

- **都市自治**（都市評議会）。統一王権なし。街道（王の大街道）が社会の動脈。
- 派閥: **車輪同盟**（隊商互助）／**灯守組合**（街道の灯・宿駅。中立調停者）／**灰狼団**（廃墟荒らしの無法者）。
- 文化: **継承祭**（灯を継ぐ年中行事）／童歌「九王数え唄」「灯継ぎ唄」／ことわざ「倒れたが滅びていない」「遺産を持ち帰るな、歴史を持ち帰れ」。
- 通貨は Gold（ゲーム実装通貨）。素材・クラフト・市場は世界設定上の営みだが、現行ゲームには未導線。

## 6. 信仰・死生観

- **灯火の信仰** — 記憶と継承を絶やさぬため灯を守る民間信仰。担い手＝**灯番**、伝える者＝**巫女**（聖職者ではない）。
- **継承の信仰** — 九王以前に継承を司ったとされる伝説的存在 **白王**（白王の神殿）。「受け継ぐことを尊ぶ」。
- **死生観** — 人は死ぬと自然へ**還る**（**還りの儀**）。死者の営みを**記録**として残すことを重んじる。
- 地域: シーゲート＝潮の祈り／グリーンホロウ＝森への礼（翠の盟約）／フロストウォール＝灯絶やしの戒め。

## 7. 生態・モンスター

- 生物は **Class I〜VII**（獣・鳥・爬虫・昆虫・水棲・菌植物・古代種）で分類。同祖先でも環境で分化。
- 生態系単位 = **Biome**。確定2つ:
  - **モーンゲート（嘆きの地下水路）** = 王都地下。適応テーマ「**鉱物化適応**」（鉱物・古代金属を体内に取り込む）。弱点 thunder/holy 系。
  - **囁きの森（ウィスパーウッド／ヴェルディア）** = 構想確定。適応テーマ「**共生適応**」（菌糸・胞子・蔓と共生）。弱点 fire・poison 中心。
- **古代種（Class VII）/ 伝説個体** — 巨大で長命なだけの生物（神でも不死者でもない）。例: 水晶骸竜セルディオン（モーンゲートのボス）／フローラベア・グランヴェル（囁きの森）。古龍種＝姿を変えず生き延びた大型飛行爬虫類。
- **重要: モンスターの固有名はオーナー確定事項。** ChatGPT は新規モンスター名を「確定」しない（提案は可、ただし仮称明記）。

## 8. 遺物・武器（概要）

## 10. ⚠ 創作ガードレール（最重要・厳守）

ChatGPT は以下を **絶対に守る**こと。違反した提案は不採用になる。

### A. 答えを出してはいけない謎（恒久未確定）

次の主題には**結論・正体・真相を創作しない**。深めてよいのは「問い」だけ。

- **灯火の正体**（何が・いつ・なぜ灯ったのか）。
- **世界樹**（一本か複数か、総数・正体）。

### B. 触れてよいが「真相」を断定しない謎

存在を匂わせる断片は可。だが**答えを書かない**（これらは作者が内部管理している）。

- 九王戦争の本当の原因。
- 「第十の王」「第十王遺産」など、九を超える継承の核心。
- 王冠の起源（「これは九王の遺産ではない」の意味）。
- 9対の伝説武器に共通する紋様の理由。

> これらは「失われた／諸説ある／記録が食い違う」として、**問いのまま**提示する。説明・解決編を書かない。

### C. 命名・確定の権限

- **モンスター名・王遺産名・伝説武器名・スキル名は確定しない**（オーナー専権）。提案する場合は必ず「（仮称）」と明記。
- それ以外の新規固有名（地名・NPC・派閥・祭り・料理・言い回し等）は提案してよいが、**§9 と重複しない**こと。

### D. 整合ルール

- 神・魔法・超自然を実在の力として描かない（§0）。
- 既存の都市・九王・九英雄・年表・信仰と矛盾しない。
- 1テーマ1まとまりで、責務を混ぜない。

## 11. 拡張してほしいテーマ（候補・自由に選択/追加可）

1. **言語・古代文字（Elder Script）** — 読み書きが失われた世界の言語観。ルーン状紋様、地名接尾語、人名の傾向。
2. **各都市の暮らし** — シーゲート/フロストウォール/グリーンホロウの住人・名物・年中行事・服飾・食。
3. **NPC・人物** — 都市評議員、隊商頭、灯守、灰狼団の首領、在野の探索者など（§9と非重複）。
4. **地方伝承・怪談** — 各地域の言い伝え、迷信、語り物。
5. **交易・経済の文化** — 隊商の品、街道の宿、物々交換の作法。
6. **探索者の生活** — ギルド支部の日常、新人の通過儀礼、隊の符牒。
