## 1. 社会構造（戦後世界の枠組み）

九王戦争と王都アステリアの陥落で中央政権は失われた。現在の人類社会は、**単一の国家ではなく、生き延びた都市と街道網が緩くつながる連合体**である。

- **都市自治:** 現存都市（シーゲート／フロストウォール／グリーンホロウ等）はそれぞれ **都市評議会** が治め、上位の王権は存在しない。
- **街道が国境を兼ねる:** 巡礼王ルミナスの遺した **王の大街道** が都市を結ぶ唯一の動脈。街道の安全＝社会の安全であり、街道を握る者が実質的な力を持つ。
- **探索者ギルドの位置:** ギルドは特定都市の所有物ではなく、街道網を横断する超都市的な組織として、調査・流通・記録を担う（詳細 `08`）。
- **拠点アイアンヘイブン:** 戦後新設の探索者都市。王権の空白を埋める「事実上の中心」として機能するが、王を名乗る者はいない。

## 2. 派閥・勢力（ギルド外）

| 勢力 | 性格 | 拠点・縄張り | 縁・備考 |

| **灯守組合（ともりもりくみあい）** | 街道の灯・宿駅の管理 | 街道の宿駅・辻 | 道々の灯を絶やさぬ職能集団。信義王アウレクスの盟約精神と「灯の信仰」（`04`）に連なる |
| **灰狼団（はいろうだん）** | 廃墟荒らし・無法者 | 廃墟・未管理区域 | 王都やレッドフォージ等の遺物を盗掘し闇で売る。ギルド・車輪同盟と対立 |

> 個々の評議員・隊商頭・首領などの人物は `13_Characters` を参照。

### 2.2. 灰狼団の三層構造

表向きの **灰狼団** の背後に、より組織化された **狼煙会（ろうえんかい）** があると、記録部は整理している。

| 階層 | 通称 | 役割 |
|---|---|---|
| 表 | 灰狼団 | 街道の盗賊。噂話・恐怖の対象 |
| 中 | 狼煙会 | 廃墟専門の盗掘ネットワーク。隊商・闇市と接続 |
| 奥 | 摘冠組（かんりょうぐみ） | 王都地下・紅炉峡谷などから高価値遺物を持ち出す極小人数の専門班 |

首領とされる **摘冠者ヴォルグ**（`13_Characters`）は、表の灰狼団の顔として語られることが多い。

### 2.3. 力学（緊張関係）

- **車輪同盟 ↔ 灰狼団:** 街道の安全 vs 略奪。交易路の護衛を巡る恒常的対立。
- **ギルド ↔ 都市評議会:** 調査の自由 vs 都市の管轄。遺物の帰属・立ち入り許可で交渉が生じる。
- **灯守組合:** 中立的調停者。灯を絶やさぬ作法が「敵味方を問わず迎える」規範を生み、しばしば仲介に立つ。

### 2.4. 車輪同盟の看板 — 連環鉄車（れんかんてっしゃ）

街道を走る **荷車の連結列車**。最大12両。中央に灯守の灯籠、末尾に武装の護衛乗車。街道の半分を制した隊商だけが編成を組める、と子どもたちは憧れる。

- 灰狼団襲撃の定番舞台。車輪同盟 ↔ 灰狼団の対立が最も可視化される
- **継承祭** の日だけ、全車の灯籠に新しい灯が一斉に灯る（S-01 の視覚的フック）

首長 **「大輪の」ハーヴィ**（`13_Characters`）が連環鉄車の編成権を握り、街道交易の実務を率いる。

### 2.5. 灯守組合の作法

座長 **灯台守ロウラ**（`13_Characters`）が組合の最高責任者。街道の灯・宿駅を守り、「灯を分かつ者を害すな」を最後の秩序として維持する。敵対する派閥のあいだでも、灯の前では武器を下ろす——この作法が灰狼団ですら守る理由のひとつとされる。

### 2.6. 闇市 — 潮下市（うしおしたいち）と灯の届かぬ取引

正規の遺物流通はギルドと車輪同盟が担うが、その外側に **灯の届かぬ市** がある。

| 名称 | 拠点 | 内容 |
|---|---|---|
| **潮下市** | シーゲート港の影（潮見の市の地下・桟橋裏） | 海域由来の遺物・来歴不明の炉印同型品（S-02）が流れる。港長マレーナは「存在を知るが公には触れない」 |
| **灰市** | アイアンヘイブン郊外・街道外れ | 灰狼団／狼煙会経由の盗掘品。セルマが時折目利きに呼ばれるが、公認はしない |

灯守組合は闇市を「灯を点けない場所」として記録するだけで、踏み込まない。ギルドは回収の正規ルートを維持することを優先する。

## 3. 暮らし・文化

### 3.1. 価値観

戦後世界の人々を貫くのは **「倒れたが滅びていない」** という王冠の銘に象徴される再起の精神である。失われた文明を悼みつつ、廃墟から拾い、次へ手渡すことを尊ぶ。継承王カエルムの「次の者へ」の理念が、王権なき後も生活倫理として残る。

### 3.2. 祭り・暦

- **継承祭（けいしょうさい）:** 年の節目に行われる最大の祭り。各家・各都市が前年の灯を新しい灯へ「継ぐ」儀礼を行う。「灯の信仰」（`04`）と継承王の理念が融合した行事で、ギルド・車輪同盟・灯守組合のいずれもこの日は手を止める。

**継承祭の儀式（概要）:**

1. **夕刻** — 各家庭が前年の灯を掲げ、街道の灯守が巡回して灯油を分ける
2. **夜** — 灯継ぎ唄（`§3.5`）を歌い、古い灯芯を新しい灯芯へ移す。古い灯芯は燃やさず、灰として盟約の地か海へ還す地域もある
3. **深夜** — 連環鉄車の全車灯が一斉に灯る（車輪同盟・ハーヴィ）
4. **明け方** — ギルド本部で新年度の調査計画が公示される。灯印の更新もこの日に行われることが多い（`08 §9`）

- **暦:** 公的な統一暦は失われ、人々は王都陥落を起点に数える非公式の年数で時を語ることが多い（厳密な紀年法は未確定）。

### 3.3. 食・物資

- 街道交易（車輪同盟）が各地の特産を運ぶ。北のフロストウォールの保存食、西海岸シーゲートの海産、ヴェルディアの森の幸が、アイアンヘイブンの中央市場に集まる。
- 通貨は Gold が広く通用する（現行ゲームの実装通貨。物々交換・素材取引は世界設定上の営みで未導線 / `08 §10/§12`）。

### 3.4. 記録と語り

- 文字記録の多くは陥落で失われ、口承・写本・遺物の銘が知識を伝える。ギルドの **調査記録（図鑑）** はその再構築の試みである。
- 航海者マレクの航海録、アステリア陥落録（`13_Characters`）など、英雄ゆかりの記録が断片的に流通する。

### 3.5. 童歌・ことわざ・迷信（民間伝承）

文字を失った戦後世界では、知識と教訓の多くが歌・言い回し・言い伝えの形で受け継がれている。いずれも史実の証拠ではなく、生活の知恵や記憶の器として機能する。中核の謎には踏み込まない。

#### 童歌（わらべうた）

- **九王数え唄（くおうかぞえうた）** — 子どもが九王の称号を覚えるための数え唄。「ひらく・まもる・しるす・ちかう／めぐる・きたう・うみおさめ／もりまもり・つぎのもの」と、開拓・守護・学識・信義・巡礼・鍛冶・海統・森護・継承の九王を順に数える。順番に意味はなく、覚えやすさで並ぶ。
- **灯継ぎ唄（ひつぎうた）** — 継承祭の夜にうたわれる短い歌。「消える前に、次へ渡せ」と繰り返す。灯の信仰（`04`）と継承王の理念が、遊びの中に溶け込んでいる。

#### ことわざ・言い回し

| 言い回し | 意味・由来 |
|---|---|
| 倒れたが滅びていない | 王冠の銘から。挫折しても終わりではない、の意。最も広く知られる |
| 遺産を持ち帰るな、歴史を持ち帰れ | 探索者ギルドの理念（王遺産の守り手エレナス由来 / `08`） |
| 灯を分かつ者を害すな | 灯守の作法から。旅人同士の最低限の信義を説く（`§4`） |
| 澱みの季に深くは潜るな | 晩秋〜初冬の瘴気活性（澱みの季 / `01 §7`）への戒め |
| 街道を外れれば、誰の灯も届かぬ | 無法地帯＝自己責任の世界、の意（`§4`） |
| 盟約の橋を渡るな、車は沈む | 盟約崩れ橋を避ける車輪同盟の経験則（S-03） |

#### 迷信・言い伝え

- 古龍を見たら名を呼ぶな——名を呼べば追われる、という北の言い伝え（実証されない / `04 §5`）。
- 時計の止まった部屋では時刻を口にするな——時を狂わせる、とされるが、実際はクロックモスの羽音など自然要因とされる（`01 §7`）。
- 継承祭の夜に灯を絶やした家は、その年の幸運も絶える——灯を大切にする習俗を支える戒め。
- 灰狼団が現れる前は、街道の鴉が鳴きやむ——盗賊の接近を察する経験則として語られる。

### 3.6. 探索者文化 — 刻印（エングレーブ）と認定決闘

ギルドの **上級専門資格**（到達形認定 / P3-D048）を受けた探索者だけが、認定所で肩・手甲に刻む **調査の印（刻印）**。魔法ではなく、焼印と鉱物染料の技術。

| 刻印 | 意味 | 備考 |
|---|---|---|
| 一線（ファースト・ライン） | 初認定 | 細い一本線 |
| 三踏（スリーフット） | 3 Biome 踏破記録 | 三角形の踏痕 |
| 深層（ディープ） | 地下深層の深度記録 | 縦の短い刻み |
| 古種（エンシェント） | Class VII 個体の調査記録 | 竜骨の簡略紋 |

刻印の多い探索者は、宿の辻灯亭で無言の敬意を払われる。

**認定決闘（けいとうとう）** — 傷つけ合う決闘ではなく、同一 Biome の **模擬踏破タイム** を競う慣習。「お前、モーンゲート何分？」は探索者同士の定番の口論。

**渡し守（わたしもり）** — 街道の断絶地点（盟約崩れ橋など）で、綱渡りや小舟により旅人を渡す職。車輪同盟の荷車が通らない区間では、彼らだけが街道を繋ぐ。トーン（`13`）が代表例。

### 3.7. 競合と名声 — 黒冠隊

ギルド公認の **黒冠隊（ブラッククラウン）** は、同じ調査目標を狙うライバル探索隊（`13_Characters`）。悪役ではなくプロの競合として、名声と記録の先を争う。

## 4. 法と無法

- **街道法（慣習）:** 統一法典はなく、街道沿いでは「灯を分かつ者を害さない」という灯守由来の慣習が最低限の秩序を保つ。
- **遺物の扱い:** 正規の遺物回収はギルド認定を要する。無認可の盗掘は灰狼団の領分とされ、流通は闇市に限られる。
- **無法地帯:** 街道から外れた廃墟・未踏査区域は誰の管轄も及ばず、自己責任の世界となる。
