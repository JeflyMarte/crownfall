# 15 — 現存都市と避難港の厚み

---

## アイアンヘイブン（探索者都市・拠点）

- **成立:** 崩落後、王都外郭の資材と街道の結節から生まれた戦後都市
- **気風:** 実務・記録・新規参入に比較的開いている。王を名乗らないことを誇りにする層と、空位を悲しむ層が同居
- **街区（追加）:**

| 街区 | 内容 |
|---|---|
| **記録坂** | ギルド記録部・写字屋・インク屋。坂の上ほど古い書類の匂い |
| **釘市通り** | 鍛冶・装備・鍛冶屋への材木と釘。赤鉄の工房への導線ロア |
| **下湯（したゆ）** | 公衆浴場と噂の交換所。隊長名の噂が先に広まる場所 |
| **灰縁（はいぶち）** | 王都側を望む外壁沿い。夜に灯りを減らす習慣 |
| **新参桟橋** | 川港。初めて来る調査隊の荷揚げ |

詳細な日常は `23_ironhaven_life.md`。

## シーゲート（港町）

- **気風:** 潮と契約。遅れは罪、嵐は人名
- **名所:** 潮見塔、塩の市場、沈んだ鐘（満潮でだけ鳴るという嘘／本当）
- **派閥温度:** 車輪同盟強め、灰狼団は港では表に出にくい
- **追加NPC枠:** 航路査定官、嵐名付け人、塩税吏

## フロストウォール（山岳都市）

- **気風:** 節約・保存・遠出への警戒（エルディオン歩き禁止文化）
- **名所:** 氷窖、狼煙台、白息門
- **食:** 干し肉、氷砂糖、発酵乳
- **迷信:** 名も無き継承者の小石を旅の護符にする

## グリーンホロウ（森の集落）

- **気風:** 返す・借りる・伐らない年を数える
- **名所:** 樹上の吊り橋、種子庫、返す杭の祠
- **対立:** ギルドの標本採取 vs 森のタブー
- **祭り:** 葉継ぎの夜（継承祭の森版）
