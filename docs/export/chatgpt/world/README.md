# Crownfall 世界観深掘りパック（ChatGPT 配布用）

**Status:** v1.0（2026-07-22・オーナー委任で拡充）  
**位置:** `docs/export/chatgpt/world/`  
**前提:** 本体 SSOT `docs/specs/world/` の公開層に接続。**内部正史（CANON_INTERNAL）は含めない。**

---

## このパックの性質

- 既存の三本柱（History / Relics / Ecology）を壊さない
- 固有名詞・土地・上位存在・怪物系譜を **大胆に追加**した「語れる厚み」
- 伝承・諸説・口承として書く（断定しすぎない）。真相の断定はしない
- ゲームβ実装と無関係な地名・存在も **ロアとして存在してよい**

## ChatGPT への渡し方

1. 親フォルダの `../README.md` システムプロンプト
2. 本 `README.md` ＋ `00_INDEX.md`
3. 相談テーマに応じて `10`〜`28` を追加

世界観専用の追加プロンプト例:

```text
あなたは Crownfall の世界観ライター兼デザイン相談相手です。
渡す Markdown は公開ロアの深掘りパックです。内部の「作者真相」は渡されていません。
新たな固有名詞を提案するときは、既存の九王・都市・Biome 名と衝突しないこと。
モンスターは魔物・悪魔・アンデッドではなく生物由来（戦後生態系）で考えてください。
上位存在は「伝承上の存在／先王時代の痕跡」として扱い、即実装の敵にしないでください。
```
