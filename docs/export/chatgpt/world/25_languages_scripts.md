# 25 — 言語・書体・符丁・職人印

---

## 1. 言語層（簡易）

| 層 | 名 | 使い手 | メモ |
|---|---|---|---|
| 通語 | **エルド通語** | ほぼ全域 | ゲームUIの日本語／英語の世界内対応 |
| 古語 | **王統古語** | 碑文・法令残滓 | 崩落以降は読み手が減った |
| 職能語 | **炉言葉** | 鍛冶 | 温度を音階で言う |
| 職能語 | **潮言葉** | 船乗り | 嵐に人名、方位に内臓比喩 |
| 地縁語 | **杭言葉** | 森・沼 | 返す／借りるが動詞の核 |
| 符丁 | **灰符** | 灰狼・白い手袋 | 牙落書き＋数字 |

先王時代の「工作主の印」は言語というより **幾何**。読めないことが特徴。

## 2. 書体

| 書体 | 印象 | 出どころ |
|---|---|---|
| **冠書** | 厳格・縦長 | 王都公文書 |
| **街道体** | 読みやすさ優先 | 里程標・宿駅 |
| **炉印体** | 角ばった記号 | オルグラン系鋳型 |
| **菌糸草書** | 流れる連なり | 森の契約杭 |
| **潮曲線** | 波状 | 海図 |

ルーンローチの甲殻「文字」は、人間の書体の模倣析出であり、意味を持たないことが多い（持っているという学派あり）。

## 3. 職人印・スタンプ

| 印 | 意味 |
|---|---|
| 車輪半分 | 車輪同盟護衛済み荷 |
| 蓋灯 | 灯守通過・中立宿駅 |
| 星屑三点 | 星屑同業の純正主張（偽印多い） |
| 牙 | 灰狼の「触るな／触れ」文脈依存 |
| 余白の「？」 | 余白学会の未確認スタンプ |

## 4. UI／フレーバーへの使い方

- 武器説明の一文を炉言葉調に
- 野外速報の文言に潮言葉を混ぜる（海イベント時）
- 図鑑の古い頁だけ王統古語っぽい硬さ
