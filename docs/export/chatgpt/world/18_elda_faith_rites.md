# 18 — エルダ・信仰・暦・祭礼

---

## 1. エルダ（自然力）再掲＋民間理解

| 相 | 学者の言い方 | 民間の言い方 |
|---|---|---|
| 炎 | 酸化と熱の偏り | カル・ヴェスラの息 |
| 氷 | 熱の欠落・結晶 | イル・サーガの足跡 |
| 電 | 電位の奔流 | トランド・アズの羽音 |
| 闇 | 瘴気・沈積有機 | ウムブラ・ネルの群れ |
| 聖 | 浄化・光触媒的比喩 | ルクス・アエオンの灯 |

学者は擬人化を嫌う。野外隊員は擬人化で生き残る。

## 2. 信仰の層

| 層 | 内容 |
|---|---|
| **灯の信仰** | 正典中心。継ぐ・絶やさない。灯守組合 |
| **相主信仰** | 五相主への機嫌伺い。童歌と迷信 |
| **白き忌避** | 白王を崇めるというより、名を避けて距離を取る |
| **祖先の杭** | 森・沼の地縁信仰。返す・沈める・杭を立てる |

組織化された国教はない。都市ごとに混ざる。

## 3. 暦（拡張）

正典: 崩落後 A.F. / 崩落前 B.F.、現在おおよそ A.F.300（G.118 表記と並立するギルド年号あり）。

### 季節の通称（民間）

| 通称 | 目安 | 行事 |
|---|---|---|
| **継ぎ芽** | 早春 | 継承祭の準備 |
| **長照** | 晩春〜夏 | 街道交易の盛り |
| **胞子月** | 森・沼の特異期 | グリーンホロウは外出制限 |
| **灯薄** | 秋 | 灯守が忙しい |
| **白息** | 冬（北） | フロストウォールは門を早めに閉じる |
| **潮狂い週** | 不定 | 海側だけが数える嵐の週間 |

### 記念日（追加）

| 名 | 内容 |
|---|---|
| **継承祭** | 正典。最大の祭り |
| **杭返しの日** | 森民。借りたものを返す／返せぬものは杭に結ぶ |
| **無名の灯夜** | 名を残さなかった継承者へ。無銘の小石に灯を置く |
| **最後の盟議の追悼** | アイゼンプレインで静かな集会（参加は任意・少ない） |
| **炉冷まし** | 鍛冶。一年で最も火を弱める日。事故の記憶 |
