# 23 — アイアンヘイブンの日常

プレイヤーが毎回帰る街の「生活の手触り」。

---

## 一日のリズム

| 時間帯 | 街の様子 |
|---|---|
| 明け方 | 記録坂で写字屋が火を起こす。インクが温まる匂い |
| 朝 | 釘市通りが開く。鍛冶の一礼。新参桟橋に荷 |
| 昼 | 下湯が混む。隊の噂、野外速報の話 |
| 夕方 | 灰縁で王都側を見る人が増える。黙りがち |
| 夜 | 灯駅からの便り。ギルド深夜部（深記録派）が動く |

## 施設の裏設定（ゲーム施設と接続）

| ゲーム上の場所 | 街の呼び名 | 裏の一文 |
|---|---|---|
| 拠点ホーム | 中央環 | 王のいない広場。旗は都市評議会とギルドの二枚 |
| 赤鉄の工房 | 釘市の奥炉 | 星屑同業とは冷たい挨拶 |
| 招待状（ガチャ） | 封書局寄り | 「人を招く」部署。魔晶石は関税議論の種 |
| 図鑑 | 記録坂の公開棚 | 真作と噂が同じ棚に並ぶ危険 |
| ダンジョン選択 | 野外掲示 | 野外速報と並べて貼られる |

## 噂のサンプル（ニーナ／NPC用）

1. 「灰縁で昨夜、柱が二本に見えた人がいる」
2. 「下湯の湯が三日早く冷めた。炉冷ましが近い？」
3. 「新参の隊長が犬曳き台地の話をしたら、北の商人が席を立った」
4. 「白い手袋が古物商を再開した。王冠の話はするなよ」
5. 「マイルストーンの灯が二度消えた。灯守が喧嘩してるらしい」

## 食・物価感（雰囲気）

- 安い: 灰パン、川魚の干物、薄いスープ
- 普通: 森の茸鍋（季節）、港の塩漬け
- 高い: 北の氷砂糖、本物の炉印付き釘（実用品なのに高い）

## 隊長としての見え方

人々は隊長を英雄扱いしすぎない。  
「許可が下りた人」「帰って記録を出す人」。失敗しても生きて戻れば、下湯では席を詰めてくれる。
