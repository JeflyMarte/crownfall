# 12 — 九英雄深掘り

九英雄は王ではない。人々の営み側の象徴。本章は生涯断片・ゆかりの地・「見つかるもの」を追加する。

---

## 地平の探求者・アステル

- **生地伝承:** グレイハイランドの羊飼い集落
- **功績の核:** 「余白を恥としない地図」——未知を黒く塗り潰さず、点線で残す作法
- **ゆかり:** ノースリーチの石標「ここより先、地図なし」
- **見つかるもの:** アステル式折地図の写し、余白に筆圧の強い「？」
- **最期の諸説:** 帰らなかった／帰って無名で死んだ／地図ごと消えた

## 王遺産の守り手・エレナス

- **出自:** 中央王統の下級記録官とも、灰狼団の祖先とも（矛盾する伝承）
- **功績の核:** 所有から継承へ——倫理の転換
- **ゆかり:** アイゼンプレインの「返す石」（置いてはならぬ供物台）
- **見つかるもの:** 手引書『持ち帰るな』の断簡（後世の偽作も多い）
- **皮肉:** 灰狼団はエレナスを「偽善者」と呼ぶ歌を持つ

## 深淵を踏破した者・リュシアン

- **出自:** 王都の下層民、あるいは排水労働者の子孫
- **功績の核:** 恐怖の対象を調査対象へ
- **ゆかり:** モーンゲート上層の「リュシアンのチョーク印」（←印は後世の観光商売疑惑あり）
- **見つかるもの:** 深度日誌、呼吸の回数を数えた奇妙な表
- **現代:** ギルドの危険区域手順書に名が残る

## 灯火を継いだ巫女・イリア

- **出自:** 灯守の系譜、あるいは無名の旅巫女
- **功績の核:** 戦乱でも灯を絶やさない
- **ゆかり:** 灯断ちの年に消えない灯台があった、という話（場所不明）
- **見つかるもの:** 油のレシピ、祈りの短い句
- **現代:** 継承祭の中心像。実在より「役割」が強い

## 星屑の鍛冶師・ラグナ

- **出自:** オルド（星炉市）の徒弟
- **功績の核:** 星屑鍛造の再現可能化（秘伝から工法へ）
- **ゆかり:** レッドフォージの「ラグナの金床」——実際は後世のレプリカが多い
- **見つかるもの:** 炉温の歌（音階で温度を覚える口伝）
- **現代:** 赤鉄の工房の師匠筋が「ラグナ流」を自称しがち

## 失われた王都の記録者・セレス

- **出自:** アステリア王宮記録局
- **功績の核:** 崩落の夜を書き続けた
- **ゆかり:** 「長夜の頁」——焼跡のある紙片が市場に出回る（真偽不明）
- **見つかるもの:** 時刻と煙の向きだけが精密な日記
- **現代:** ギルドが真作と認定した断片はごく少数

## 竜と盟約を結んだ者・カイル

- **注:** 「竜」は異世界のドラゴンではなく、**巨きき祖／巨大生物**の古名
- **功績の核:** 討伐ではなく距離と盟約——観察距離の倫理
- **ゆかり:** ヴェルディア奥地の「盟約の枯木」
- **見つかるもの:** 巨大な爪痕の横に、人の掌痕が並ぶ岩
- **現代:** 生態調査の「不用意な討伐禁止」条項の祖と語られる

## 世界の果てを見た航海者・マレク

- **出自:** タレッサ（潮環港）の水夫
- **功績の核:** 帰還して記録を残した（行って帰らぬ英雄ではない）
- **ゆかり:** シャッタードアイルズ最西の「マレクの石錨」
- **見つかるもの:** 航海録の写し、星の角度表
- **現代:** シーゲートの酒場で「マレク嘘」（大げさな話）という言葉がある——愛憎両様

## 名を残さなかった継承者

- **名:** 意図的に無い（あるいは多すぎて特定不能）
- **功績の核:** 意思だけを託す
- **ゆかり:** どこにでもある無銘の墓、無銘の灯
- **見つかるもの:** 「次へ」とだけ刻まれた小石
- **現代:** プレイヤー（隊長）への鏡——名は自分で付ける

## 九英雄と九王の対応（民間の勝手な対応表）

学者は否定するが、童歌では一対一で覚える。

| 英雄 | よく結び付けられる王 |
|---|---|
| アステル | エルディオン |
| エレナス | カエルム／アウレクス |
| リュシアン | セラディス |
| イリア | ルミナス |
| ラグナ | オルグラン |
| セレス | カエルム |
| カイル | シルヴァリア |
| マレク | ネレイオス |
| 無名 | （白王／継承そのもの） |
