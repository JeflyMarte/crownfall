# 27 — 未踏査・地図の外・空と地下

実装ロードマップ外の「世界が続く感」。

---

## 1. 水平の外縁

| 噂の方角 | 内容 |
|---|---|
| 西のさらに西 | グレイヴシーの向こうに **第二の潮環** がある、というマレク偽作 |
| 北の白 | ノースリーチの先に **角座の都市**（人が住まない） |
| 東の雷 | イースタン・オーシャンを越えると **炉の大陸**（鍛冶王起源説と混線） |
| 南の黒 | ブラックシーの向こうは星しかなく、陸が「下」にあるという酔談 |

## 2. 垂直の外縁

| 方向 | 内容 |
|---|---|
| 地下 | モーンゲート最深の下——白き庭／双影の門／冠を負う脊 |
| 樹上 | 翠冠のさらに上、葉の上の海（霧） |
| 氷下 | 聴氷穴の先の「潮」 |
| 空 | 螺旋の空井戸が指す先。星屑の正体論争 |

## 3. 「別のエルド」仮説（ギルド内少数説）

崩落は大陸の終焉ではなく、**継承の切り替え**だった——世界は同じ地形のまま「次の版」に移った、という奇説。  
証拠は弱い。詩としては強い。中核の謎の答えとしては使わない。
