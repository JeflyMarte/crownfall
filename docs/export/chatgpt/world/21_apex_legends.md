## 1. メイン Boss を「生態の顔」として語る

| Boss | Biome | 生態的意味の一文 |
|---|---|---|
| セルディオン | モーンゲート | 地下の鉱物循環を閉じる最後の捕食者 |
| グランヴェル | ウィスパーウッド | 森が人を退出させたあとの「根の意志」に見える巨体 |
| モルドガル | ミストフェン | 沈下する土地の菌と獣の境界 |
| ネレイオン | ブラックショア | 潮が壊れた海の記憶を泳ぐ |
| エルディオン（ボス名） | フロストリッジ | ※開拓王と同名の畏れ——王の名を生物に被せた民間習慣 |

> ボス名と王名の一致は「同一存在」ではない。命名の畏れ・混線として扱う。

## 3. 征討の社会的意味

征討は「強敵を倒す祭り」ではなく、ギルド的には **危険均衡の再計測**。  
討伐後に生態が悪化する事例が報告書に残る（公開してよい失敗談）。

## 4. 巨きき祖との距離

伝説個体のさらに向こうに `10_higher_beings` の巨きき祖がいる。  
プレイヤーが「倒した」と感じる天井と、世界が持つ天井を分けておくと世界が狭くならない。
