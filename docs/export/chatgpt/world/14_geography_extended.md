# 14 — 地理拡張（未踏査・異名・微地形）

エルド大陸の既知地図に、**通称・微地名・未踏査**を足す。実装ダンジョン必須ではない。

---

## 1. 中央

| 地名 | 種別 | メモ |
|---|---|---|
| **冠裂け谷** | 断層谷 | クラウンリバー上流の裂け目。王都崩落の余波と噂 |
| **灰の並木** | 旧街道脇 | 焼かれた並木が白く残る。夜に灯るという迷信 |
| **沈黙の楕円** | 広場跡 | アステリア旧王立広場 |
| **鉄の避港** | 河港跡 | アイアンヘイブン南。物資揚陸の旧拠点 |
| **双鏡湖（そうきょうこ）** | 湖 | ミラーレイクの古名。表と裏で景色が違うという誇張 |

## 2. 北

| 地名 | メモ |
|---|---|
| **白息峠** | フロストウォール手前。息が白いからではなく、風が白い粉を運ぶ |
| **犬曳き台地** | ケレン周辺。足跡化石が多い |
| **聴氷穴** | 氷の下で潮騒がする穴。巨きき祖「冠を負う脊」の噂と接続 |
| **無里程の原** | 里程標が意図的に無い地帯。アステルゆかり |

## 3. 西・森

| **囁き台** | ウィスパーウッド名の語源候補地。風が人声に聞こえる |

## 6. 未踏査ラベル（地図の外縁）

ギルド地図の端にスタンプされる公式ラベル。

| ラベル | 意味 |
|---|---|
| **INKED?** | 記録はあるが現地未確認 |
| **SEASONAL** | 季節で地形が消える／現れる |
| **DO NOT CAMP** | 野営死亡率が高い（理由多様） |
| **ECHO** | 音響異常。呼び声が返る／返らない |
| **PRE-NINE?** | 先王層の疑い |
