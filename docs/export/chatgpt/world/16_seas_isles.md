# 16 — 海域・群島・航路

---

## 1. 主要な海

| 名 | 位置 | 性格 |
|---|---|---|
| **西の外洋・グレイヴシー** | シャッタードアイルズの西 | マレクが「果て」を見たとする海。波が高く、帰る船が減る |
| **クラウン河口湾** | 西海岸 | 大河の濁りと潮が混ざる。港の命 |
| **南海ブラックシー** | 南 | 暗い水。星がよく映るが魚は少ない、という矛盾した噂 |
| **東端イースタン・オーシャン** | サンダーピーク沖 | 火山灰と雷。航路としてはほぼ死んでいる |

## 2. シャッタードアイルズ（群島）深掘り

| 島／通称 | メモ |
|---|---|
| **三灯本島** | 旧行政。灯台が三基建つが、どれが本物か議論がある |
| **骨環島** | 干潮で骨楼のような岩礁。巨きき祖伝承 |
| **無口島** | 鳥さえ鳴かない。ECHO ラベル付き |
| **塩歯の列岩** | 船底を削る。保険が効かない |
| **マレクの石錨** | 最西。錨の形をした天然岩＋人工の鑿痕 |

## 3. 航路（伝承・現行）

| 航路名 | 結ぶもの | 状態 |
|---|---|---|
| **潮環路** | シーゲート↔タレッサ跡 | 干潮依存。観光と調査 |
| **灰の航路** | シーゲート↔アイアンヘイブン河港 | 現行主力（河＋沿岸） |
| **嵐名付け航路** | 群島内周回 | 嵐の人名リストが海図に併記される |
| **閉じた東路** | かつてレッドフォージへ鉄を運んだ | 灯断ち以降ほぼ廃航 |

## 4. 海の禁忌・船乗りの掟

1. 嵐に嘘の名を付けるな（本当の嵐が嫉妬する、という冗談めかした掟）
2. 白王の話を船底でするな（浸水の迷信）
3. 王冠を見た、と港で言うな（灰狼団が来る——実害）
4. 沈んだ鐘の音を「吉兆」と言い切るな（吉でも凶でも人が集まる）

## 5. 海の生物ロア（戦闘実装は任意）

| 通称 | 祖先想像 | 備考 |
|---|---|---|
| **塩噛みエイ** | エイ＋鉱物歯 | 船底を削る実害 |
| **霧舵クラゲ** | 群棲クラゲ | 舵が効かない感覚を起こす毒粘液 |
| **冠食いカモメ** | カモメ大型 | 光るものを盗む。モーンゲートの冠喰いネズミと民俗接続 |
| **潮の骨楼の影** | 巨きき祖 | 遠影のみ |
