# 13 — 失われた諸王国・属領・滅び方

正典の主要諸王国に、属領・地方名・滅びの様式を追加する。

---

## 1. 中央王統アステリア圏

| 名 | 役割 | 滅び方の伝承 |
|---|---|---|
| **王都アステリア** | 盟主・継承の座 | 長夜の崩落。勝者なし |
| **外郭環区アウレア** | 貴族・市場 | 灰狼団の巣窟化→ギルドが一部封鎖 |
| **下層工区カルド** | 排水・鍛冶下請け | 地下へ沈みモーンゲート上層と同化 |
| **学府支都セレニカ** | セラディス学派の分校 | 禁書を運び出せず自焼した、という噂 |
| **盟約前庭フェデル** | 諸王会合地 | 円石だけ残し人が去った |

**追加固有名詞:** 王宮の別名「**冠の匣（はこ）**」、王立広場「**沈黙の楕円**」

---

## 2. 北 — 辺境開拓諸侯

| 名 | 内容 |
|---|---|
| **カルンヴェール** | 開拓庁跡。氷の文書庫 |
| **霜砦ホルム** | 狼煙の中継。いまは空家の石核 |
| **犬曳きの里ケレン** | セピア系犬種の原産伝承地（モーンゲートの犬系敵と民俗接続可） |
| **フロストウォール** | 現存。諸侯の名残を「評議会」に変えた山岳都市 |

滅びの様式: 戦争というより **補給線の死**。灯断ちの年に北が先に暗くなった、と歌う。

---

## 3. 西 — 森の盟約国

| 名 | 内容 |
|---|---|
| **翠冠** | 樹上二層都。根が街を飲み、グリーンホロウへ縮小 |
| **返す森の十二杭** | 国境杭。一本は盗まれ闇市場に出た（真偽不明） |
| **胞子の議会場** | 湿原側の会議地。いまはミストフェン寄り道のロアフックに使える |

滅びの様式: 戦火より **森の再占領**。人が去ったのではなく、森が人を退出させた、という語り。

---

## 4. 西海 — 海洋連合

| 名 | 内容 |
|---|---|
| **タレッサ** | 潮環港。干潮遺跡 |
| **シーゲート** | 現存港町。連合の実務的後継 |
| **三灯諸島** | シャッタードアイルズ内の旧行政中心 |
| **沈艦墓域オルト** | ブラックショアの難破船墓地の公的名（伝承） |

滅びの様式: 航路の崩壊と **潮位の異常**（諸説）。海統の時刻表が狂った、と船乗りは言う。

---

## 5. 東 — 鍛冶諸侯／守護砦国

| 名 | 内容 |
|---|---|
| **オルド（星炉市）** | 炉の都。レッドフォージの核 |
| **釘市ヴァラン** | 規格釘の卸。いまは錆の丘 |
| **ヴァレンドア** | 盾環都市。三重城壁 |
| **峠関ストームゲート** | 東西交易の関所跡 |

滅びの様式: 炉の事故・砦の包囲・街道寸断の三重苦。鍛冶と守護は最期に互いを責め合った手紙が残る（偽作多し）。

---

## 6. 南 — 沼と湿地の周縁国（新規追加）

正典で薄い南を厚くする。

| 名 | 内容 |
|---|---|
| **沈封侯国ミレシア** | ミストフェン一帯を治めた小侯。封印と治水の国 |
| **泥の首都ヴァールム** | 杭上都市。いまは折れた杭の林 |
| **黒霧の税関** | 沼を渡る通行税。いまはエリート出現フックに転用可 |
| **崩落街道橋領** | broken_marsh の政治的前身 |

滅びの様式: **沈下**。戦争の記録より、地盤と水位の記録が多い。

---

## 7. 「滅び方」の類型（物語用）

| 類型 | 例 | 雰囲気 |
|---|---|---|
| 長夜型 | アステリア | 一夜で中枢死 |
| 灯断ち型 | 街道網 | ゆっくり暗くなる |
| 森蝕型 | 翠冠 | 自然が再占領 |
| 潮狂型 | タレッサ | 時刻表が壊れる |
| 沈下型 | ヴァールム | 地面が裏切る |
| 炉爆型 | オルド | 技術が自壊 |

プレイヤー向け開示は断片のみ。類型名は内部・ライター用。
