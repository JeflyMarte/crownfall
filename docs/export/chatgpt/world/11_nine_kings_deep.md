# 11 — 九王深掘り（治世・逸話・遺構）

既存の九王名・称号は正典どおり。本章はその **厚み** を追加する。

---

## 開拓王エルディオン

- **治世の像:** 地図の外縁に杭を打ち、杭の向こうを「次の既知」と呼んだ
- **都の伝承:** **辺境庁カルンヴェール**（現ノースリーチの氷下遺跡）
- **逸話:** 最初の遠征で隊の半分が帰らず、エルディオンは「失った名」を地図の余白に墨で記した。墨が滲んだ場所が、今も地図の染みとして残るという（実物未確認）
- **遺構:** 氷の観測塔「**霜針**」、狼煙台の列「**北の点線**」
- **人々の記憶:** 勇敢というより「帰らない男」。フロストウォールでは今も「エルディオン歩き」（無理な遠出）を戒める

## 守護王ヴァルケイン

- **治世の像:** 国境を線ではなく **厚み** で考えた。砦・空地・監視の三段
- **都:** **盾環都市ヴァレンドア**（ストームクラウン山麓の多重城壁跡）
- **逸話:** 敵将と盟約し、盟約の夜に自軍の一部が裏切った。ヴァルケインは裏切者を処刑せず、境界の外側へ追放した——「内側で腐らせぬ」
- **遺構:** **三重の空堀**、銘板「盾は退かず」
- **現代の残響:** 灯守の「敵味方を問わず迎える」規範と対立する厳格派が、ヴァルケインを引用する

## 学識王セラディス

- **治世の像:** 知識を集め、分類し、**閉じた**
- **都:** **書環（しょかん）**— 円形に広がる書庫都市。中心ほど古い
- **逸話:** 「開く鍵は閉ざすためにも要る」。禁書庫の目録は三部あり、二部は空白、一部は焼け焦げている
- **遺構:** 王立図書院奥の **無扉の回廊**、インク鉱物の染み
- **現代:** ギルド記録部の分類癖は、セラディス学派の遠い子孫だと冗談めかして言われる

## 信義王アウレクス

- **治世の像:** 約束を法より上に置いた（あるいは法そのものを約束と呼んだ）
- **都:** **盟台都市フェデル**（アイゼンプレイン中央の円形広場都市跡）
- **逸話:** 最後の盟議で、アウレクスは剣ではなく指環を卓に置いた。決裂後、指環だけが行方不明になった——盟約の指環伝承と接続
- **遺構:** **誓いの円石**、名前を削られた石柱列
- **ことわざ:** 「フェデルで嘘をつくと、風が名前を持っていく」

## 巡礼王ルミナス

- **治世の像:** 道を信仰に近い公共財にした
- **「都」:** 定住都を持たず、**移動宮廷**（季節ごとに宿駅を宮とした）
- **逸話:** 一夜で街道に百の灯を立てた、という誇張譚。実測では宿駅間隔の標準化が功績
- **遺構:** 王の大街道、里程標「**光里（ひかり）**」
- **現代:** 車輪同盟の憲章冒頭にルミナスの名がある

## 鍛冶王オルグラン

- **治世の像:** 火と鉄と契約——「刃は嘘をつけない」
- **都:** **星炉市オルド**（レッドフォージの原型）
- **逸話:** 星屑を炉に入れた夜、空が三日暗くなった（火山灰との混同説あり）
- **遺構:** **星炉の底石**、炉印の鋳型倉庫
- **現代鍛冶:** 赤鉄の工房の儀式的な「炉への一礼」はオルグラン流の残骸

## 海統王ネレイオス

- **治世の像:** 潮の時刻表で国を動かした
- **都:** **潮環港タレッサ**（シーゲート沖、干潮にだけ姿を出す石環）
- **逸話:** 嵐を「敵」ではなく「時刻の狂い」と呼んだ。航路図に嵐の名前を付けた最初の王
- **遺構:** 潮境の海図の写し（断片）、沈んだ灯台「**深灯**」
- **現代:** シーゲートの船乗りは嵐に人名を付ける習慣を残す

## 森護王シルヴァリア

- **治世の像:** 伐採許可を「何年後に返すか」で決めた
- **都:** **翠冠（すいかん）**— 樹上と地上の二層都市。現グリーンホロウの祖
- **逸話:** 世界樹に語りかけた、という詩がある。植物学者は「根系の震動を聴診した」と再解釈する
- **遺構:** **返す森の境界杭**、種子庫の石箱
- **現代:** ヴェルディアの伐採タブーと接続

## 継承王カエルム

- **治世の像:** 最後の中央王統。治世は短く、言葉は長い
- **都:** 王都アステリア
- **逸話:** 「次の者へ」を法令の冒頭に置いた。崩落の夜、名を残さなかった継承者（九英雄）との関係が諸説
- **遺構:** 崩落した王宮、地下への封印扉（モーンゲート接続）
- **現代:** 継承祭の精神的祖

---

## 九王の相互関係（公開用・諸説）

| 関係 | 伝承 |
|---|---|
| エルディオン↔シルヴァリア | 辺境と森の境界線を十年かけて引いた |
| オルグラン↔ネレイオス | 船釘と錨の規格戦争→やがて共通規格 |
| セラディス↔カエルム | 禁書の一部を継承王が「閉じたまま運ぶ」ことを許した |
| アウレクス↔ヴァルケイン | 盟約と境界——「約束は壁より強い／壁は約束より速い」論争 |
| ルミナス | ほぼ全員と街道で結びついた「接着剤」 |

九王が同時に生きたのか、称号の継承系譜なのかは **未確定のまま残す**。
