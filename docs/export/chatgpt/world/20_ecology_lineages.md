# 20 — 生態系譜と変異の型

原則維持: **環境 → 生態系 → モンスター**。悪魔・アンデッド・スライムは出さない。

---

## 1. 変異の型（ライター用ラベル）

| 型 | 説明 | 代表 Biome |
|---|---|---|
| **鉱物化** | 金属・結晶の体内析出 | モーンゲート |
| **菌糸共生** | 胞子・毒・擬態 | ウィスパーウッド／ミストフェン |
| **潮塩適応** | 塩耐性・発光粘液・船害 | ブラックショア |
| **凍圧適応** | 脂肪・角・聴覚特化 | フロストリッジ |
| **炉縁耐熱** | 耐熱皮膚・金属食 | レッドフォージ／サンダーピーク（Future） |
| **暗順応** | 視覚退化・振動覚 | 地下深部・封緘区 |
| **人跡依存** | 遺構・ごみ・灯油を餌にする | 王都外郭・街道沿い |

## 2. 系譜樹（例・モーンゲート）

```text
犬系祖先
 ├─ セピアハウンド（嗅覚・灰褐色）
 └─ （伝承）冠喰いネズミと競合 → 地下でのニッチ分化

齧歯祖先
 └─ 冠喰いネズミ（光物嗜好）

節足祖先
 ├─ ルーンローチ（甲殻に文字状析出）
 └─ クロックモス（調時室の歯車環境に適応した Elite）

爬虫／獣混合的適応
 └─ 水晶サソリ・骸面系（結晶・擬態）
```

ボス **セルディオン** は「地下生態の頂点」——水晶と骸のイメージは鉱物化の極端形として語る。

## 3. 食物網の書き方（イベント／図鑑用）

常に三点を書く。

1. **何を食べるか**
2. **何に食べられるか**
3. **人が介入すると何が崩れるか**

例: クロックモスを狩りすぎると調時室の害虫が爆発し、ルーンローチが記録庫を汚す——「討伐＝正義」ではない。

## 4. エルダと生態

エルダは魔法の呪いではなく **環境勾配**。  
弱点属性は「その土地の相に対する生理的脆さ」として説明できると望ましい。
