# 08 — 世界観要約（Brief）

> 詳細正典は `docs/specs/world/`。ここでは ChatGPT 向けの短い共有用。

## 三本柱

**History / Relics / Ecology（戦後生態系）**

## 時代の空気

数百年前の **九王戦争** で旧文明は崩壊。  
いまは安全圏の都市と危険な野外圏。人々は遺跡と記録から歴史の断片を拾う。

プレイヤーは世界を救う選ばれし英雄ではなく、探索者ギルドの **調査隊長**（無数の継承者の一人）。

ギルド理念（要約）: **「遺産を持ち帰るな、歴史を持ち帰れ」**。

## モンスター観

魔物・不死者ではなく、**実在生物を祖先に持つ**戦後生態の生き物。  
分類 Class I〜VII。Biome ごとに生態が違う。

## 属性の世界観

5属性は自然力「**エルダ**」の相。超自然の魔法ではない  
（闇＝瘴気、聖＝浄化の光、錬成＝エルダを物質的に扱う技術、など）。

## 中核の謎（開示は断片的）

- 九王戦争の真実
- 消された第十の王
- 王冠の失墜 — 「倒れたが滅びていない」（タイトル語源）
- 伝説武器の由来
- 世界樹は一本ではない

開示手段: 武器説明・古文書・遺物・NPC・図鑑発見度。長尺ムービー必須にしない。

## 導入で出す／出さない

- 出す: 戦後の空気、ギルド、モーンゲートへの出立
- 出さない: 中核の謎の真相、操作講習
