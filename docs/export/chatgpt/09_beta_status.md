1. **βで遊べないコンテンツを「今の本編」として書かない**（②以降は将来）
2. 鑑定屋必須ループ、3人固定、旧ジョブ名（戦士／騎士…）を現行に戻さない

4. 数値を勝手に大きく変える提案は、体験への影響が大きいので **案を並べて確認待ち**にする
5. 矛盾したらこの `docs/export/chatgpt` を優先し、古い game 仕様の歴史節は無視
