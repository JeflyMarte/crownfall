# 05 — キャラクター・ジョブ・ガチャ・オトモ

## 基本ジョブ（5種・上位／下位職なし）

| ジョブ | ざっくり役割 |
|---|---|
| ソードマン | 近接アタッカー |
| レンジャー | 遠距離 DPS |
| ヴァンガード | タンク／ヘイト |
| アルケミスト | 呪術・回復・デバフ |
| ビーストテイマー | 獣・群れ・オトモ連携 |

ジョブはスキルリストだけでなく **AI 行動思想**を決める（Combat Vision）。

## 編成

- 人間: 最大 **4人**（陣形あり。空き枠の埋め方に注意＝複製バグ修正済）
- **オトモ「ジャック」**: NG開始から随伴。4人枠外・★1・装備不可・パッシブなし・常時前衛・共有EXP。ドットは後差し可

## スターター加入

- 開始時: **1人選択**
- 章クリアで初期5が順次加入（β Extra OFF＝本番寄り。①内は開始1＋1-5付近で最大2人など）
- メイン5の★帯は ★3 寄せ

## 成長

| 項目 | 現行 |
|---|---|
| EXP | ラン成功時、受取対象へ（人間＋オトモ） |
| 上限 | Lv99（51〜は逓減。スキル習得は〜50） |
| 成長 | HP/ATK（BalanceConfig・STAT_SCALE 反映） |
| 進化 | Lv30到達後、英雄管理で手動昇格。専門深化＋常時パッシブ2 |

## スキル習得の目安（各職10）

Lv 1 / 6 / 12 / 18 / 24 / 30 / 36 / 42 / 48 / 50。  
装備できるジョブスキルは **1本**（必殺・レジェンド武器スキルは別枠）。

例（ソードマン）: 一閃 → … → 極意斬  
（他職は本体 `06_キャラクター_ジョブ.md`）

## 固有パッシブ

- メイン5＋ガチャ助っ人の固有パッシブ見直し済
- 味方死亡時トリガー廃止方向
- ★帯の職パッシブ自動付与は撤去（案α）

## 招待状（ガチャ）

| 項目 | 現行 |
|---|---|
| UI名 | **招待状**（召喚という語は避ける） |
| プレイ | 常時 ON |
| ボタン | **単発魔晶石＋チケット**（10連・複数タブ削除） |
| プール | 6体（★2×3 / ★3×2 / ★4×1） |
| 排出率目安 | ★2 50% / ★3 35% / ★4 15% |
| 重複 | **限界突破**: パッシブ効果×(1+0.1N)、上限+5、還元半減 |
| 演出 | 封書リビール（閉→開封→顕現）。Featured Idle 回転 |

スターター5職はガチャ除外。助っ人はロスター参加・装備可・共有EXP。

## 指揮官

- 調査許可ランク D〜S、SP 進捗
- 隊長名は台帳から変更可
- TopBar に指揮官表示
