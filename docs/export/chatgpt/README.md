## 何のためのフォルダか

ChatGPT に「現行のゲーム仕様」を渡すための **要約・現行正パッケージ**。  
リポジトリ内の古い節・歴史記述・実装運用文書をそのまま投げると矛盾するので、**いまプレイ可能な体験の正**に寄せて分割してある。

---

## ChatGPT への渡し方（推奨）

### 最短（設計相談・バランス・文言）

1. 本 `README.md`
2. `00_INDEX.md`
3. `01`〜`07` を順に（または話題に関係する章だけ）

### 世界観・ロア相談を含む場合

上記に加え `08_world_brief.md`  
**大幅深掘り**はサブフォルダ `world/`（`world/README.md` から。上位存在・失われた国・追加怪物など）

### 開発優先度・凍結事項の確認

`09_beta_status.md`

### 注意

- **一度に全部を貼るとコンテキストを圧迫**する。相談テーマに合わせて 2〜4 ファイルから始める
- 矛盾が出たら「`docs/export/chatgpt` の記述を正とし、古い game 仕様の歴史節は無視」と指示する
- このフォルダは **要約**。数値の最終確認が必要なら本体 `docs/specs/game/` / `docs/specs/decisions/` を参照
