# 03 — ダンジョン（Biome・章・イベント）

## 設計単位

ダンジョンは単なる「攻略マップ」ではなく **生態系単位（Biome）**。

| 種別 | 役割 |
|---|---|
| メイン | ストーリー脊柱・次帯解放の鍵。Biome あたり **5章**（x-1〜x-5） |
| 寄り道 | 素材・整備。必須ゲートにしない（βでは非表示） |
| 征討 | Boss 超えの高難度（βでは非表示） |

## 章ルール（現行実装）

| 項目 | 方針 |
|---|---|
| 1章 = 1ラン | EXIT で拠点戻り |
| x-1〜x-4 | Boss **なし**。floor／敵Lvは章ごと |
| x-5 | 多フロア＋**最終Fで Boss** |
| 解放（正式） | x-5 Boss 初回討伐で次メイン第1章 |
| β | プレイは **①モーンゲートのみ**。②以降 UI 封鎖 |

## 部屋種別（概念）

START / COMBAT / EVENT / TREASURE / ELITE / MID_BOSS（エリート扱い）/ BOSS / EXIT など。  
実際の並びは章の `floor_count` と重みで生成。

## 危険度ティア

同一ダンジョンに **ノーマル／ハード／ナイトメア**。  
解放は前ティアクリア。敵Lv・レア重み・報酬が上昇。  
**β体験外**（キャンペーン周回帯）。

## 放浪希少種（ラン中差し込み）

- 遠旅スズメ — 高EXP相当・短時間で逃走・武器ほぼ無し
- 聖遺甲虫 — ELITE級・武器ドロップ高め
- 出現率は周回帯（N／H／NM）で上昇
- ダック／レイヴンは放浪プールにも補完（神話は別枠など）

## 罠

ダメージは **最大HP割合**（例: 単体10%/15%、全体は低め％）。探索・罠部屋で適用。

## バランスメモ（直近）

- ステ全体は過去に ×8 スケール済み（装備・敵・成長・回復／罠／DoT 等追随）
