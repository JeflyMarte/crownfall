## 優先度の高い「現行上書き」

本体 `docs/specs/game/` より新しい確定事項:

| トピック | 正 |
|---|---|
| 公開β範囲 | モーンゲート（①）のみ。②以降は UI 封鎖・データ残置 |
| 新規導入 | 世界観スクロール→隊長名→ニーナ→初期1人選択→拠点（操作チュートなし） |
| ガチャUI名 | 「召喚」ではなく **招待状**。単発＋チケットのみ |
| 野外イベント | 週次ローテ廃止 → **30分スロット「今日のダンジョン状態」** |
| 戦闘スキル枠 | 装備スキル **1本** ＋ 必殺ゲージ（与ダメ／被ダメチャージ） |
| パーティ | 人間最大4枠＋随伴オトモ「ジャック」（枠外・常時前衛） |
| サブステージ | `1-1`〜`x-5` 有効。Boss は **x-5 のみ** |
| 識別屋導線 | オミット。ドロップ／クラフト生成時は鑑定済み |
