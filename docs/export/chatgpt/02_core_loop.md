# 02 — コアループと拠点

## 基本ループ（現行）

```text
Title（はじめから / つづきから）
  ↓ 新規のみ: 導入フロー
拠点 Base（備え）
  ↓
ダンジョン選択 → 自動探索 Dungeon（部屋ステップ）
  ↓
結果 Result（報酬・EXP・MVP 等）
  ↓
拠点へ戻り → 装備／鍛冶／編成 → 再出撃
```

識別専用シーンへの必須遷移は **オミット**（ドロップ等は鑑定済み想定）。

## 新規ゲーム導入（確定）

```text
はじめから
→ 世界観ナレーション（縦スクロール・自動クロール・スキップ可）
→ 隊長名入力
→ ニーナ吹き出し（最大3・文字送り）
→ 初期隊員 1人選択
→ 拠点解放
```

- 強制ダンジョン突入なし
- 操作チュートなし（属性／陣形／鍛冶の講習は図鑑・現地へ委譲）

導入ナレーション要旨: 九王戦争後の世界 → 探索者ギルド → 最初の地はモーンゲート。

## 拠点でできること（主な導線）

| 機能 | 内容 |
|---|---|
| 英雄管理／キャラ | 装備・戦術プリセット・スキル／必殺・陣形 |
| 赤鉄の工房（鍛冶） | クラフト・炉研ぎ・分解・錬成（同種合成で装備Lv） |
| 招待状（旧・召喚） | 助っ人ガチャ（単発＋チケット） |
| 図鑑 | 生態／記録／手引き |
| ダンジョン選択 | Featured＋Biomeカード。βは①のみ選択可 |
| 日課 | 固定3件／日（5:00 JST リセット） |
| 隊長台帳 | 指揮官名・調査許可・統計 |
| 野外速報バナー | 「今日のダンジョン状態」（30分スロット） |
| ニーナ案内 | 右上・おすすめ1件ローテ |

## 1ランの体感時間

- 目安 **5〜10分／章**（サブステージ）
- 章クリアごとに拠点戻り
- 判断頻度の設計目標: おおよそ30秒に何か起きる（部屋遷移・イベント・宝箱・エリート等）

## 探索中の操作（現行）

| 操作 | 状態 |
|---|---|
| 探索速度 | あり |
| 帰還／探索終了 | あり |
| 部屋進行 | 部屋単位ステップ |
| ラン中の方針切替 | **Backlog**（拠点準備が中心） |

## 進行（メイン）

```text
① 1-1 クリア → 拠点
→ 1-2 … 1-4
→ 1-5（多フロア＋最終 Boss）
→（正式では）次メイン 2-1 解放
```

**βでは** ①クリア後も ②以降は解禁しない（アップデート枠）。
