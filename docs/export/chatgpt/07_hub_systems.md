# 07 — 拠点システム・UI・その他

## 今日のダンジョン状態（野外速報）

旧「今週の野外」週次ローテを置換。

| 項目 | 現行 |
|---|---|
| 切替 | **30分固定スロット**（端末間で一致する決定的ハッシュ） |
| UI名 | 「今日のダンジョン状態」 |
| 最頻 | **穏やか（補正なし）** |

プール例: 天候固定（雨／夜／霧）、ダック／レイヴン放浪↑、敵Lv+2、群れ↑、ELITE部屋↑、弱めの EXP/Gold/武器/図鑑 等。

## ニーナ案内（拠点ホーム）

- 右上: 顔＋吹き出し
- 10秒自動切替／タップで次へ
- ローテ: **おすすめ1件** → 天気／野外状態 → 雑談
- おすすめ優先: 日課受取 → 日課未完 → 編成空き → 次の探索 → フォールバック
- タップでの画面遷移はしない（文言のみ）

## 日課

- 固定3件／日
- 5:00 JST リセット
- 報酬: Gold／魔晶石／素材

## 下ナビ・画面群（概念）

拠点系は BottomNav で統一（冒険／招待状／図鑑／ホーム等）。  
セーフエリアは **実機のみ**適用（Mac の模擬は既定 OFF）。

## 結果画面

報酬 → レベルアップバー → MVP などのウィザード。  
クリア時は **次のダンジョンへ** 導線あり（次章がある場合）。

## オーディオ

- BGM: title / hub / explore / battle / boss / result
- SE: UI・戦闘・宝箱・必殺・結果・招待状など配線済（要実機）

## UI／タイポ方針（要約）

- 本文: Noto（太め）

- 戦闘数字: Dela Gothic 系
- 可読性優先（ボタン全面画像化は撤回・凍結寄り）
