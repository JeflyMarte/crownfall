# Crownfall

2D見下ろし型・自動探索ハクスラRPG（スマホ向け）

## 概要

プレイヤーは冒険者を直接操作しない。探索隊の指揮官として、方針・装備・編成を決める。滅びた旧世界の遺跡を周回し、伝説の武器を発掘・鑑定・厳選するゲーム。

## 技術スタック

- Godot 4.6.3 Standard
- GDScript
- iOS優先 / Android対応
- Landscape固定 / 1280×720
