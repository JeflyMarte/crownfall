# 歴史

人類文明の繁栄は、**九王戦争** という大戦によって終わりを迎えた。その真相の多くは失われ、いまは各地の碑文・古文書・景観の断片からわずかに輪郭をたどれるのみである。

---

## 崩落（the Fall）

王国時代の終わりを告げた出来事が **崩落（the Fall）** である。統一王権の首都 **王都アステリア** が失われ、王国時代は幕を閉じた。これは歴史の基準点とされ、長期年代は「崩落後（A.F.）」で数えられる。記録の散逸により正確な年数は不確定で、**現在は崩落後およそ300年（諸説あり）** と伝わる。

現行の実用暦は、探索者ギルド創設を起点とする **ギルド暦（G.）**。1年=12ヶ月=360日、1ヶ月=3旬（1旬=10日）。**現在 G.118**。

!!! info "澱みの季（よどみのとき）"
    晩秋から初冬にかけて、地下や遺構で闇のエルダ（瘴気）の活性が高まる危険期。生物が活発化し、探索の難度が上がる。

---

## 伝承年表（復元 / 諸説）

ギルド記録部が各地の記録を突き合わせて復元した、崩落前後のおおまかな流れ。**年代はすべて復元値で、地域の記録は互いに食い違う**。

| 時期 | 出来事 |
|---|---|
| 崩落前 約20年 | **最後の盟議** — 諸王が盟約の地アイゼンプレインに集った最後の会議。決裂が戦争拡大の転機とされる |
| 崩落前 約15〜5年 | 諸国の対立が拡大し、王の大街道の要所が次々に寸断。各地が孤立へ |
| 崩落前 数年 | **灯断ちの年** — 街道・神殿の灯が各地で絶え、交流網が事実上崩壊 |
| **崩落（A.F. 0年）** | **アステリアの長夜** — 王都アステリアが失われた夜。王のいない世界が始まる |
| 崩落後 〜約200年 | 静寂の時代。人口激減・知識喪失・遺構の自然化 |
| 崩落後 約180年 | 探索者ギルドの前身が発足 |
| 崩落後 約300年（現 G.118） | 探索者の時代。モーンゲート調査が本格化し、物語が始まる |

## 九王

王国時代を支えたとされる九人の王。それぞれが領域と理念を担い、後世へ **王遺産** を託したと伝わる。

| 称号 | 名 |

| 開拓王 | エルディオン |
| 守護王 | ヴァルケイン |
| 学識王 | セラディス |
| 信義王 | アウレクス |
| 巡礼王 | ルミナス |
| 鍛冶王 | オルグラン |
| 海統王 | ネレイオス |
| 森護王 | シルヴァリア |

## 王冠（The Crown）

唯一の王権の証として伝わるもの。作品名 *Crownfall* の語源であり、「**倒れたが、滅びてはいない**」と語り継がれる。九王より前、「継承」を司ったとされる伝説的存在 **白王（the White King）** の名とともに、最古の信仰の跡が残る（→[信仰](faith.md)）。

---

## 先王時代（Pre-Nine）

九王時代より前の、起源不明の超古代。**世界遺産（World Monument）** と呼ばれる起源不明の構造物や、未解読の **古代文字（Elder Script）** を遺したとされる。生物が遺物・鉱物を取り込んで形成する「ルーン」状の紋様は、この古代文字を取り込んだ模様にすぎず、魔法ではない。
