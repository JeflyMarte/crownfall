# 社会と派閥

王都アステリアの陥落で中央政権は失われた。いまの人類社会は **単一の国家ではなく、生き延びた都市と街道網が緩くつながる連合体** である。

## 社会の枠組み

- **都市自治** — 現存都市（シーゲート／フロストウォール／グリーンホロウ等）はそれぞれ **都市評議会** が治め、上位の王権はない。
- **街道が動脈** — 巡礼王ルミナスの遺した[王の大街道](geography.md)が都市を結ぶ唯一の道。街道の安全がそのまま社会の安全となる。
- **拠点アイアンヘイブン** — 戦後新設の探索者都市。王権の空白を埋める事実上の中心だが、王を名乗る者はいない。

## ギルド外の主な勢力

| 勢力 | 性格 | 縄張り |
|---|---|---|
| **車輪同盟** | 街道交易を担う隊商の互助組合 | 王の大街道全域 |
| **灯守組合** | 街道の灯・宿駅を守る職能集団。中立的な調停者 | 街道の宿駅・辻 |
| **灰狼団** | 廃墟を荒らす無法者・盗掘団 | 廃墟・未管理区域 |
| **都市評議会** | 各都市の自治機構 | 各現存都市 |

!!! info "力学"
    街道の安全を守る **車輪同盟** と、遺物を盗掘する **灰狼団** は恒常的に対立。**灯守組合** は「灯を分かつ者を害さない」作法ゆえ、しばしば双方の仲介に立つ。

## 暮らしと文化

- **再起の精神** — 「倒れたが滅びていない」という王冠の銘に象徴される価値観。廃墟から拾い、次へ手渡すことを尊ぶ。
- **継承祭** — 年の節目に、前年の灯を新しい灯へ「継ぐ」最大の祭り。[灯の信仰](faith.md)と継承王カエルムの「次の者へ」の理念が融合した行事。
- **交易と食** — 車輪同盟が各地の特産を運ぶ。北の保存食、西海岸の海産、森の幸がアイアンヘイブンの中央市場に集まる。
- **記録と語り** — 文字記録の多くは陥落で失われ、口承・写本・遺物の銘が知識を伝える。ギルドの[調査記録（図鑑）](../index.md)はその再構築の試み。

## 民間伝承

文字を失った戦後世界では、知識と教訓の多くが歌・言い回し・言い伝えの形で受け継がれている。

- **九王数え唄** — 子どもが九王の称号を覚える数え唄。「ひらく・まもる・しるす・ちかう／めぐる・きたう・うみおさめ／もりまもり・つぎのもの」。
- **灯継ぎ唄** — 継承祭の夜にうたわれ、「消える前に、次へ渡せ」と繰り返す。

| ことわざ | 意味 |
|---|---|
| 倒れたが滅びていない | 挫折しても終わりではない（王冠の銘） |
| 遺産を持ち帰るな、歴史を持ち帰れ | 探索者ギルドの理念 |
| 灯を分かつ者を害すな | 旅人同士の最低限の信義 |
| 澱みの季に深くは潜るな | 瘴気が高まる危険期への戒め |

!!! tip "言い伝え"
    「古龍を見たら名を呼ぶな」「継承祭の夜に灯を絶やした家は幸運も絶える」「灰狼団が現れる前は街道の鴉が鳴きやむ」——各地に伝わる迷信は、危険な世界を生き抜く経験則でもある。

!!! question "残された問い"
    継承祭で「継ぐ灯」は、本来何の灯だったのか。車輪同盟が大街道の「ある区間」を理由なく避けるのはなぜか——語られないままの謎が、社会の片隅に残っている。
