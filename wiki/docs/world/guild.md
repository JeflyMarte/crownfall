# 探索者ギルド

**探索者ギルド（Seekers Guild）** は、探索者を支援し、調査記録を蓄積する調査機関である。プレイヤーは、このギルドに属する探索隊の指揮官として任務にあたる。

最高意思決定機関は **ギルド評議会（Guild Council）**。加盟都市の代表と分野長による合議制で、命令ではなく調整と承認を担う。

---

## 三大任務と五部門

ギルドの活動は三つの大任務に集約され、五つの部門がこれを支える。

| 任務 | 内容 |
|---|---|
| **遺物回収** | 旧文明の遺物を調査・回収する |
| **生態調査** | 戦後生態系の生物を観察し、危険度を判定する |
| **地図更新** | 探索路・崩落地点・危険区域を地図化する |

| 部門 | 担当 |
|---|---|
| **回収部** | 遺物の調査・回収・目利き |
| **調査部** | 生物の生態調査・危険度判定 |
| **測量部** | 探索路・崩落地点・危険区域の地図化 |
| **記録部** | 調査記録・図鑑の整理 |
| **補給部** | 装備・物資の調達、市場との仲介 |

---

## 資格と等級

- **上級探索者（Senior Seeker）** — 一つの専門を究めた者への上級資格。ジョブの到達形（例: ソードセイバー）に対応する。
- **調査許可等級** — 隊長（プレイヤー）に与えられるギルド評価。調査成果で上昇し、危険区域・上位任務・補給枠を解放する。

## 拠点アイアンヘイブンの施設

主要施設は、ゲーム内の拠点機能に対応する。

| 施設 | 通称 | 役割 |
|---|---|---|
| ギルド本部 | 記録庁を擁する本館 | 任務の発令・調査報告・図鑑管理（ダンジョン選択 / 図鑑 / 帰還報告） |
| 鍛冶場 | 赤鉄の工房 | 遺物素材の加工・武器の鍛造 |
| 市場・商館 | 中央市場 | 素材の取引、補給部を通じた物資調達 |
| 宿場・酒場 | 辻灯亭（つじびてい） | 各地から集う探索者の合流・勧誘（助っ人・編成） |
| 訓練場・認定所 | 認定所 | 専門資格（上級探索者）の認定 |
