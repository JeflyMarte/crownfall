# 用語集

Crownfall の世界を読み解くための主要用語の索引。固有名詞の正規表記はこのページを基準とする。

---

## 世界・時間

| 用語 | 定義 |
|---|---|
| **戦後生態系（Postwar Ecology）** | 九王戦争後、数百年をかけて形成された新たな自然環境。本作の上位世界観方針。 |
| **九王戦争** | 王国時代を終わらせた大戦。真相は失われている。 |
| **崩落（the Fall）** | 王都アステリアが失われ王国時代が終わった出来事。歴史の基準点。 |
| **崩落後（A.F.）** | 崩落を起点とする長期年代尺度。現在＝崩落後 約300年（諸説）。 |
| **ギルド暦（G.）** | 探索者ギルド創設を起点とする現行暦。1年=12ヶ月=360日。現在 G.118。 |
| **旬（じゅん）** | ギルド暦の区切り。10日＝1旬、3旬＝1ヶ月。 |
| **澱みの季（よどみのとき）** | 晩秋〜初冬、地下・遺構で瘴気の活性が高まる危険期。 |

## 力・自然

| 用語 | 定義 |
|---|---|
| **エルダ（Elda）** | エルド大陸に遍在する自然力。五属性はエルダの相。超自然ではない。 |
| **瘴気（闇のエルダ）** | 腐敗・汚染の気。属性「闇」の正体。状態異常「呪い」もこれによる。 |
| **浄化の光（聖のエルダ）** | 瘴気や異常な個体に作用する力。属性「聖」の正体。神の力ではない。 |
| **錬成** | エルダを物質的に扱う技術。アルケミストの「魔法」の正体。杖はその触媒器。 |

## 王権・遺産

| 用語 | 定義 |
|---|---|
| **王冠（The Crown）** | 唯一の王権の証。作品名 Crownfall の語源。「倒れたが滅びていない」。 |
| **王遺産（Royal Relic）** | 九王が後世へ託した継承物。九対の伝説武器と対をなす。 |
| **第十王遺産** | 古い目録に一度だけ現れる記述。最終行に「これは九王の遺産ではない」。 |
| **先王時代（Pre-Nine）** | 九王時代より前の起源不明の超古代（伝承）。 |
| **白王（the White King）** | 九王以前に「継承」を司ったとされる伝説的存在。 |
| **灯火** | 古い物語に繰り返し現れる、誰も消させなかった火。真相は語られない。 |

## 人と社会

| 用語 | 定義 |
|---|---|
| **探索者（Seeker）** | 遺物回収と生態調査を担う調査員。プレイヤーが率いる立場。 |
| **探索者ギルド（Seekers Guild）** | 探索管理機構（調査機関）。 |
| **安全圏 / 野外圏** | 城壁の内側（生存圏）／外側（探索の舞台）。 |
| **隊商（交易隊）** | 護衛を伴い都市間を結ぶ交易隊。 |
| **ゴールド（Gold）** | 旧王国の金貨。事実上の基軸通貨。 |
| **目利き（めきき）** | 遺物・素材の素性と真贋を見極める営み。 |

## 生態

| 用語 | 定義 |
|---|---|
| **Biome** | 地域ごとの自然環境・生態系の単位。 |
| **鉱物化適応** | 遺物・鉱物を体内へ取り込み、結晶・金属・古代文字状の組織を形成する適応（モーンゲートの特徴）。 |
| **共生適応** | 菌糸・胞子・蔓・苔と共生／寄生して形質を得る適応（囁きの森の特徴）。 |
| **伝説個体（Legendary）** | 各地で語り継がれる極めて稀な超大型・古代の生物。神でも不死でもなく生物。例: セルディオン。 |
| **古龍種（Elder Reptiles）** | 王国時代から姿を変えず生き延びた大型飛行爬虫類の総称。セルディオンの骨格の由来。 |
| **古代文字（Elder Script）** | 最古の遺構に刻まれた未解読の文字。「ルーン」と俗称される生物の紋様はこれを取り込んだ模様で魔法ではない。 |
