# 戦闘の仕組み

戦闘は **自動** で進行する。指揮官（プレイヤー）は戦闘中に直接指示を出さず、**事前の編成・装備・ジョブ** で結果を決める。ここでは、与ダメージと行動順がどう決まるかを解説する。

---

## 行動順（イニシアチブ）

各ユニットの行動順は **イニシアチブ** で決まり、値が高いほど早く動く。

- 味方のイニシアチブ ＝ **武器の攻撃速度 × ジョブ補正 × 装備（Affix）補正**
- 敵のイニシアチブ ＝ その敵の **攻撃速度**

!!! tip "手数か、一撃か"
    攻撃速度の高い **短剣・短刀** は手数が多く先手を取りやすい。**大剣** は一撃が重い代わりに遅い。隊の役割に合わせて選ぼう。

同じイニシアチブの場合は、味方が優先され、編成順で並ぶ。

---

## ダメージの決まり方

与ダメージは、おおむね次の要素から組み立てられる。

| 要素 | 効果 |
|---|---|
| **武器の攻撃力** | ダメージの基礎値。 |
| **ジョブ補正** | 攻撃役は高く、支援役は控えめ。 |
| **レベル / 成長** | レベルアップでステータスが上昇。 |
| **Affix（装備の特性）** | 攻撃・クリ・状態異常付与などを上乗せ。 |
| **クリティカル** | 一定確率でダメージが増加。武器・装飾でクリ率が上がる。 |
| **敵の防御力** | 与ダメージを軽減する。 |
| **弱点（属性）** | 敵の弱点を突くと **×1.25**（→[属性と状態異常](elements.md)）。 |
| **特効（Bane）** | 武器が特定の生態分類に対して持つ追加倍率（例: 獣類 ×1.3）。 |

---

## スキル

各[スキル](../data/skills.md)は **クールダウン** を持ち、戦闘中に条件を満たすと自動で発動する。

- **武器固有スキル** — 武器ごとに設定された必殺技（例: 燻鉄の大剣 → 焚き付け）。
- **ジョブスキル** — ジョブが習得するスキル。攻撃・回復・強化など。

スキルの属性が空欄の場合、**装備武器の属性を継承** する。炎属性武器の攻撃スキルなら、炎弱点の敵に通りやすい。

---

## 自動探索

ダンジョンに入ると、隊は部屋を自動で進む。敵がいる部屋では自動戦闘が始まり、勝利すると次の部屋へ。最深層のボスを倒すと探索クリアとなり、[リザルト](getting-started.md)で報酬を得る。

→ 敵ごとのステータスと弱点は[モンスター図鑑](../data/monsters.md)で確認できる。
