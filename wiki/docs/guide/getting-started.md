# はじめに

Crownfall は、プレイヤーが冒険者を直接操作しない **自動探索型** のハクスラRPG。あなたは探索隊の **指揮官** として、戦う前の準備——編成・装備・方針——で勝敗を決める。

---

## ゲームの流れ

```
編成・装備を整える
      │
      ▼
ダンジョンを選ぶ
      │
      ▼
自動で探索（部屋を進み、敵と自動戦闘）
      │
      ▼
リザルト（経験値・ゴールド・ドロップ）
      │
      ▼
強化して再挑戦
```

1. **編成** — 探索隊のメンバーと[ジョブ](../data/jobs.md)を決める。
2. **装備** — 各メンバーに[武器](../data/weapons.md)・[防具・装飾品](../data/equipment.md)をセットする。属性武器は、狙う敵の弱点に合わせると効果的。
3. **ダンジョン選択** — 推奨レベルを確認して挑戦先を選ぶ。
4. **自動探索** — 隊は自動で部屋を進み、敵と遭遇すると[自動戦闘](combat.md)に入る。指揮官は事前の準備で結果を左右する。
5. **リザルト** — 経験値・ゴールド・装備ドロップを獲得。レベルアップでメンバーが成長する。
6. **強化と再挑戦** — 得た装備で隊を強化し、より深い探索へ。

---

## 指揮官としての要点

- **弱点を突く** — 敵の弱点[属性](elements.md)に合った武器を持たせると、与ダメージが上がる。
- **行動順を意識する** — 攻撃速度の速い武器ほど先に動ける（→[戦闘の仕組み](combat.md)）。
- **役割を分ける** — 前線を支えるジョブ、火力を出すジョブ、支援するジョブを組み合わせる。
- **調査を進める** — 敵を倒し調査を重ねると、[モンスター図鑑](../data/monsters.md)の記録が段階的に開放される。

---
