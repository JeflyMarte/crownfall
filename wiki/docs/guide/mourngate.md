# モーンゲート攻略

**王都地下モーンゲート** は、最初に挑む探索地。かつての首都・王都アステリアの地下に広がる、崩れた地下水路から王座の深淵まで続く遺構である。生物が遺物や鉱物を体内へ取り込む **鉱物化適応** が見られるのが特徴。

## 攻略の指針

!!! tip "属性を二段構えで"
    道中（炎弱点が多い）には **炎属性の武器**、ボス戦には **電気または聖属性の武器** が有効。両方を隊で分担して持たせると、道中もボスも安定する。

- **炎属性枠**：[燻鉄の大剣](../data/weapons.md) などで道中の獣類・昆虫類を素早く処理。獣類特効が付く武器ならさらに有利。
- **電気/聖枠**：ボスのセルディオンと、電気弱点のクロックモス・水晶ハリネズミ対策。
- **エリート対策**：クロックモスはスタンを撒いてくる。前線を支える役を置き、火力役が落とされないよう編成する。
- **レベル**：推奨レベル1から挑戦可能。装備が揃わないうちは、ドロップを集めて再挑戦しながら隊を強化していく。

→ 戦闘の基礎は[戦闘の仕組み](combat.md)、弱点の詳細は[属性と状態異常](elements.md)を参照。
