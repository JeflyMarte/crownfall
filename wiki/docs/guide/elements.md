# 属性と状態異常

Crownfall の戦闘には、**属性（Element）** と **状態異常（Status）** という二つの体系がある。

| 体系 | 役割 |
|---|---|
| **属性** | 弱点を突いたときのみ追加ダメージ。属性そのものに特殊効果はない。 |
| **状態異常** | 継続ダメージ・行動阻害・与/被ダメージ補正など、時間軸で効く効果。 |

---

## 属性 — 5種

| 属性 | id | 正体（エルダの相） |
|---|---|---|
| 炎 | `fire` | — |
| 氷 | `ice` | — |
| 電気 | `thunder` | — |
| 闇 | `dark` | 瘴気 |
| 聖 | `holy` | 浄化の光 |

### ダメージ補正

| 条件 | 倍率 |
|---|---|
| 攻撃に属性があり、かつ敵がその属性弱点 | **×1.25** |
| 上記以外 | ×1.0 |

- **無属性**（武器・スキルの属性が空）は弱点ボーナスなし＝通常ダメージのみ。
- **耐性によるペナルティ（×0.75）は採用していない**（モンハン型の簡略ルール）。弱点を突けば得、それ以外は等倍と覚えればよい。

!!! warning "炎属性 ≠ 炎上"
    **炎属性（fire）** は弱点時の追加ダメージのみ。**炎上（ignite）** は継続ダメージを与える状態異常で、別物。たとえば炎属性の大剣は通常攻撃で「弱点:炎」、固有スキルで「炎上」を付与する、というように使い分ける。

---

## 状態異常 — 6種

戦闘の時間単位は **1 tick = 1.5秒**。

| 状態 | id | 効果 |
|---|---|---|
| 毒 | `poison` | 継続ダメージ 4/tick、5 tick、最大3スタック |
| 冷却 | `chill` | 敵の行動を50%スキップ、3 tick |
| 感電 | `shock` | 被ダメージ ×1.15 ＋ 行動30%スキップ、3 tick、最大2スタック |
| 炎上 | `ignite` | 継続ダメージ 3/tick、4 tick、最大3スタック |
| 呪い | `curse` | 与ダメージ ×0.75、4 tick |
| スタン | `stun` | 行動を100%スキップ、2 tick |

### 主な付与経路

| 状態 | 付与源 |
|---|---|
| 毒 | Affix `septic` / スキル |
| 冷却 | Affix `chilling` / スキル「鎮め」系（霜刃） |
| 感電 | Affix `shocking` / スキル「帯電」系（電撃短刀） |
| 炎上 | Affix `igniting` / スキル「焚き付け」系（炎属性大剣） |
| スタン | クロックモスの被弾（約20%）/ Affix・スキル |
| 呪い | プレイヤー→敵のデバフ（スキル） |

---

→ どの敵がどの属性に弱いかは[モンスター図鑑](../data/monsters.md)を参照。
