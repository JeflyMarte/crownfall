# ジョブ

探索隊メンバーの職能。武器の得意種別と習得スキルが異なる。一定レベルで上位職へ進化する。

## アルケミスト

*杖で魔法ダメージ・デバフに加え、回復と鼓舞（与ダメ上昇）で味方を支える支援役。*

| 項目 | 内容 |
|---|---|
| 役割 | support |
| 得意武器 | 杖 |
| 初期スキル | — |
| 習得スキル | アンブラボルト / 治癒 / 鼓舞 / ボルトエッジ / イグナイトエッジ / フロストエッジ |
| 進化先 | セージ（Lv10）|

## ビーストテイマー

*弓・杖を扱い、生物の生態を利用して戦う支援職。（召喚は将来実装）*

| 項目 | 内容 |
|---|---|
| 役割 | support |
| 得意武器 | 弓 / 杖 |
| 初期スキル | スネアアロー |
| 習得スキル | スネアアロー / スナイプショット / アンブラボルト / フロストエッジ |
| 進化先 | ビーストロード（Lv10）|
